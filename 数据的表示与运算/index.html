# 数据的表示

# 概述

计算机中的数据有不同的形式，包括数字、文本、音频、图像、视频等。这些数据都采用统一的表示方法在计算及内部进行表示，表示方法称为位模式 (bit pattern)。位模式指的是根据每个数据存储为一串序列，序列的每一位对应计算机的一个存储元，为 0 或 1。在这里，我们先研究数的表示。

# 整数的表示

整型数可以采用无符号数 (unsigned) 和有符号数 (signed) 两种方式表示。其中，无符号数指的是一定范围内的非负整数，范围与机器字长有关。例如，字长为 16 的机器，无符号数的范围为 0~65 535，与之对应的是，有符号数指一定范围内的整数。字长为 16 的机器对应有符号数的范围为 - 32 768~+32 767。

# 无符号数

无符号数在计算机内根据二进制直接表示。
