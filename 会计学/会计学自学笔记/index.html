# 总论

# 会计的含义与特点

会计职业存在的根本目的是提高企业的经营管理水平，让企业得以以更高效的方式运行，提高经济效益。具体说来，会计的主要工作是进行企业经济管理，

# 会计的分类

根据会计信息的使用者进行分类，会计包括财务会计和管理会计。根据会计所服务的领域分类，会计可以分为企业会计、非营利组织会计和政府会计。

财务会计是对外报告会计，为企业所有者、债权人和其他利益相关方提供会计信息。管理会计是对内报告会计，向企业管理者提供内部经营管理和经营决策的会计信息。

# 财务报告

财务报告又称财务会计报告，是企业对外提供的反映企业某一特定日期财务状况和某一会计期间经营成果、现金流量的书面文件。其包含会计报表及其附注，以及其余应在财务报告中披露的相关信息和资料。

# 主要会计报表

财务报告中应该披露的主要会计报表包括

  • 资产负债表：反映企业在某一特定日期财务状况的报表。其包括所拥有或控制的经济资源、所承担的现时义务和所有者对净资产的要求权。即资产、负债和所有者权益。
  • 利润表：该表是反映企业一定会计期间经营成果的报表。其由各项收入、费用，以及构成利润的各个项目 (包括营业利润、利润总额和净利润) 所组成的。
  • 现金流量表
  • 所有者权益变动表

# 财务报告要素

财务报告要素即财务报告的具体内容，又称会计要素，是会计核算和监督的对象。包括以下几类：

  • 反映财务状况的要素
    • 资产：企业拥有的资源
    • 负债：导致经济利益流出的企业现时义务
    • 所有者权益：企业资产扣除负债后所有者享有的剩余权益，即投资者对企业净资产的所有权。包括所有者投入的资本和企业的留存收益等。
  • 反映经营成果的要素
    • 收入：企业在日常活动中形成的、会导致所有者权益增加的、与所有者投入资本无关的经济利益的总流入。
    • 费用：费用是指企业在日常活动中发生的、会导致所有者权益减少的、与向所有者分配利润无关的经济利益的总流出。
    • 利润：利润是指企业在一定会计期间的经营成果。包括收入减去费用后的净额（营业利润）、直接计入当期利润的利得和损失。
