# Appearance

高真实感 Appearance 特点：

  • High dimensionality
  • High frequency: 视角微小移动带来极大变化 (glance)
  • High rank

因此，高真实的 Appearance 不能简单用贴图实现。

Nerf 不足以描述外观。因为关于其材质等信息存留于 Nerf 场内。

# 总结

  • 照片级真实感渲染需要理解物体详细的几何结构
  • 降低几何复杂度是加快渲染速度的方式
  • 看不到也很重要
  • 简化几何结构，需要保留光学属性
