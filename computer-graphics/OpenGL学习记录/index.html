# OpenGL дёӯзҡ„ж•°жҚ®зұ»еһӢ

OpenGL дёӯеҸҜд»ҘдёҚйҮҮз”Ё C/C++ еҺҹз”ҹзҡ„ж•°жҚ®зұ»еһӢпјҢеҰӮ int , float зӯүгҖӮеҸ–иҖҢд»Јд№Ӣзҡ„жҳҜ OpenGL дёӯиҮӘе®ҡд№үзҡ„ж•°жҚ®зұ»еһӢгҖӮдёҖиҲ¬йҮҮз”Ё GL* иЎЁзӨәпјҢеҰӮ GLuint (unsigned int еҪўејҸ), GLboolean (boolean еҪўејҸ) зӯүгҖӮ

# OpenGL дёӯзҡ„еҮҪж•°зұ»еһӢ

OpenGL дёӯеҜ№еҜ№иұЎзҡ„ж“ҚдҪңжңүеӣӣз§Қеҹәжң¬зҡ„еҮҪж•°зұ»еһӢпјҡ

  • void glGen*() еҪўејҸзҡ„еҮҪж•°иҙҹиҙЈиҝӣиЎҢ OpenGL еҜ№иұЎеҗҚз§°зҡ„еҲҶй…ҚгҖӮзұ»дјјдәҺдёҖдёӘжҢҮй’ҲеҸҳйҮҸзҡ„еҲҶй…ҚгҖӮ
  • void glBind*() еҪўејҸзҡ„еҮҪж•°иҙҹиҙЈиҝӣиЎҢеҜ№иұЎз»‘е®ҡгҖӮ
  • void glDelete*() еҪўејҸзҡ„еҮҪж•°иҙҹиҙЈеҜ№иұЎзҡ„йҮҠж”ҫгҖӮ
  • GLboolean glls*() еҪўејҸзҡ„еҮҪж•°иҙҹиҙЈжЈҖжҹҘжҹҗдёӘеҗҚз§°жҳҜеҗҰиў«еҲҶй…ҚдёәеҜ№иұЎгҖӮ
