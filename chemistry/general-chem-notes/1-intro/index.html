# 什么是化学？

化学是一门关于如何创造新物质的科学，它的任务是研究物质的性质、组成、结构和化学变化及其能量变化的规律。

# 化学变化的特征

化学变化包括如下三个特征：

  1. 化学变化是质变，即反应中设计化学键的断裂和形成，是反应过程中各个原子之间的重新组合。
  2. 化学变化是定量的变化。
  3. 化学变化伴随着能量的变化。

# 化学的疆域

一般将化学分为五大领域：

  1. 无机化学：研究无机化合物的性质及反应。
  2. 有机化学：研究碳氢化合物及其衍生物。
  3. 分析化学：分析物质的组成、结构和性质，并分离和提纯物质。
  4. 物理化学：使用物理方法研究化学问题，包括化学热力学、化学动力学、量子化学、表面化学、催化和电化学等。
  5. 高分子化学：研究高聚物的合成、反应化学和物理性质以及应用。
