# 电导率

# 基本概念

电导率 (electric conductivity) 是电解质溶液的一种与尺寸无关的特性，其被定义为单位立方厘米体积电解质溶液的电导。对截面积 (AAA) 均一的电解池，若两电极之间的距离为 lll, 则电解质溶液的电导率 κI\kappa_IκI​ 为：

κI=lAL.\kappa_I = \frac{l}{A}L.κI​=Al​L.

# 理想稀溶液的电导率

从微观层面来看，对于离子之间不发生相互干扰的理想稀溶液，电导率可以采用如下方式计算：

κI=e0∑iniziui∝∑inizi2ηri,\kappa_I = e_0\sum_in_iz_iu_i \propto \sum_i \frac{n_i z_i^2}{\eta r_i},κI​=e0​i∑​ni​zi​ui​∝i∑​ηri​ni​zi2​​,

其中 η\etaη 是溶液的黏度，nin_ini​ 是带电离子在溶液中的密度，正比于溶液的浓度，iii 取遍溶液中所有的带电离子。

因此，要想使用公式预测电导率，需要使用两个较难获得的量，分别是溶液的黏度 η\etaη 和溶液中离子的半径 rir_iri​.

# 摩尔电导率和 Kohlrausch 定律

在上述理论推导式中，可以看到电导率正比于溶液的浓度。因此可以定义与浓度无关的电导率值，即摩尔电导率 (molar conductivity) Λ\LambdaΛ 为：

Λ=κIc,\Lambda = \frac{\kappa_I}{c},Λ=cκI​​,

其中 ccc 是溶液的摩尔体积浓度。

在实际测量中，溶液中离子间的强相互作用会使得电导率和浓度呈现非线性关系，即摩尔电导率 Λ\LambdaΛ 随浓度 ccc 的变化而变化。科尔劳施定律 (Kohlrausch's law) 描述了这种关系：

Λ=Λ0−kc/c0,\Lambda = \Lambda_0 - k\sqrt{c/c_0},Λ=Λ0​−kc/c0​​,

其中 Λ0\Lambda_0Λ0​ 是无限稀溶液的摩尔电导率，kkk 是经验常数，c0=1mol⋅dm−3c_0 = 1 \text{ mol} \cdot \text{dm}^{-3}c0​=1 mol⋅dm−3 是标准溶液的摩尔浓度。
