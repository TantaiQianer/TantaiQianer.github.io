# 势博弈

# 定义

给定一个正则形式博弈，若存在函数 \Phi: A \ti \mathbb{R}, 使得对 ∀i,ai,ai′,a−i\forall i, a_i, a_i', a_{-i}∀i,ai​,ai′​,a−i​, 都满足

那么称该博弈为一个势博弈 (potential game), Φ\PhiΦ 称为势函数 (potential function).

# 性质

任何有限势博弈一定存在一个纯策略纳什均衡。

势博弈的局部最优解也可能是均衡。
