# 斯塔克伯格博弈 (Stackelberg Game)

标准形式的斯塔克伯格博弈 (Stackelberg Game) 由两个玩家组成，记为 N={1,2}N = \{1, 2\}N={1,2}, 其中玩家 1 是领导者 (leader), 玩家 2 是跟随者 (follower). 两者的动作集分别为 AlA_lAl​ 和 AfA_fAf​, 效用函数分别记作 ul(a),uf(a)u_l(a), u_f(a)ul​(a),uf​(a), 其中 a=(al,af)a = (a_l, a_f)a=(al​,af​). 该博弈过程如下：

  1. 领导者选择策略 sls_lsl​ 并将其告知跟随者；
  2. 跟随者选择对该策略的最优回应 afa_faf​;
  3. 领导者根据 sls_lsl​ 选择动作 ala_lal​, 最终领导者与跟随者同时做出动作。

# 定义

给定一个正则形式博弈，若存在函数 Φ:A↦R\Phi: A \mapsto \mathbb{R}Φ:A↦R, 使得对 ∀i,ai,ai′,a−i\forall i, a_i, a_i', a_{-i}∀i,ai​,ai′​,a−i​, 都满足

ui(ai,a−i)−ui(ai′,a−i)=Φ(ai,a−i)−Φ(ai′,a−i),u_i(a_i, a_{-i}) - u_i(a_i', a_{-i}) = \Phi(a_i, a_{-i}) - \Phi(a_i', a_{-i}),ui​(ai​,a−i​)−ui​(ai′​,a−i​)=Φ(ai​,a−i​)−Φ(ai′​,a−i​),

那么称该博弈为一个势博弈 (potential game), Φ\PhiΦ 称为势函数 (potential function).

# 性质

任何有限势博弈一定存在一个纯策略纳什均衡。

势博弈的局部最优解也可能是均衡。
