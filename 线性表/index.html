# 概述

线性表是一些数据元素的有限序列。根据存储方式的不同可以分为顺序表示和链式表示。

对于一个基本的线性表，有一些常见的操作，我们用一个抽象类简单表示。

# 线性表的顺序表示

线性表的顺序表示中，线性表的各项如同数组，存储在地址连续的存储单元中。这样存储带来的好处是查找数据花费时间仅为O(1)O(1)O(1)，但缺点在于插入和删除数据花费时间为O(n)O(n)O(n)，且需要预分配存储空间，存储空间分配不当容易造成空间浪费或数据溢出。
