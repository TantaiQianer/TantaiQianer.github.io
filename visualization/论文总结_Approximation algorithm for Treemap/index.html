# 问题

对于矩形的分划，先前已有三个基本的问题作为铺垫：

  • 如何将原矩形 R0R_0R0​ 划分成矩形集 R\mathscr{R}R, 使得周长和 ps(R)ps(\mathscr{R})ps(R) 最小。称为周长和最小问题 (PERI-SUM)。
  • 如何将原矩形 R0R_0R0​ 划分成矩形集 R\mathscr{R}R, 使得最大周长 pm(R)pm(\mathscr{R})pm(R) 最小。称为最大周长最小问题 (PERI-MAX)。
  • 如何将原矩形 R0R_0R0​ 划分成矩形集 R\mathscr{R}R, 使得最大长宽比 ρ(R)\rho(\mathscr{R})ρ(R) 最小。称为最大长宽比最小问题 (ASPECT-RATIO)。

以上三个问题都是 NP-hard 问题 。

而本篇论文中的算法，可以在O(nlog⁡n)O(n\log n)O(nlogn) 复杂度内使矩形的划分R\mathscr{R}R 同时达到：

  • PERI-SUM 问题的23\frac{2}{\sqrt{3}}3​2​ 倍最优解
  • PERI-MAX 问题的1.251.251.25 倍最优解
