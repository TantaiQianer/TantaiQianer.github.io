# 拓扑空间的基本概念

# 拓扑空间

# 定义

设 XXX 是一个集合，τ\tauτ 是 XXX 的一个子集族，若 τ\tauτ 满足：

  1. X,∅∈τX, \varnothing \in \tauX,∅∈τ.
  2. 若 A,B∈τA, B \in \tauA,B∈τ, 则 A∩B∈τA \cap B \in \tauA∩B∈τ.
  3. 若 τ1⊆τ\tau_1 \subseteq \tauτ1​⊆τ, 则 ∪A∈τ1A∈τ\cup_{A \in \tau_1}A \in \tau∪A∈τ1​​A∈τ.

则称 τ\tauτ 是 XXX 的一个拓扑 (topology), 并称 (X,τ)(X,\tau)(X,τ) 是一个拓扑空间 (topological space). 上述三个条件称为拓扑公理。

换句话说，拓扑空间实际上是一个配对，基于集合 XXX 给出子集族 τ\tauτ 以满足拓扑条件。

需要注意这里拓扑空间的定义，有限多成员的交在 τ\tauτ 中，任意多成员的并在拓扑 τ\tauτ 中。

# 例子

基于上述拓扑公理，下述两种拓扑是容易构造的：

  1. 对任一集合 XXX, {X,∅}\{X, \varnothing\}{X,∅} 是 XXX 上的拓扑，称为 XXX 的平凡拓扑。
  2. 对任一集合 XXX, 2X2^X2X 是 XXX 上的拓扑，称为 XXX 的离散拓扑。

实际上，对于给定的 XXX, 上述两拓扑实际上是针对 XXX 的最小和最大的拓扑。对于两拓扑空间 (X,τ1)(X, \tau_1)(X,τ1​) 和 (X,τ2)(X, \tau_2)(X,τ2​), 如果 τ1⊂τ2\tau_1 \subset \tau_2τ1​⊂τ2​, 那么我们称 τ2\tau_2τ2​ 是更加精细的。面对不同的研究对象，我们需要选取合适粒度的拓扑。

下面，再给出几种常见的拓扑构造：

设 XXX 是无穷集，τf={Ac∣∣A∣<∞}∪{∅}\tau_f = \{A^c|\,|A|<\infty\} \cup \{\varnothing\}τf​={Ac∣∣A∣<∞}∪{∅} 是 XXX 的一个拓扑，称为 XXX 上的余有限拓扑 (cofinite topology), 即余集为有限集的全体组成的拓扑。

设 XXX 是不可数无穷集，τc={Ac∣A⊆X,Ais countable}∪{∅}\tau_c = \{A^c|A \subseteq X, A \ \text{is countable}\}\cup\{\varnothing\}τc​={Ac∣A⊆X,A is countable}∪{∅} 也是 XXX 的拓扑，称为 XXX 上的余可数拓扑，即余集为可数集的全体组成的拓扑。

τe={U∣U=⋃i=1n(ai,bi),ai<bi,n∈N}\tau_e = \{U|U = \bigcup_{i=1}^n (a_i, b_i), a_i < b_i, n \in \mathbb{N}\}τe​={U∣U=⋃i=1n​(ai​,bi​),ai​<bi​,n∈N} （即 UUU 是若干开区间的并集）是 R\mathbb RR 上的拓扑，称为 R\mathbb RR 上的欧式拓扑，记作 E1=(R,τe)\mathbb E^1 = (\mathbb R, \tau_e)E1=(R,τe​).

Ac≔X∖AA^c \coloneqq X \setminus AAc:=X∖A 称为 AAA 的余集，即全集与 AAA 的差集。

# 度量空间与度量拓扑

# 度量空间

集合 XXX 上的一个度量 (metric) 是一个满足如下三个条件的映射 d:X×X→Rd: X \times X \to \mathbb Rd:X×X→R:

  1. 正定性：d(x,x)=0,∀x∈Xd(x,x) = 0, \forall x \in Xd(x,x)=0,∀x∈X, d(x,y)>0,∀x,y∈X,x≠yd(x,y) > 0, \forall x,y \in X, x \neq yd(x,y)>0,∀x,y∈X,x=y.
  2. 对称性：d(x,y)=d(y,x),∀x,y∈Xd(x,y) = d(y,x), \forall x,y \in Xd(x,y)=d(y,x),∀x,y∈X.
  3. 三角不等式：d(x,z)≤d(x,y)+d(y,z),∀x,y,z∈Xd(x,z) \leq d(x,y) + d(y,z), \forall x,y,z \in Xd(x,z)≤d(x,y)+d(y,z),∀x,y,z∈X.

规定了度量 ddd 的集合 XXX 称为一个度量空间 (metric space).

例如，对于 Rn\mathbb{R}^nRn, 我们可以定义通常的距离概念：

d((x1,x2,…,xn),(y1,y2,…,yn))=∑i=1n(xi−yi)2d((x_1,x_2,\dots,x_n), (y_1, y_2,\dots,y_n)) = \sqrt{\sum_{i=1}^n(x_i - y_i)^2}d((x1​,x2​,…,xn​),(y1​,y2​,…,yn​))=i=1∑n​(xi​−yi​)2​

容易验证，ddd 是一个度量。我们称该度量空间 (Rn,d)(\mathbb{R}^n, d)(Rn,d) 为 nnn 维欧式空间，记作 EnE^nEn.

# 度量拓扑

在度量空间 XXX 中，我们定义

B(x0,ε):={x∈X∣d(x0,x)<ε}B(x_0,\varepsilon):=\{x \in X|d(x_0,x) < \varepsilon\}B(x0​,ε):={x∈X∣d(x0​,x)<ε}

是以 x0x_0x0​ 为球心，ε\varepsilonε 为半径的球形邻域。

可以证明 (X,d)(X,d)(X,d) 的任意两个球形邻域的交集是若干球形邻域的并集。

若干也可以是不可数若干，因此对于在两球形邻域交集中的每一个点 xxx, 构造一个以 xxx 为球心且不超出该交集的半径的球形邻域即可完成构造。

因此，规定 τd:={U∣U=⋃B(x,εx)}\tau_d := \{U|U = \bigcup B(x, \varepsilon_x)\}τd​:={U∣U=⋃B(x,εx​)}. 根据上述证明，τd\tau_dτd​ 是一个拓扑，称之为度量拓扑.

# 拓扑空间内的研究对象

# 开集和闭集

对于拓扑空间 (X,τ)(X,\tau)(X,τ), 我们称 τ\tauτ 中的元素为该拓扑空间的开集。对拓扑空间 XXX 中的集合 AAA, 若 AcA^cAc 是开集，那么称 AAA 为闭集。显然，在拓扑空间 XXX 中， XXX 和 ∅\varnothing∅ 是满足既是开集也是闭集的集合。

通过拓扑公理和 De Morgan 律，我们容易推出闭集的两个基本性质：

  • 任意多个闭集的交还是闭集
  • 有限多个闭集的并是闭集

# 邻域

设 AAA 是拓扑空间 XXX 中的一个子集，点 x∈Ax \in Ax∈A, 若存在开集 UUU 使得 x∈U⊆Ax \in U \subseteq Ax∈U⊆A, 则称 xxx 是 AAA 的一个内点，称 AAA 是 xxx 的一个邻域。AAA 的所有内点组成的集合称为 AAA 的内部，记作 A˚\mathring AA˚ 或 A∘A^\circA∘.

关于邻域和内部，有以下较显然的性质：

  • 若 A⊂BA \subset BA⊂B, 则 A˚⊂B˚\mathring A \subset \mathring BA˚⊂B˚
  • A˚\mathring AA˚ 是包含在 AAA 中的所有开集的并集，因此是包含在 AAA 中的最大开集
  • A˚=A\mathring A = AA˚=A 等价于 AAA 是开集
  • (A∩B)∘=A˚∩B˚(A \cap B)^\circ = \mathring A \cap \mathring B(A∩B)∘=A˚∩B˚
  • (A∪B)∘⊃A˚∪B˚(A \cup B)^\circ \supset \mathring A \cup \mathring B(A∪B)∘⊃A˚∪B˚

# 聚点和闭包

设 AAA 是拓扑空间 XXX 的子集，x∈Xx \in Xx∈X. 若 xxx 的每个邻域都含有 A∖{x}A \setminus \{x\}A∖{x} 中的点，则称 xxx 是 AAA 的聚点。AAA 的所有聚点的集合称为 AAA 的导集，记作 A′A'A′, 称集合 Aˉ:=A∪A′\bar A := A \cup A'Aˉ:=A∪A′ 为 AAA 的闭包，也记作 cl(A)\text{cl}(A)cl(A).

根据聚点和闭包的定义，我们可以立刻得到下面的性质：

  • 若拓扑空间 XXX 的子集 AAA 和 BBB 互为余集，则 Aˉ\bar AAˉ 和 B˚\mathring BB˚ 也互为余集。
  • 若 A⊂BA \subset BA⊂B, 则 Aˉ⊂Bˉ\bar A \subset \bar BAˉ⊂Bˉ.
  • Aˉ\bar AAˉ 是所有包含 AAA 的闭集的交集，即是包含 AAA 的最小闭集。
  • Aˉ=A\bar A = AAˉ=A 等价于 AAA 是闭集。
  • A∪B‾=Aˉ∪Bˉ\overline{A \cup B} = \bar A \cup \bar BA∪B=Aˉ∪Bˉ.
  • A∩B‾⊂Aˉ∩Bˉ\overline{A \cap B} \subset \bar A \cap \bar BA∩B⊂Aˉ∩Bˉ.

# 稠密

拓扑空间 XXX 的子集 AAA 称为稠密的，当且仅当 Aˉ=X\bar A = XAˉ=X. 若 XXX 的稠密子集是可数的，那么称 XXX 为可分拓扑空间。

# 子空间

AAA 是拓扑空间 (X,τ)(X, \tau)(X,τ) 的一个非空子集，规定 AAA 的子集族

τA:={U∩A∣U∈τ}\tau_A:=\{U \cap A| U \in \tau\}τA​:={U∩A∣U∈τ}

容易验证 τA\tau_AτA​ 是 AAA 上的一个拓扑，称为 τ\tauτ 导出的 AAA 上的子空间拓扑，称 (A,τA)(A, \tau_A)(A,τA​) 为(X,τ)(X,\tau)(X,τ) 的子空间。

# 拓扑空间的序关系

对于拓扑空间 (X,τ)(X,\tau)(X,τ) 内的两个元素 x,yx, yx,y, 我们可以定义其特殊化预序 (specialization preorder) x≤yx \leq yx≤y 当且仅当 xˉ≤yˉ\bar{x} \leq \bar yxˉ≤yˉ​. 可以证明，满足该条件的关系是一个预序关系，即满足自反性和传递性。

# 拓扑中的连续性

# 连续的定义

定义 设 XXX 和 YYY 都是拓扑空间，f:X→Yf:X \to Yf:X→Y 是一个映射，x∈Xx \in Xx∈X. 如果对于 YYY 中 f(x)f(x)f(x) 的任一邻域 VVV, f−1(V)f^{-1}(V)f−1(V) 总是 xxx 的邻域，则说 fff 在 xxx 处连续。若在任一点 x∈Xx \in Xx∈X 处 fff 都连续，那么称 fff 是连续映射。

值得注意的是，在拓扑空间中，连续的概念可能与直观并不完全一致。例如我们取一个连续映射

f:R→R,f(x)=x2f:\mathbb{R} \to \mathbb{R}, f(x) = x^2f:R→R,f(x)=x2

显然 f(2)=4f(2) = 4f(2)=4. 此时取 V=(4−ε,4+ε),ε>0V = (4-\varepsilon, 4+\varepsilon), \;\varepsilon>0V=(4−ε,4+ε),ε>0 是 f(2)f(2)f(2) 的一个邻域，而

f−1(V)=(2−ε,2+ε)∪(−2−ε,−2+ε)f^{-1}(V) = (2-\varepsilon, 2+\varepsilon) \cup (-2-\varepsilon, -2+\varepsilon)f−1(V)=(2−ε,2+ε)∪(−2−ε,−2+ε)

这显然是拓扑意义下 222 的一个邻域，然而这不同于欧式空间中我们所定义的邻域。尽管邻域的定义不同，但最终连续的结果是相同的。这是可以证明的。

# 连续映射的性质

性质 1 连续映射可以采用开集描述。对于映射 f:X→Yf:X \to Yf:X→Y, 下面的三个条件相互等价：

  1. fff 是连续映射
  2. YYY 的任一开集在 fff 下的原像是 XXX 的开集
  3. YYY 的任一闭集在 fff 下的原像是 XXX 的闭集

性质 2 离散拓扑上的映射是连续映射。映射到平凡拓扑的映射是连续映射。

性质 3 (复合映射的连续性) 设 X,YX,YX,Y 和 ZZZ 都是拓扑空间，映射 f:X→Yf:X \to Yf:X→Y 在 xxx 处连续，g:Y→Zg : Y \to Zg:Y→Z 在 f(x)f(x)f(x) 处连续，则复合映射 g∘f:X→Zg \circ f:X \to Zg∘f:X→Z 在 xxx 处连续。

性质 4(粘接定理) 设 {A1,A2,…,An}\{A_1, A_2, \dots, A_n\}{A1​,A2​,…,An​} 是 XXX 的一个有限闭覆盖．如果映射 f:X→Yf:X \to Yf:X→Y 在每个 AiA_iAi​ 上的限制都是连续的，则 fff 是连续映射。

这个定理看上去十分显然，但却是我们后续构造许多连续曲面的基础。

# 同胚映射

定义 如果 f:X→Yf:X \to Yf:X→Y 是双射，且 fff 及其逆 f−1:Y→Xf^{-1}: Y \to Xf−1:Y→X 都是连续的，则称 fff 是一个同胚映射，或称拓扑变换，或简称同胚。当存在 XXX 到 YYY 的同胚映射时，就称 XXX 与 YYY 同胚，记作 X≅YX \cong YX≅Y.

下面是几个同胚的例子。

例 1 开区间 (作为 E1E^1E1 的子空间) 同胚于 E1E^1E1.

例 2 以 BnB^nBn 表示 EnE^nEn 中的单位球，那么 B˚n≅En\mathring{B}^n \cong E^nB˚n≅En.

例 3 以 OOO 表示原点，那么 En∖{O}≅En∖BnE^n \setminus \{O\} \cong E^n \setminus B^nEn∖{O}≅En∖Bn.

例 4 S2S^2S2 删去一点后与 E2E^2E2 同胚。实际上，这可以用复数的球面表示和代数表示作为映射方式。

例 5 任何包含内部的凸多边形相互同胚。一般地，EnE^nEn 内的凸集相互同胚。这可以通过转化成基本的 BnB^nBn 证明。

# 乘积空间

对于两拓扑空间 (X1,τ1),(X2,τ2)(X_1, \tau_1), (X_2, \tau_2)(X1​,τ1​),(X2​,τ2​), 我们希望构造一个 X1×X2X_1 \times X_2X1​×X2​ 上的最小拓扑，同时包含 τ1\tau_1τ1​ 与 τ2\tau_2τ2​. 这实际上和线性空间的直和类似。我们定义如下：

对两拓扑空间 (X1,τ1),(X2,τ2)(X_1, \tau_1), (X_2, \tau_2)(X1​,τ1​),(X2​,τ2​), 我们称

B‾={U1×U2∣Ui∈τi}\overline{\mathscr{B}} = \{U_1 \times U_2|U_i \in \tau_i\}B={U1​×U2​∣Ui​∈τi​}

为 X1×X2X_1 \times X_2X1​×X2​ 上的乘积拓扑 (product topology), 并称 (X1×X2,B‾)(X_1 \times X_2, \overline{\mathscr{B}})(X1​×X2​,B) 是 (X1,τ1)(X_1, \tau_1)(X1​,τ1​) 与 (X2,τ2)(X_2, \tau_2)(X2​,τ2​) 的乘积空间 (product space).

# 拓扑基

拓扑基同时是集合和拓扑空间上的概念，两者存在区别。对于集合 X, 其拓扑基是它的一个子集族 B\mathscr{B}B, 使得 Bˉ\bar{\mathscr{B}}Bˉ 是 XXX 的一个拓扑；对于拓扑空间 (X,τ)(X, \tau)(X,τ), 其拓扑基 B\mathscr{B}B 同样是 XXX 的一个子集族，但是要求 Bˉ=τ\bar{\mathscr{B}} = \tauBˉ=τ.
