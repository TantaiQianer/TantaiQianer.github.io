# 分离公理

分离公理 (separation axioms) 是用来描述两个点或者闭集之间能否用邻域分隔的性质，是对拓扑空间的附加要求。在这里用一张 sola 文章 中的图解释：

  1. T1T_1T1​ 公理：任意两个不同点 xxx 与 yyy, xxx 有邻域不含 yyy, yyy 有邻域不含 xxx.
  2. T2T_2T2​ 公理：任何两个不同点有不相交的邻域。

T2T_2T2​ 公理是最重要的分离公理，满足该公理的空间称为豪斯多夫空间 (Hausdorff space).

其实其它几个也有名字来着，尽管不太常用。满足 T1T_1T1​ 公理的叫弗雷歇空间 (Fréchet space), 满足 T3T_3T3​ 公理的叫正则豪斯多夫空间 (regular Hausdorff space), 满足 T4T_4T4​ 公理的叫正规豪斯多夫空间 (normal Hausdorff space).

注意实数集上的余有限拓扑 (R),τf(\mathbb{R}), \tau_f(R),τf​, 对于点 xxx, 其存在邻域 R∖{y}\mathbb{R} \setminus \{y\}R∖{y} 不含 yyy. 类似，点 yyy 也存在邻域 R∖{x}\mathbb{R} \setminus \{x\}R∖{x} 不含 xxx, 因此其满足 T1T_1T1​ 公理。但由于该拓扑是 “余有限” 的，因此任意两邻域必定相交，因此不满足 T2T_2T2​ 公理。

命题 2 在 Hausdorff 空间中，一个序列不会收敛到两个以上的点。

  1. T3T_3T3​ 公理：任意一点与不含它的任一闭集有不相交的（开）邻域。
  2. T4T_4T4​ 公理：任意两个不相交的闭集有不相交的（开）邻域。

# 可数公理

可数公理 (countability axioms) 包括第一可数公理和第二可数公理，分别简称 C1C_1C1​ 公理和 C2C_2C2​ 公理。满足 CiC_iCi​ 公理的空间称为 CiC_iCi​ 空间。

第一可数公理：任一点都有可数的邻域基。
