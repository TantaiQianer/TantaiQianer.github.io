# 离散概率分布

# 伯努利试验

我们考虑最基本的事件域 F={∅,A,Aˉ,Ω}\mathcal F= \{\varnothing, A, \bar{A}, \Omega\}F={∅,A,Aˉ,Ω}, 并将 AAA 称为成功，将 Aˉ\bar AAˉ 称为失败。这样的只有两个可能结果的试验称为伯努利试验。

对于伯努利试验，为简化公式，我们一般记事件AAA 发生的概率为 ppp, 事件 Aˉ\bar AAˉ 发生的概率为 q=1−pq=1-pq=1−p. 离散概率分布大多与伯努利试验相关。

# 伯努利分布

只进行一次伯努利试验，事件 AAA 发生次数的概率分布称为伯努利分布，又称 0-1 分布。其期望为 ppp, 方差为 pqpqpq. 证明是显然的。

# 一般的正态分布

我们称密度函数为

f(x)=12πσe−(x−μ)22σ2,x∈Rf(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}, \; x \in \mathbb{R}f(x)=2π​σ1​e−2σ2(x−μ)2​,x∈R

EX=12πσ∫−∞+∞xe−(x−μ)22σ2dx=12πσ∫−∞+∞(x−μ)e−(x−μ)22σ2d(x−μ)+12πσ∫−∞+∞μe−(x−μ)22σ2dx=0+μ=μ\begin{aligned} \mathbb{E}X & = \frac{1}{\sqrt{2\pi}\sigma}\int_{-\infty}^{+\infty}xe^{-\frac{(x-\mu)^2}{2\sigma^2}} \mathrm dx\\ & = \frac{1}{\sqrt{2\pi}\sigma}\int_{-\infty}^{+\infty}(x-\mu)e^{-\frac{(x-\mu)^2}{2\sigma^2}} \mathrm d(x-\mu) + \frac{1}{\sqrt{2\pi}\sigma}\int_{-\infty}^{+\infty}\mu e^{-\frac{(x-\mu)^2}{2\sigma^2}} \mathrm dx\\ & = 0 + \mu\\ & = \mu \end{aligned}EX​=2π​σ1​∫−∞+∞​xe−2σ2(x−μ)2​dx=2π​σ1​∫−∞+∞​(x−μ)e−2σ2(x−μ)2​d(x−μ)+2π​σ1​∫−∞+∞​μe−2σ2(x−μ)2​dx=0+μ=μ​

# 标准正态分布

我们称 N(0,1)N(0,1)N(0,1) 为标准正态分布。于是其期望为 000, 方差为 111. 通常情况下，标准正态分布记作 UUU, 其密度函数记作 ϕ(u)\phi(u)ϕ(u), 分布函数记作 Φ(u)\Phi(u)Φ(u). Φ(u)\Phi(u)Φ(u) 的值一般可以通过查表获得。

标准正态分布有一些基本的性质，我们不加证明地列举如下：

  • Φ(−u)=1−Φ(u).\Phi(-u) = 1-\Phi(u).Φ(−u)=1−Φ(u).
  • P(U>u)=1−Φ(u).\mathbb{P}(U>u) = 1-\Phi(u).P(U>u)=1−Φ(u).
  • P(a<U<b)=Φ(b)−Φ(a).\mathbb{P}(a<U<b) = \Phi(b) - \Phi(a).P(a<U<b)=Φ(b)−Φ(a).

对于一般的正态分布 X∼N(μ,σ2)X \sim N(\mu, \sigma^2)X∼N(μ,σ2), 有 X∗≔(X−μ)/σ∼N(0,1)X^* \coloneqq (X-\mu)/\sigma \sim N(0,1)X∗:=(X−μ)/σ∼N(0,1). 我们称将 XXX 变换为 (X−μ)/σ(X-\mu)/\sigma(X−μ)/σ 的过程为正态分布的标准化。显然，通过标准化求一般正态分布的期望和方差是更加便捷的。
