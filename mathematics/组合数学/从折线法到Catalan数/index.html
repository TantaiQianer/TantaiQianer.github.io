# 折线法

最基本的折线法是无限制的情况。问题的具体描述如下：

在一个平面直角坐标系 xOyxOyxOy 中，我们取起点 A(x1,y1)A(x_1,y_1)A(x1​,y1​) 及终点 B(x2,y2)B(x_2,y_2)B(x2​,y2​), 其中 xi,yi∈Zx_i,y_i \in \mathbb{Z}xi​,yi​∈Z. 假设一个质点从起点 AAA 出发，每步操作可以选择向右上走一步 (即横纵坐标值同时 +1+1+1 ) 或向右下走一步 (即横坐标 +1+1+1, 纵坐标 −1-1−1), 问存在多少种不同的走法？

首先，我们需要考虑能否到达终点。这要求横纵坐标差的奇偶性相同，即 x2−x1≡y2−y1(mod2)x_2-x_1 \equiv y_2-y_1 (\mathrm{mod}\,2)x2​−x1​≡y2​−y1​(mod2). 此外，还要求 x2−x1≥y2−y1≥0x_2-x_1 \geq y_2-y_1 \geq 0x2​−x1​≥y2​−y1​≥0, 否则即便一直向上走也达不到。

在这种情况下，显然在总共 x2−x1x_2-x_1x2​−x1​ 步中，有 (x2+y2−x1−y1)/2(x_2+y_2-x_1-y_1)/2(x2​+y2​−x1​−y1​)/2 步向右上，(x2−y2−x1+y1)/2(x_2-y_2-x_1+y_1)/2(x2​−y2​−x1​+y1​)/2 步向右下。于是总的取法为 (x2−x1(x2+y2−x1−y1)/2)\binom{x_2-x_1}{(x_2+y_2-x_1-y_1)/2}((x2​+y2​−x1​−y1​)/2x2​−x1​​). 我们令 x=x2−x1x = x_2-x_1x=x2​−x1​，y=y2−y1y=y_2-y_1y=y2​−y1​, 则结果简化为 (x(x+y)/2)\binom{x}{(x+y)/2}((x+y)/2x​).

这样的折线法并没有什么意义，因为其就是组合数 (nm)\binom{n}{m}(mn​) 的再叙述。但如果将折线法的问题强化为下面的形式，那么折线法在组合计数中将具有相当的意义。

题目背景同上，如果起点 A(x1,y1)A(x_1,y_1)A(x1​,y1​), 终点 B(x2,y2)B(x_2,y_2)B(x2​,y2​) 同在 xxx 轴上方，那么质点移动路径中，与 xxx 轴有公共点的路径有多少？

这时，我们考虑一个反射。作 A′(x1,−y1)A'(x_1,-y_1)A′(x1​,−y1​). 显然，若存在 A→BA\to BA→B 的路径，则必然存在 A′→BA' \to BA′→B 的路径。根据连通性，A′→BA' \to BA′→B 必然与 xxx 轴相交，我们设其中横坐标最小的交点为 (x0,0)(x_0,0)(x0​,0). 那么我们将路径 A′→BA'\to BA′→B 中位于 x=x0x=x_0x=x0​ 左侧的部分沿 xxx 轴进行轴对称，便得到了 A→BA \to BA→B 的且与 xxx 轴相交的路径。容易验证，这种映射是双射。我们称此结论为反射原理。

根据反射原理，可以得到与 xxx 轴有公共点的路径总数为 (x2−x1(x2−x1+y2+y1)/2)\binom{x_2-x_1}{(x_2-x_1+y_2+y_1)/2}((x2​−x1​+y2​+y1​)/2x2​−x1​​).

在此基础上，我们可以计算出 A→BA \to BA→B 中与 xxx 轴不相交的路径数目。为 (x2−x1(x2−x1+y2−y1)/2)−(x2−x1(x2−x1+y2+y1)/2)\binom{x_2-x_1}{(x_2-x_1+y_2-y_1)/2} - \binom{x_2-x_1}{(x_2-x_1+y_2+y_1)/2}((x2​−x1​+y2​−y1​)/2x2​−x1​​)−((x2​−x1​+y2​+y1​)/2x2​−x1​​).

于是，便引出了第三个问题：

题目背景同上，A,BA,BA,B 同在 xxx 轴上方，问从 AAA 到 BBB 的路径中，不穿过 xxx 轴的路径总数有多少？

注意到，不穿过 xxx 轴等价于与轴 y=−1y=-1y=−1 不相交，于是将路径整体上移一个单位，就转化成不与 xxx 轴相交的问题。

构造点 A′(x1,y1+1)A'(x_1,y_1+1)A′(x1​,y1​+1), B′(x2,y2+1)B'(x_2,y_2+1)B′(x2​,y2​+1)，则从 A′A'A′ 到 B′B'B′ 不与 xxx 轴相交的路径数为 (x2−x1(x2−x1+y2−y1)/2)−(x2−x1(x2−x1+y2+y1+2)/2)\binom{x_2-x_1}{(x_2-x_1+y_2-y_1)/2} - \binom{x_2-x_1}{(x_2-x_1+y_2+y_1+2)/2}((x2​−x1​+y2​−y1​)/2x2​−x1​​)−((x2​−x1​+y2​+y1​+2)/2x2​−x1​​).

这个问题就具有一定的意义了。我们将折线的路径作对应，可以解决许多相关的计数问题。例如我们可以构造以下的问题：

甲乙两人参加竞选，甲获得 mmm 张选票，乙获得 nnn 张选票，且 m>nm>nm>n, 那么在逐一唱票的过程中，甲的票数一直领先乙的点票记录有多少种可能？

这属于与 xxx 轴恒不相交的问题，起点和终点依次为 (1,1)(1,1)(1,1) 和 (m+n,m−n)(m+n,m-n)(m+n,m−n) (因为第一票必定为甲获得)。于是代入公式，其总可能数为 (m+n−1m−1)−(m+n−1m)=m−nm+n(m+nm)\binom{m+n-1}{m-1}-\binom{m+n-1}{m} = \frac{m-n}{m+n}\binom{m+n}{m}(m−1m+n−1​)−(mm+n−1​)=m+nm−n​(mm+n​).

这是一个经典的问题，我们称之为选择问题。

收银员向 2n2n2n 人各收费 505050 元。2n2n2n 人中，有 nnn 人带了 111 张 100100100 元的钞票，另外 nnn 人带了一张 505050 元的钞票。若收银员最开始没有零钱，问这 2n2n2n 人有多少种排队方法，使得收银员能够完成收银而不需要借钱找零？

这可以转化成 (0,0)(0,0)(0,0) 到 (2n,0)(2n,0)(2n,0) 的，不穿过 xxx 轴的总路径问题。于是其结果为 (2nn)−(2nn+1)=1n+1(2nn)\binom{2n}{n} - \binom{2n}{n+1} = \frac{1}{n+1}\binom{2n}{n}(n2n​)−(n+12n​)=n+11​(n2n​).

而 1n+1(2nn)\frac{1}{n+1}\binom{2n}{n}n+11​(n2n​) 就是著名的 Catalan 数。
