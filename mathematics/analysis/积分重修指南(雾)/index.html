# 第一换元法

第一换元法，又称凑微分法或直接代换法。指若 ∫f(u)du=F(u)+C\displaystyle \int f(u) \mathrm du = F(u) + C∫f(u)du=F(u)+C, 且 u=u(x)u = u(x)u=u(x) 可微，则

∫f(u(x))u′(x)dx=F(u(x))+C\int f(u(x)) u'(x)\mathrm dx = F(u(x)) + C∫f(u(x))u′(x)dx=F(u(x))+C

它实际上是把 dx\mathrm dxdx 配凑 u′(x)u'(x)u′(x), 得到 du\mathrm dudu 从而简化被积函数的形式。

下面我们来举几个例子。

∫eλxdx\int e^{λx} \mathrm dx∫eλxdx

其中 λ≠0λ≠0λ=0.

解：

∫tan⁡xdx\int \tan x \mathrm dx∫tanxdx

解：

∫tan⁡xdx=∫sin⁡xcos⁡xdx=∫dcos⁡xcos⁡x=ln⁡cos⁡x+C\int \tan x \mathrm dx = \int \frac{\sin x}{\cos x}\mathrm dx = \int \frac{\mathrm d\cos x}{\cos x} = \ln \cos x + C∫tanxdx=∫cosxsinx​dx=∫cosxdcosx​=lncosx+C

# 第二换元法

第二换元法又称代入换元法或逆代换法。指若 x=x(t)x = x(t)x=x(t) 可微且存在反函数 t=t(x)t = t(x)t=t(x), 又若 ∫f(x(t))x′(t)dx=F(t)+C\displaystyle \int f(x(t))x'(t) \mathrm dx= F(t) + C∫f(x(t))x′(t)dx=F(t)+C, 则

∫f(x)dx=F(t(x))+C\int f(x) \mathrm dx = F(t(x)) + C∫f(x)dx=F(t(x))+C

# 分部积分法

根据 d(uv)=udv+vdu\mathrm d(uv) = u \mathrm dv + v\mathrm dud(uv)=udv+vdu, 有 udv=d(uv)−vduu \mathrm dv = \mathrm d(uv) - v\mathrm duudv=d(uv)−vdu, 从而将 ∫udv\displaystyle\int u\mathrm dv∫udv 转换成 ∫vdu\displaystyle \int v\mathrm du∫vdu 的计算。

例题 求

∫xexdx\int xe^x \mathrm dx∫xexdx

解

∫xexdx=xex−∫exdx+C=(x−1)ex+C\int xe^x \mathrm dx = xe^x - \int e^x\mathrm dx + C = (x-1)e^x + C∫xexdx=xex−∫exdx+C=(x−1)ex+C

在此处，我们将xexxe^xxex 的积分转化为exe^xex 的积分。类似地，对于指数函数与多项式相乘的形式，每一次分部积分可以进行一次降次。从而类似的积分都可以计算。

# 三角函数有理分式的积分

可以通过基本的代换进行计算。关于具体的代换方式，我们可以将三角函数的有理式表示为关于sin⁡x\sin xsinx 和cos⁡x\cos xcosx 的二元函数R(sin⁡x,cos⁡x)R(\sin x, \cos x)R(sinx,cosx), 再分成以下三种情况讨论：

若R(−sin⁡x,cos⁡x)=−R(sin⁡x,cos⁡x)R(-\sin x, \cos x) = -R(\sin x, \cos x)R(−sinx,cosx)=−R(sinx,cosx), 则令t=cos⁡xt = \cos xt=cosx；

若R(sin⁡x,−cos⁡x)=−R(sin⁡x,cos⁡x)R(\sin x, -\cos x) = -R(\sin x, \cos x)R(sinx,−cosx)=−R(sinx,cosx), 则令t=sin⁡xt = \sin xt=sinx；

若R(−sin⁡x,−cos⁡x)=R(sin⁡x,cos⁡x)R(-\sin x, -\cos x) = R(\sin x, \cos x)R(−sinx,−cosx)=R(sinx,cosx), 则令t=tan⁡xt = \tan xt=tanx。

此时，可以转化为有理函数的积分。
