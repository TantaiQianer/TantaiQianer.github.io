# 重修笔记前的碎碎念

其实之前是简单整理过一点点关于数学分析中极限理论的知识的，但是因为当时有更重要的事情在忙（或许这个只是借口），以及自己实际上真的傻傻分不清那些知识和概念，比如实数的那些基本定理等等，于是最终也没写多少。

现在看来，一个很重要的原因是，数学分析中所研究的 R\mathbb{R}R 或者说 Rn\mathbb{R}^nRn 的空间性质太好了，以致于许多性质都太过 trival. 但是好的空间总是可遇不可求的。因此，抱着复习和为拓扑学与微分几何打基础的心态，还是需要将这些知识重新梳理一下。

其实，分析学中进行严谨化的办法就是 ϵ−N\epsilon-Nϵ−N 或者 ϵ−δ\epsilon-\deltaϵ−δ 语言。然后按照离散情况与连续情况进行划分（对应数列极限与函数极限），每一部分都可以从单变元 R\mathbb{R}R 推广到多变元 Rn\mathbb{R}^nRn. 这样的话，打算还是这样来写。至于实数的那几个让人掉头发的基本定理，我觉得还是放到单变元数列极限中进行梳理比较好。

最后呢，打算直接在原有笔记的基础上进行一个修补，就不再新开一篇笔记了。

# 单变元数列极限

# 定义

给定数列 {an}n=0∞⊆R\{a_n\}_{n=0}^\infty \subseteq \mathbb{R}{an​}n=0∞​⊆R, 假若 ∃a∈R\exists a \in \mathbb{R}∃a∈R, 使得 ∀ε>0\forall \varepsilon>0∀ε>0, 若 ∃N∈R\exists N \in \mathbb{R}∃N∈R 使得 ∀n>N\forall n>N∀n>N 均有

# 收敛数列的性质

收敛数列的基本性质包括：

  1. 极限存在必唯一 (唯一性)
  2. 极限存在必有界 (有界性)
  3. 极限可以四则运算
  4. 夹逼定理
  5. 单调有界必有极限
  6. 数列的极限是其任意子数列的极限
  7. 任一有界数列中必存在一个趋于有限极限的数列 (闭区间套定理证明)

下面介绍一些数列极限不那么基本的性质。

∣xn−xm∣<ε|x_n-x_m|<\varepsilon∣xn​−xm​∣<ε

对于其余极限问题，柯西收敛准则类似。

# 确界原理

非空有上界实数集必有上确界，非空有下界实数集必有下确界。

# 单调有界收敛定理

单调递增有上界数列必收敛，单调递减有下界数列必收敛。

# 有界集、开集、闭集及相关概念

假设 SSS 是 Rn\mathbb{R}^nRn 中的点集。若存在 M>0M>0M>0 使得 S⊆BMnS \subseteq \mathbb{B}_M^nS⊆BMn​ 成立，则称 SSS 是有界集 (bounded set).

称 x∈Rn\bm{x} \in \mathbb{R}^nx∈Rn 是 SSS 的内点 (interior point), 如果存在 r>0r>0r>0 满足 Bn(x,r)⊆S\mathbb{B}^n(\bm{x}, r) \subseteq SBn(x,r)⊆S. SSS 的所有内点组成的集合记作 S∘S^\circS∘ 或 Int(S)\mathrm{Int}(S)Int(S), 称为 SSS 的内部 (interior).

称 x∈Rn\bm{x} \in \mathbb{R}^nx∈Rn 是 SSS 的外点 (exterior point), 如果存在 r>0r>0r>0 满足 Bn(x,r)⊆Rn∖S\mathbb{B}^n(\bm{x}, r) \subseteq \mathbb{R}^n \setminus SBn(x,r)⊆Rn∖S. SSS 的所有外点组成的集合记作 Ext(S)\mathrm{Ext}(S)Ext(S), 称为 SSS 的外部 (exterior).

称 x∈Rn\bm{x} \in \mathbb{R}^nx∈Rn 是 SSS 的边界点 (boundary point), 如果 x∉Int(S)∪Ext(S)\bm{x} \notin \mathrm{Int}(S) \cup \mathrm{Ext}(S)x∈/Int(S)∪Ext(S). SSS 的所有边界点组成的集合记作 ∂S\partial S∂S 或 Bdy(S)\mathrm{Bdy}(S)Bdy(S), 称为 SSS 的边界 (boundary). 即 x∈∂S\bm{x} \in \partial Sx∈∂S 当且仅当对 ∀r>0\forall r > 0∀r>0, 球 Bn(x,r)\mathbb{B}^n(\bm{x}, r)Bn(x,r) 既包含 SSS 的点又包含了 Rn∖S\mathbb{R}^n \setminus SRn∖S 中的点。

称 x∈Rn\bm{x} \in \mathbb{R}^nx∈Rn 是 SSS 的孤立点 (isolated point), 如果存在 r>0r>0r>0 满足 Bn(x,r)∩S={x}\mathbb{B}^n(\bm{x}, r) \cap S = \{\bm{x}\}Bn(x,r)∩S={x}. SSS 的所有孤立点组成的集合记作 Iso(S)\mathrm{Iso}(S)Iso(S), 显然 Iso(S)⊆∂S∩S\mathrm{Iso}(S) \subseteq \partial S \cap SIso(S)⊆∂S∩S, 即孤立点一定在边界上。

注意这里孤立点的要求是存在某个 rrr 满足要求即可，因为若 rrr 满足 Bn(x,r)∩S={x}\mathbb{B}^n(\bm{x}, r) \cap S = \{\bm{x}\}Bn(x,r)∩S={x}, 则 Bn(x,r/2)∩S={x}\mathbb{B}^n(\bm{x}, r/2) \cap S = \{\bm{x}\}Bn(x,r/2)∩S={x} 同样成立。

聚点的概念是孤立点反过来。

称 x∈Rn\bm{x} \in \mathbb{R}^nx∈Rn 是 SSS 的聚点 (cluster point), 如果对于任意 r>0r>0r>0 都有 Bn(x,r)\mathbb{B}^n(\bm{x}, r)Bn(x,r) 包含无穷多 SSS 中的点。SSS 的所有聚点组成的集合记作 S′S'S′, 称为 SSS 的导集 (derived set).

SSS 的闭包 (closure) 定义为 S‾≔S∪S′\overline{S} \coloneqq S \cup S'S:=S∪S′.

若 S∘=SS^\circ = SS∘=S, 则称 SSS 是开集 (open set).

若 S‾=S\overline{S} = SS=S, 则称 SSS 是闭集 (closed set).
