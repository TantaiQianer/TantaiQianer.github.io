# 实数

# 实数系统基本定理

# 确界原理

非空有上界实数集必有上确界，非空有下界实数集必有下确界。

# 单调有界收敛定理

单调递增有上界数列必收敛，单调递减有下界数列必收敛。

# Cantor 区间套定理

设闭区间序列 {[an,bn]}n≥1\{[a_n,b_n]\}_{n \geq 1}{[an​,bn​]}n≥1​ 满足

[a1,b1]⊇[a2,b2]⊇⋯⊇[an,bn]⊇⋯[a_1,b_1] \supseteq [a_2, b_2] \supseteq \cdots \supseteq [a_n,b_n] \supseteq \cdots[a1​,b1​]⊇[a2​,b2​]⊇⋯⊇[an​,bn​]⊇⋯

且 lim⁡n→∞(bn−an)=0\lim_{n \to \infty} (b_n-a_n) = 0limn→∞​(bn​−an​)=0, 那么存在唯一的 x∈⋂n≥1[an,bn]x \in \bigcap_{n \geq 1}[a_n,b_n]x∈⋂n≥1​[an​,bn​].

# Bolzano-Weierstrass 定理

有界数列必有收敛子列。

# Cauchy 收敛定理

Cauchy 数列必收敛。

# Heine-Borel 有限覆盖定理

设 I\mathscr{I}I 是闭区间 [a,b][a,b][a,b] 的开覆盖，则存在有限个开区间 I1,…,Ik∈II_1, \dots, I_k \in \mathscr{I}I1​,…,Ik​∈I 满足 [a,b]⊆⋃1≤i≤kIi[a,b] \subseteq \bigcup_{1 \leq i \leq k}I_i[a,b]⊆⋃1≤i≤k​Ii​.

上述六定理等价。

# 极限

# 数列极限

# 定义

给定数列 {an}n∈N⊆R\{a_n\}_{n \in \mathbb{N}} \subseteq \mathbb{R}{an​}n∈N​⊆R, ∀ε>0\forall \varepsilon>0∀ε>0, 若 ∃N∈R\exists N \in \mathbb{R}∃N∈R 使得 ∀n>N\forall n>N∀n>N 均有

∣an−a∣<ε|a_n - a| < \varepsilon∣an​−a∣<ε

则称数列 {an}n=0∞\{a_n\}_{n=0}^\infty{an​}n=0∞​ 的极限是 aaa, 记作 lim⁡n→∞an=a\lim_{n \to \infty} a_n = alimn→∞​an​=a 或 an→aa_n \to aan​→a.

# 性质

数列极限的基本性质包括：

  1. 极限存在必唯一 (唯一性)
  2. 极限存在必有界 (有界性)
  3. 极限可以四则运算
  4. 夹逼定理
  5. 单调有界必有极限
  6. 数列的极限是其任意子数列的极限
  7. 任一有界数列中必存在一个趋于有限极限的数列 (闭区间套定理证明)

# 柯西收敛准则

数列 {an}n=0∞\{a_n\}_{n=0}^\infty{an​}n=0∞​ 趋于有限极限的充要条件是 ∀ε>0\forall \varepsilon>0∀ε>0, ∃N∈N\exists N \in \mathbb{N}∃N∈N 使得 ∃n,m>N\exists n,m > N∃n,m>N 时，恒有

∣xn−xm∣<ε|x_n-x_m|<\varepsilon∣xn​−xm​∣<ε

对于其余极限问题，柯西收敛准则类似。

# 函数极限

# 定义

函数的极限包括在某一有限值 aaa 和无穷远处的极限。

若函数 f(x)f(x)f(x) 在 aaa 的某一去心邻域内有意义，且 ∀ε>0\forall \varepsilon>0∀ε>0, ∃δ>0\exists \delta>0∃δ>0 使得 0<∣x−a∣<δ0<|x-a|<\delta0<∣x−a∣<δ 时，恒有

∣f(x)−A∣<ε|f(x)-A| < \varepsilon∣f(x)−A∣<ε

则称 xxx 趋于 aaa 时，f(x)f(x)f(x) 的极限是 AAA, 记作

lim⁡x→af(x)=a\lim_{x \to a}f(x) = ax→alim​f(x)=a

# 基本性质

函数极限的基本性质与数列类似，包括：

  1. 唯一性
  2. 局部有界性

# Heine 定理

给定函数 f:U∘(a,ρ)→Rf:U^\circ(a,\rho) \to \mathbb{R}f:U∘(a,ρ)→R 及 A∈RA \in \mathbb{R}A∈R, 则

Heine 定理是一个重要的定理，沟通了函数极限和数列极限。
