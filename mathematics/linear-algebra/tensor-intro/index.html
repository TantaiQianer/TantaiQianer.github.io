# 预备知识

很基础的问题，就是为什么要用张量。这里首先考虑三维欧式空间中的矢量和坐标系。

# 坐标系

三维欧式空间中的坐标系包括三个基，用 {e1,e2,e3}\{\boldsymbol{e}_1, \boldsymbol{e}_2, \boldsymbol{e}_3\}{e1​,e2​,e3​} 表示，称为坐标基矢。按照手的螺旋顺序从 e1\boldsymbol{e}_1e1​ 到 e2\boldsymbol{e}_2e2​ 得到 e3\boldsymbol{e}_3e3​ 方向，若右手自然，则称为右手系，反之称为左手系。

# 哑标和自由标

注意到，有些角标变量的引入仅仅为了描述方便，脚标变量的各取值在计算结果中占等价地位，例如 xi′=∑j=13aijxjx_i' = \sum_{j=1}^3 a_{ij}x_jxi′​=∑j=13​aij​xj​ 中的角标 jjj, 我们称这样的角标为哑标 (dummy index), 反之 iii 称为自由标 (free index).

基于哑标特点，Einstein 提出了 Einstein 求和约定，即省略求和号，对于同项的重复指标作求和处理。例如上式 xi′=∑j=13aijxjx_i' = \sum_{j=1}^3 a_{ij}x_jxi′​=∑j=13​aij​xj​ 可以简化为 xi′=aijxjx_i' = a_{ij}x_jxi′​=aij​xj​. 该求和约定在数学、物理学和 AI 领域都得到了广泛使用。下面我们的式子中也采用该约定。
