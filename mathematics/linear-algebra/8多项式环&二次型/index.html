# 基本概念

# 二次型

若域 F\mathbb{F}F 上一个 nnn 元二次齐次多项式

满足 A=A⊤A = A^\topA=A⊤, 则称其为域 F\mathbb{F}F 上的一个 nnn 元二次型，矩阵 AAA 称为二次型 X⊤AXX^\top A XX⊤AX 的矩阵。

# 线性替换

设 CCC 为可逆矩阵，则称 XXX 用 CYCYCY 代入是一个变量 x1,x2,…,xnx_1, x_2, \dots, x_nx1​,x2​,…,xn​ 到变量 y1,y2,…,yny_1, y_2, \dots, y_ny1​,y2​,…,yn​ 的一个非退化线性替换。

于是，nnn 元二次型 X⊤AXX^\top A XX⊤AX 经非退化线性替换 X=CYX = CYX=CY 变为

(CY)⊤A(CY)=Y⊤(C⊤AC)Y≜Y⊤BY(CY)^\top A (CY) = Y^\top (C^\top A C) Y \triangleq Y^\top B Y(CY)⊤A(CY)=Y⊤(C⊤AC)Y≜Y⊤BY

显然 B=C⊤ACB = C^\top A CB=C⊤AC 是对称矩阵，则 BBB 是二次型 Y⊤BYY^\top B YY⊤BY 的矩阵。

# 二次型的等价

# 定义

对域 F\mathbb{F}F 上的两个 nnn 元二次型 X⊤AXX^\top A XX⊤AX 和 Y⊤BYY^\top B YY⊤BY, 若存在非退化线性替换 X=CYX = CYX=CY 使得 X⊤AX=Y⊤BYX^\top AX = Y^\top BYX⊤AX=Y⊤BY, 则称二次型 X⊤AXX^\top A XX⊤AX 和 Y⊤BYY^\top B YY⊤BY 等价，记作 X⊤AX≅Y⊤BYX^\top A X \cong Y^\top B YX⊤AX≅Y⊤BY.

容易证明，二次型的等价是一个等价关系。

# 性质

域 F\mathbb{F}F 上 nnn 元二次型 X⊤AX≅Y⊤BY⟺A≃BX^\top AX \cong Y^\top BY \iff A \simeq BX⊤AX≅Y⊤BY⟺A≃B.
