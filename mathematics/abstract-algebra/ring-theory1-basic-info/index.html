# 基本概念

# 环的定义

一个环 (ring) 是一个具有两种二元运算（通常记为加法 +++ 和乘法）的非空集合 RRR, 满足如下条件：

  1. (R,+)(R, +)(R,+) 是一个阿贝尔群；
  2. 乘法结合律 (associate multiplication): (ab)c=a(bc),∀a,b,c∈R(ab)c = a(bc), \forall a,b,c \in R(ab)c=a(bc),∀a,b,c∈R;
  3. 左右分配律 (left and right distributive law): a(b+c)=ab+ac,(a+b)c=ac+bc,∀a,b,c∈Ra(b+c) = ab + ac, (a+b)c = ac + bc, \forall a,b,c \in Ra(b+c)=ab+ac,(a+b)c=ac+bc,∀a,b,c∈R.
    以下是非必要的条件：
  4. 乘法交换律 (commutative multiplication): ab=ba,∀a,b∈Rab = ba, \forall a,b \in Rab=ba,∀a,b∈R. 此时称环 RRR 是一个交换环 (commutative ring).
  5. 乘法单位元：若环 RRR 中存在元素 1R1_R1R​ 使得 1Ra=a1R=a,∀a∈R1_Ra = a1_R = a, \forall a \in R1R​a=a1R​=a,∀a∈R. 此时称环 RRR 是一个幺环 (ring with identity).

# 环的基本性质

这里说的性质主要是两种运算交互产生的性质，如果仅涉及一种运算，则可以在群的概念下研究。这些简单性质包括：

  1. 0a=a0=0,∀a∈R0a = a0 = 0, \forall a \in R0a=a0=0,∀a∈R;
  2. (−a)b=a(−b)=−ab,∀a,b∈R(-a)b = a(-b) = -ab, \forall a,b \in R(−a)b=a(−b)=−ab,∀a,b∈R;
  3. (−a)(−b)=ab,∀a,b∈R(-a)(-b) = ab, \forall a,b \in R(−a)(−b)=ab,∀a,b∈R;
  4. (na)b=a(nb)=n(ab),∀n∈Z,a,b∈R(na)b = a(nb) = n(ab), \forall n \in \mathbb{Z}, a,b \in R(na)b=a(nb)=n(ab),∀n∈Z,a,b∈R;
  5. (∑i=1nai)(∑j=1mbj)=∑i=1n∑j=1maibj,∀ai,bj∈R,n,m∈N(\sum_{i=1}^n a_i)(\sum_{j=1}^m b_j) = \sum_{i=1}^n \sum_{j=1}^m a_i b_j, \forall a_i, b_j \in R, n,m \in \mathbb{N}(∑i=1n​ai​)(∑j=1m​bj​)=∑i=1n​∑j=1m​ai​bj​,∀ai​,bj​∈R,n,m∈N.

# 整环 & 除环

# 零因子

若环 RRR 中存在非零元 a,ba,ba,b 满足 ab=0ab = 0ab=0, 则称 aaa 是一个左零因子 (left zero divisor), bbb 是一个右零因子 (right zero divisor), 两者统称零因子 (zero divisor). 一个双边零因子 (two-sided zero divisor) 是一个同时可作为左零因子和右零因子的元素。

如果一个环中没有零因子，那么该环成立消去律：

ab=acorba=ca⇒b=c.ab = ac \quad \mathrm{or} \quad ba=ca \quad \Rightarrow \quad b=c.ab=acorba=ca⇒b=c.

# 整环

一个具有幺元 1R≠01_R \neq 01R​=0, 且不存在零因子的交换环 RRR 称为一个整环 (integral domain).

# 可逆

若环 RRR 中存在 a,ba,ba,b 满足 ab=1Rab = 1_Rab=1R​, 则称 aaa 是 bbb 的左逆 (left inverse), bbb 是 aaa 的右逆 (right inverse), 两者统称可逆元 (invertible element).

# 除环

幺环中的每个非零元都存在可逆元。如果一个环中每个非零元都存在可逆元，则称该环为除环 (division ring).
