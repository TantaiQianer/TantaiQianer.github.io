# 性质

在这里，我们证明一个重要的性质：有限维线性赋范空间上的模等价。我们首先给出范数等价的定义。

设线性空间 XXX 上定义了两个范数 ∥⋅∥1,∥⋅∥2\|\cdot\|_1, \|\cdot\|_2∥⋅∥1​,∥⋅∥2​. ∥⋅∥2\|\cdot\|_2∥⋅∥2​ 比 ∥⋅∥1\|\cdot\|_1∥⋅∥1​ 强，当且仅当

∥xn∥2→0⇒∥xn∥1→0\|x_n\|_2 \to 0 \Rightarrow \|x_n\|_1 \to 0∥xn​∥2​→0⇒∥xn​∥1​→0

若 ∥⋅∥1\|\cdot\|_1∥⋅∥1​ 强于 ∥⋅∥2\|\cdot\|_2∥⋅∥2​ 且 ∥⋅∥2\|\cdot\|_2∥⋅∥2​ 强于 ∥⋅∥1\|\cdot\|_1∥⋅∥1​, 则称 ∥⋅∥1\|\cdot\|_1∥⋅∥1​ 与 ∥⋅∥2\|\cdot\|_2∥⋅∥2​ 等价。

换种说法，∥⋅∥1\|\cdot\|_1∥⋅∥1​ 与 ∥⋅∥2\|\cdot\|_2∥⋅∥2​ 等价，当且仅当 ∃m,M\exists m, M∃m,M 使得

m∥x∥1≤∥x∥2≤M∥x∥1,∀x∈Xm\|x\|_1 \leq \|x\|_2 \leq M\|x\|_1, \quad \forall x \in Xm∥x∥1​≤∥x∥2​≤M∥x∥1​,∀x∈X

显然，基于此方式定义的范数等价是一个等价关系。因此，我们可以转而证明所有范数和欧式范数 ∥⋅∥E\|\cdot\|_E∥⋅∥E​ 等价。

对 XXX 中的一组基 {ei}i=1n\{e_i\}_{i=1}^n{ei​}i=1n​, 设 x=∑i=1nxiei∀x∈Xx = \sum_{i=1}^n x_ie_i \; \forall x \in Xx=∑i=1n​xi​ei​∀x∈X. 那么

∥x∥=∥∑i=1nxiei∥=∑i=1n∣xi∣∥ei∥≤∑i=1nxi2∑i=1n∥ei∥2=M∥x∥E\|x\| = \left\|\sum_{i=1}^n x_ie_i \right\| = \sum_{i=1}^n |x_i| \|e_i\| \leq \sqrt{\sum_{i=1}^n x_i^2} \sqrt{\sum_{i=1}^n \|e_i\|^2} = M \|x\|_E∥x∥=∥∥∥∥∥∥​i=1∑n​xi​ei​∥∥∥∥∥∥​=i=1∑n​∣xi​∣∥ei​∥≤i=1∑n​xi2​​i=1∑n​∥ei​∥2​=M∥x∥E​

# 度量空间

设数域 K\mathbb{K}K 上的线性空间 XXX 上存在一个双变量实值函数 ρ(⋅,⋅):X×X→[0,+∞)\rho(\cdot, \cdot): X \times X \to [0, +\infty)ρ(⋅,⋅):X×X→[0,+∞) 满足

  1. ρ(x,y)≥0,∀x,y∈X\rho(x,y) \geq 0, \forall x,y \in Xρ(x,y)≥0,∀x,y∈X, 且 ρ(x,y)=0⟺x=y\rho(x,y) = 0 \iff x = yρ(x,y)=0⟺x=y
  2. ρ(x,y)=ρ(y,x)\rho(x,y) = \rho(y,x)ρ(x,y)=ρ(y,x)
  3. ρ(x,z)≤ρ(x,y)+ρ(y,z),∀x,y,z∈X\rho(x,z) \leq \rho(x,y) + \rho(y,z), \forall x,y,z \in Xρ(x,z)≤ρ(x,y)+ρ(y,z),∀x,y,z∈X
    则称 ρ\rhoρ 是 XXX 上的一个距离 (distance)， 称 (X,ρ)(X, \rho)(X,ρ) 是一个度量空间。

赋范空间 (X,∥⋅∥)(X, \|\cdot\|)(X,∥⋅∥) 可以通过

ρ(x,y)=∥x−y∥\rho(x,y) = \|x-y\|ρ(x,y)=∥x−y∥

诱导得到度量空间 (X,ρ)(X, \rho)(X,ρ). 这样通过范数诱导得到的度量空间具有平移不变性：

ρ(x+z,y+z)=ρ(x,y),∀x,y,z∈X\rho (x+z, y+z) = \rho(x,y), \quad \forall x,y,z \in Xρ(x+z,y+z)=ρ(x,y),∀x,y,z∈X

相反地，度量空间不一定是赋范空间。因为赋范空间的定义要求赋范空间定义于线性空间上，而度量空间不一定是线性空间。

# Cauchy 列

赋范空间 XXX 中的序列 {xn}n=1∞\{x_n\}_{n=1}^\infty{xn​}n=1∞​ 若满足：∀ε>0\forall \varepsilon > 0∀ε>0, ∃N∈N+\exists N \in \mathbb{N}_+∃N∈N+​ 使得 ∀m,n>N\forall m,n > N∀m,n>N, 有

∥xn−xm∥≤ε\|x_n - x_m\| \leq \varepsilon∥xn​−xm​∥≤ε

则称 {xn}n=1∞\{x_n\}_{n=1}^\infty{xn​}n=1∞​ 为柯西列 (Cauchy sequence), 也叫基本列。

# Banach 空间

对赋范空间 XXX, 若其中的 Cauchy 列均收敛，即 ∀{xn}n=1∞\forall \{x_n\}_{n=1}^\infty∀{xn​}n=1∞​ 为 Cauchy 列，都 ∃x∈X\exists x \in X∃x∈X 使得

lim⁡n→∞∥xn−x∥=0\lim_{n \to \infty}\|x_n-x\| = 0n→∞lim​∥xn​−x∥=0

则称赋范空间 (X,∥⋅∥)(X, \|\cdot\|)(X,∥⋅∥) 是完备赋范空间 (complete normed space), 又称巴拿赫空间 (Banach space) 或 B - 空间。

# Cauchy 列等价

(X,∥⋅∥)(X, \|\cdot\|)(X,∥⋅∥) 是线性赋范空间，ξ={xn}i=1∞,η={yn}i=1∞\xi = \{x_n\}_{i=1}^\infty,\; \eta = \{y_n\}_{i=1}^\inftyξ={xn​}i=1∞​,η={yn​}i=1∞​ 是其上的 Cauchy 列。若 ξ,η\xi, \etaξ,η 满足

∥xn−yn∥→0,n→∞\|x_n-y_n\| \to 0, \quad n \to \infty∥xn​−yn​∥→0,n→∞

则称 ξ\xiξ 与 η\etaη 等价，记作 ξ∼η\xi \sim \etaξ∼η.

Cauchy 列等价为等价关系的证明

lim⁡n→∞∥xn−xn∥=0\lim_{n \to \infty} \|x_n - x_n\| = 0n→∞lim​∥xn​−xn​∥=0

因此满足自反性。

若 ξ∼η\xi \sim \etaξ∼η, 则

lim⁡n→∞∥yn−xn∥=lim⁡n→∞∥xn−yn∥=0\lim_{n \to \infty} \|y_n - x_n\| = \lim_{n \to \infty} \|x_n - y_n\| = 0n→∞lim​∥yn​−xn​∥=n→∞lim​∥xn​−yn​∥=0

因此满足对称性。

设 ξ={xn}i=1∞\xi = \{x_n\}_{i=1}^\inftyξ={xn​}i=1∞​, η={yn}i=1∞\eta = \{y_n\}_{i=1}^\inftyη={yn​}i=1∞​, ζ={zn}i=1∞\zeta = \{z_n\}_{i=1}^\inftyζ={zn​}i=1∞​ 均为 Cauchy 列，且 ξ∼η\xi \sim \etaξ∼η, η∼ζ\eta \sim \zetaη∼ζ. 则基于三角不等式，有

lim⁡n→∞∥xn−zn∥≤lim⁡n→∞(∥xn−yn∥+∥yn−zn∥)=lim⁡n→∞∥xn−yn∥+lim⁡n→∞∥yn−zn∥=0+0=0\begin{aligned} \lim_{n \to \infty} \|x_n- z_n\| & \leq \lim_{n \to \infty} (\|x_n-y_n\| + \|y_n-z_n\|) \\ & = \lim_{n \to \infty} \|x_n-y_n\| + \lim_{n \to \infty}\|y_n-z_n\| \\ & = 0 + 0 = 0 \end{aligned}n→∞lim​∥xn​−zn​∥​≤n→∞lim​(∥xn​−yn​∥+∥yn​−zn​∥)=n→∞lim​∥xn​−yn​∥+n→∞lim​∥yn​−zn​∥=0+0=0​

因此满足传递性。

易于验证 Cauchy 列的等价关系满足自反、对称、传递性，因此上述定义的 Cauchy 列等价是等价关系。
