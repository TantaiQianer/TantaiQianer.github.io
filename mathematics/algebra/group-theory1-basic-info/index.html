# 代数系统 - 半群 - 幺半群 - 群：代数系统的完善化

# 代数系统

设非空集合 AAA 上有 f1,f2,…,fnf_1, f_2, \dots, f_nf1​,f2​,…,fn​ 共 nnn 个不同的运算，则称 AAA 在 nnn 个运算下构成一个代数系统 (algebraic system), 记作 ⟨A,f1,f2,…,fn⟩\lang A, f_1, f_2, \dots, f_n \rang⟨A,f1​,f2​,…,fn​⟩.

代数系统只规定了运算是封闭的，并没有规定各个运算的具体性质。所以代数系统是代数学中最笼统最没用的概念，其表明代数学研究的对象是集合上的一些运算及性质。按照从简单到复杂的规律，我们首先研究（具有某些性质的）仅含有一个元素的代数系统，即半群和群。

# 半群

# 概念

若代数系统 ⟨S,∘⟩\lang S, \circ \rang⟨S,∘⟩ 中运算 ∘\circ∘ 满足结合律，即 ∀x,y,z∈S\forall x,y,z \in S∀x,y,z∈S, 有

(x∘y)∘z=x∘(y∘z)(x \circ y) \circ z = x \circ (y \circ z)(x∘y)∘z=x∘(y∘z)

则称代数系统 ⟨S,∘⟩\lang S, \circ \rang⟨S,∘⟩ 为半群 (semigroup). 记号 ∘\circ∘ 通常叫做乘法，在不引起混淆的前提下，可以省略记号 ∘\circ∘, 并将半群 ⟨S,∘⟩\lang S, \circ \rang⟨S,∘⟩ 简记为 SSS. 对半群 ⟨S,∘⟩\lang S, \circ \rang⟨S,∘⟩, 若 ∃T⊆S\exists T \subseteq S∃T⊆S 使得 ⟨T,∘⟩\lang T, \circ \rang⟨T,∘⟩ 也为半群，则称 TTT 为 SSS 的子半群。

若半群的运算满足交换律，那么称该半群为可交换半群 (commutative semigroup). 对于可交换半群，我们有时采用 +++ 作为运算的符号。

下面是半群的一些例子：

  1. 对任意集合 AAA, ⟨P(A),∩⟩\lang P(A), \cap \rang⟨P(A),∩⟩, ⟨P(A),∪⟩\lang P(A), \cup \rang⟨P(A),∪⟩ 是半群。
  2. 由集合 AAA 到 AAA 的全体映射构成的集合 AAA^AAA, 在映射的复合运算下构成半群 ⟨AA,∘⟩\lang A^A, \circ \rang⟨AA,∘⟩.
  3. 全体偶数在乘法意义下构成半群 ⟨2Z,×⟩\lang 2\mathbb{Z}, \times \rang⟨2Z,×⟩.

# 半群的幂和指数律

在半群 SSS 中，若 A∈SA \in SA∈S 我们定义幂运算 ana^nan 为连续 nnn 个 aaa 相乘。那么，我们有以下指数律成立：

  1. aman=am+na^ma^n = a^{m+n}aman=am+n.
  2. (am)n=amn(a^m)^n = a^{mn}(am)n=amn.

对于可交换半群，我们还有：(ab)n=anbn(ab)^n = a^nb^n(ab)n=anbn.

# 幺半群

若半群 SSS 中存在单位元 e∈Se \in Se∈S 使得 ∀s∈S,es=se=s\forall s \in S, es = se = s∀s∈S,es=se=s. 则称 SSS 为幺半群 (monoid), 有时也称之为独异点。

# 群的基本概念

# 定义

若某幺半群的各元素均存在逆元，则其为群。也即对代数系统 ⟨G,∘⟩\lang G, \circ \rang⟨G,∘⟩, 我们称之为群 (group), 当其满足以下三条件：

  1. 结合律：∀a,b,c∈G,(a∘b)∘c=a∘(b∘c)\forall a,b,c \in G, (a \circ b) \circ c = a \circ (b \circ c)∀a,b,c∈G,(a∘b)∘c=a∘(b∘c).
  2. 存在单位元 (identity element) eee 使得 ∀a∈G,ea=ae=e\forall a \in G, ea = ae = e∀a∈G,ea=ae=e.
  3. ∀a∈G\forall a \in G∀a∈G, 存在逆元 (inverse element) a−1a^{-1}a−1 使得 aa−1=a−1a=eaa^{-1} = a^{-1}a = eaa−1=a−1a=e.

注意到，这三项条件不是相互独立的， 2 可以由 1 , 3 推出，但是由于使群的性质更清晰我们同样将 2 作为群定义的条件之一。

此外，还需要注意的是，在许多代数学书籍上，群的定义还包括一个条件：封闭性 (closure) : ∀a,b∈G,ab∈G\forall a,b \in G, ab \in G∀a,b∈G,ab∈G. 但我们的定义是基于代数系统进行的，代数系统天然是封闭的，因此在我们的定义中并不需要这一条件。

# 其它零碎的概念

# 交换群

如果群 GGG 的运算满足交换律，即 ∀a,b∈G\forall a,b \in G∀a,b∈G, ab=baab = baab=ba. 那么称这个群为交换群 (commutative group), 也叫阿贝尔群 (Abelian group).

# 阶

若群 GGG 中只含有有限个元素，称其为有限群，否则称其为无限群。对有限群 ⟨S,∘⟩\lang S, \circ \rang⟨S,∘⟩, 其元素个数称为 SSS 的阶；对无限群，我们规定其阶为 ∞\infty∞. 群 GGG 的阶采用 ∣G∣|G|∣G∣ 表示。

# 子群

对群 GGG, 若 ∃H⊆G\exists H \subseteq G∃H⊆G 使得 HHH 也为群，则称 HHH 为 GGG 的子群 (subgroup), 记作 H≤GH \leq GH≤G.

# 生成元

设 集合 XXX 是群 GGG 的子集，令 {Hi}\{H_i\}{Hi​} 是所有包含 XXX 的子集，则 ⋂iHi\bigcap_{i} H_i⋂i​Hi​ 称为群 GGG 由集合 XXX 生成的子群，记作 ⟨X⟩\langle X \rangle⟨X⟩, 并称 XXX 是子群 ⟨X⟩\langle X \rangle⟨X⟩ 的生成元 (generator).

# 同态

# 概念

若 GGG 和 HHH 是两个半群，若映射 f:G→Hf:G \to Hf:G→H 满足

f(ab)=f(a)f(b),∀a,b∈Gf(ab) = f(a)f(b), \quad \forall a,b \in Gf(ab)=f(a)f(b),∀a,b∈G

那么称 fff 是 GGG 到 HHH 的同态 (homomorphism), 并称 GGG 与 HHH 同态 (homomorphic), 记作 G∼HG \sim HG∼H.

进一步，有：

  • 若 fff 是单射 (injective), 则称 fff 是 GGG 到 HHH 的单同态 (monomorphism).
  • 若 fff 是满射 (surjective), 则称 fff 是 GGG 到 HHH 的满同态 (epimorphism).
  • 若 fff 是双射 (bijective), 则称 fff 是 GGG 到 HHH 的同构 (isomorphism), 记作 G≅HG \cong HG≅H.

# 置换

有限集 SSS 到自身的一一映射称为置换，一般用 σ\sigmaσ 表示。若 ∣S∣=n|S| = n∣S∣=n, 我们称这样的置换为
nnn 次置换。显然，在 nnn 元集合 SSS 中，共有
n!n!n! 个置换。置换可以采用下列符号表示：

σ=(12⋯nσ(1)σ(2)⋯σ(n))\sigma = \left(\begin{matrix} 1 & 2 & \cdots & n \\ \sigma(1) & \sigma(2) & \cdots & \sigma(n) \end{matrix}\right)σ=(1σ(1)​2σ(2)​⋯⋯​nσ(n)​)

它表示将 SSS 中的元素 1,2,⋯,n1,2,\cdots,n1,2,⋯,n 重新排列，使得 111 映射到 σ(1)\sigma(1)σ(1), 222 映射到 σ(2)\sigma(2)σ(2), 以此类推。

# 对称群

含有 nnn 个元素的集合 SSS 上的所有置换组成一个群，称为 SSS 的对称群 (symmetric group), 记作 SnS_nSn​.

对称群的某一子群由部分置换组成，称为置换群 (permutation group), 通常表示为 PnP_nPn​.
