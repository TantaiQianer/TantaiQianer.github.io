# nnn 元关系和二元关系

关系采用集合进行定义。一个 nnn 元关系 (relation) 是 AnA^nAn 的子集。我们称 a1,a2,…,ana_1, a_2, \dots, a_na1​,a2​,…,an​ 具有 nnn 元关系 RRR, 当且仅当 (a1,a2,…,an)∈R(a_1, a_2, \dots, a_n) \in R(a1​,a2​,…,an​)∈R, 记作 R(a1,a2,…,an)R(a_1, a_2, \dots, a_n)R(a1​,a2​,…,an​).

特别地，非空集合 AAA 上的一个二元关系 (binary relation) RRR 定义为 A×AA \times AA×A 的一个子集。若 (a1,a2)∈R(a_1, a_2) \in R(a1​,a2​)∈R, 那么称 a1,a2a_1, a_2a1​,a2​ 具有关系 RRR, 记作 R(a1,a2)R(a_1, a_2)R(a1​,a2​) 或 a1Ra2a_1 R a_2a1​Ra2​.

# 等价关系和等价类

# 等价关系

对于 AAA 上的二元关系 RRR, 若有

  1. 反身性 (reflexive), 即 aRa,∀a∈AaRa,\forall a \in AaRa,∀a∈A.
  2. 对称性 (symmetric), 即 a1Ra2⟺a2Ra1a_1Ra_2 \iff a_2Ra_1a1​Ra2​⟺a2​Ra1​.
  3. 传递性 (transitive), 即 a1Ra2,a2Ra3⇒a1Ra3a_1Ra_2,\, a_2Ra_3\,\Rightarrow a_1Ra_3a1​Ra2​,a2​Ra3​⇒a1​Ra3​.

那么我们称 RRR 是一个等价关系 (equivalence relation), 通常用 ∼\sim∼ 表示。此时 a1a_1a1​ 与 a2a_2a2​ 等价，记作 a1∼a2a_1 \sim a_2a1​∼a2​.

# 集合的划分

若对集合 AAA, 存在一系列集合（有限个或无限个） A1,A2,…,An,…A_1, A_2, \dots, A_n, \dotsA1​,A2​,…,An​,… 满足：

  1. Ai∩Aj=∅,∀i,j∈N+,i≠jA_i \cap A_j = \varnothing, \forall i, j \in \mathbb{N}_+, i \neq jAi​∩Aj​=∅,∀i,j∈N+​,i=j.
  2. ⋃i=1∞Ai=A\bigcup_{i=1}^\infty A_i = A⋃i=1∞​Ai​=A.

那么，称 {Ai∣i∈N+}\{A_i|i \in \mathbb{N}_+\}{Ai​∣i∈N+​} 是集合 AAA 的一个划分 (partition), 也称为 AAA 的一个商集 (quotient set).

# 等价类

若 AAA 上有等价关系 ∼\sim∼, 那么 ∀a∈A\forall a \in A∀a∈A, 称集合

aˉ≔{x∈S∣x∼a}\bar{a} \coloneqq \{x \in S|x \sim a\}aˉ:={x∈S∣x∼a}

是 aaa 的等价类，也记作 [a][a][a], 并称 aaa 是等价类 aˉ\bar{a}aˉ 的一个代表元。

关于等价类，有如下基本性质：

  1. x∼a⟺x∈aˉx \sim a \iff x \in \bar{a}x∼a⟺x∈aˉ.
  2. aˉ=bˉ⟺a∼b\bar{a} = \bar{b} \iff a \sim baˉ=bˉ⟺a∼b.
  3. 若 aˉ≠bˉ\bar{a} \neq \bar{b}aˉ=bˉ, 则 aˉ∩bˉ=∅\bar{a} \cap \bar{b} = \varnothingaˉ∩bˉ=∅.
  4. 若集合 AAA 上存在等价关系 ∼\sim∼, 则所有等价类组成的集合是 AAA 的一个划分。

# 偏序关系和对偶原理

# 偏序关系

对于 AAA 上的二元关系 RRR, 若有

  1. 反身性 (reflexive), 即 aRa,∀a∈AaRa,\forall a \in AaRa,∀a∈A.
  2. 反对称性 (antisymmetric), 即 a1Ra2,a2Ra1⇒a1=a2a_1Ra_2, a_2Ra_1 \Rightarrow a_1 = a_2a1​Ra2​,a2​Ra1​⇒a1​=a2​.
  3. 传递性 (transitive), 即 a1Ra2,a2Ra3⇒a1Ra3a_1Ra_2,\, a_2Ra_3\,\Rightarrow a_1Ra_3a1​Ra2​,a2​Ra3​⇒a1​Ra3​.

那么我们称 RRR 是一个偏序关系 (partial relation), 通常用 ⪯\preceq⪯ 表示。
