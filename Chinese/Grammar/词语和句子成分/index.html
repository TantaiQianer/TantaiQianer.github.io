词语和句子成分

# 概述

高考考纲的要求包括两部分：

  1. 现代汉语：词类及其语法功能，单句及其成分。
  2. 古代汉语：实词的词类活用，文言特殊句式。

在日常生活中，正确使用词语的要求包括两点：理解词义（多积累）和掌握用法（学习语法）。

汉语词语的分类包括：

  • 实词：名动形代数量
  • 虚词：副介连助叹拟

其中，实词和虚词的主要区别是，实词可以作为句子成分（主谓宾定状补）而虚词不可。

# 名词

# 分类

名词包括三类：

  1. 名称名词：即最普通的名词，不再单独解释
  2. 时间名词
  3. 方位名词

# 时间名词

  1. 例子：今天、明天、今年、去年、星期天、国庆节、早晨、中午等等。
  2. 特点：有明确的时间段
  3. 所作成分：
    • 可以作主语、宾语。例如：

      今天是星期一。

    • 可以作定语，例如：

      现在是星期一晚上。

所以 “立刻”、“马上” 等不属于时间名词，其不表示一个时间段，属于时间副词。

# 方位名词

  1. 例子：东西南北中、上下左右前后等等。
  2. 所作成分：
    • 可以作主语、宾语。例如：

      前面出现了敌人。

假如句子为：“在前面出现了敌人。” 则 “在前面” 作状语。

# 名词的语法功能

名词可以作主语、宾语、定语、谓语，不能作状语、补语。

下面是一些例子：

# 名词作定语

  1. 例子：

    这是水泥地板。

    数学老师。

  2. 特点：

    • 往往表性质：表性质的含义是不能由两个相同性质的词同时修饰。例如数学老师一定不是物理老师，“数学” 和 “物理” 不能同时作定语。
    • 名词性定语和中心语之间不加 “的”。

# 名词作谓语

  1. 例子：

    今天星期几了？今天星期一了。

    山海关，这号称天下第一关的山海关！（这句话是个单句，逗号前的 “山海关” 为主语，加粗部分为谓语。）

    刘备天下枭雄。——《赤壁之战》

  2. 特点：
    • 名词作谓语不接宾语。

# 定语

  1. 概念：名词或代词（要求代词执行名词功能）的修饰成分。
  2. 例子：

    美丽的校园中的同学，学习很刻苦。（“美丽的校园中” 为宏观定语，修饰同学；“美丽” 为微观定语，修饰 “校园”。）

    道之所存。（之用于定语和中心语之间，不译。）

    飞速前进的列车。（动词可以作定语）

  3. 总结：偏正结构词语，中心语为名词，则前为定语。

# 动词

关键词：动作行为、趋向、可能意愿。

# 分类

  • 一般动词
    • 及物动词
    • 不及物动词
  • 趋向动词
  • 能愿动词

# 及物动词与非及物动词

# 及物动词

  1. 概念：具备支配能力（作用）的动词，表现为后面接名词。
  2. 举例：

    调整物价。

    完成作业。

# 不及物动词

  1. 概念：不具备支配能力（作用）的动词，又称 “自动词”。
  2. 举例：

    咳嗽：我咳嗽了。我咳嗽他。

    汽车奔驰（于）马路。（因此，后接名词实际是省略了介词 “于”。）

    将军战（于）河北。

  3. 总结：能接宾语的只有及物动词。

# 趋向动词

趋，本义小步快走，在这里表现方向性。

  1. 举例：去、来、上来、下去、出来、进去。
  2. 概念：表示动作方向性的动词。

# 趋向动词的语法功能

可以作谓语、补语。

趋向动词在一般动词前作状语的频率极低，一般不考虑。趋向动词在前一般为连动结构，即与其后的动词并列作谓语。例如：

天线宝宝出来玩。

  1. 趋向动词作谓语，如：

    他上去了。

  2. 趋向动词作补语，如：

    他冲上去了。

    此处 “冲” 为中心语，“上去” 为补语。

  3. 总结：如果句中只有趋向动词，则它可以直接作谓语。如果趋向动词在一般动词后，则它作补语。

# 能愿动词

  1. 举例：能、能够、可以、会、愿意、应该、应当、要。
  2. 概念：表示可能意愿的动词。

# 能愿动词的语法功能

可以作谓语。

  1. 能愿动词单独作谓语，如：

    我愿意。

  2. 能愿动词与动词共同组成谓语，如：

    我愿意接受这项任务。

  3. 总结：如果句中只有能愿动词，则它可以直接作谓语。如果能愿动词在一般动词前，则一般动词与能愿动词构成能愿合成谓语。

# 动词的语法功能

可以作谓语、定语。
