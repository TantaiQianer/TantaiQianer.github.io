# 国内生产总值 (GDP)

# 含义

国内生产总值 (Gross Domestic Product, GDP) 是一定时期内在一国（或地区）境内生产的所有最终产品与服务的市场价值总和。

  • GDP 是一个市场价值概念。
  • GDP 是过程量。
  • GDP 计量的对象是最终产品。
  • GDP 按地域衡量。

# 衡量

GDP 通常有三种衡量方式，分别为增值法、收入法和支出法。

# 增值法

增值法通过加总经济中各个产业的产品和服务的增值来求得 GDP 的核算值。

# 收入法

收入法通过加总经济中各企业的要素收入求得 GDP 的核算值。

要素收入包括工资、租金、利息和利润。

因此，GDP = 工资 + 利息 + 利润 + 租金 + 间接税和企业转移支付 + 折旧。

# 支出法

支出法通过衡量在一定时期内整个社会购买最终产品和服务的总支出求得 GDP 的核算值。

为有效统计最终购买者，将一国经济对产品和服务需求的角度划分为四个部门。其对应支出类型如下：

  • 家庭部门：其对最终产品和服务的支出被称为消费支出，简称消费，记作 CCC.
  • 企业部门：其最终支出称为投资支出，简称投资，记作 III. 投资的特点是资本存量发生变更。总投资定义为净投资与折旧之和，净投资则定义为当年年终资本存量减去上年年终资本存量。
  • 政府部门：将各级政府购买产品和服务的支出定义为政府购买，记作 GGG.

    政府购买不同于政府支出。政府支出还包括社会福利支出，这类支出属于转移支付，不用于交换产品和服务，不计入 GDP 中。

  • 国际部门：国际部门对一国产品和服务的支出用净出口衡量。净出口定义为出口额与进口额的差值，一般用 XXX 表示出口，用 MMM 表示进口，用 NXNXNX 表示净出口。那么 NX=X−MNX = X-MNX=X−M. NX>0NX>0NX>0 称为贸易顺差，否则称为贸易逆差。

基于上述部门分类，支出法衡量的 GDP 可以表示为

GDP=C+I+G+NXGDP = C + I + G + NXGDP=C+I+G+NX

如果一个经济体是开放的，那么需要考虑国际部门。此时 GDP=C+I+G+NXGDP=C+I+G+NXGDP=C+I+G+NX. 如果一个经济体是封闭的，那么只需要考虑三部门 (家庭、企业和政府) 或两部门 (家庭和企业)。

# 名义 GDP 和实际 GDP

名义 GDP 是用生产产品和服务的当年价格计算的全部最终产品和服务的市场价值。实际 GDP 是选定某一时期作为基期，然后以基期价格核算出的某年所生产的全部最终产品和服务的市场价值。

相对于名义 GDP, 实际 GDP 可以排除不同时期下，价格因素变化带来的影响。

# 与 GDP 相关的其它指标

# 人均实际 GDP

人均实际 GDP 是一国某年的实际 GDP 与该国当年人口数之差。其既可以被用来纵向比较一国经济在不同时期的发展状况，又可以用来横向比较不同国家的生活水平差异。

# 国民生产总值 (GNP)

国民生产总值 (GNP), 是经济社会（一国或地区）成员在一定时期运用生产要素所生产的全部最终产品和服务的市场价值。

GNP 的核算遵循国民原则，即凡是本国国民所创造的收入，不管生产要素是否在国内，都被计入本国 GNP 中。

若采用公式表示 GNP 与 GDP 的关系，则 GNP = GDP + 本国公民在国外的要素收入额 - 国外公民在本国的要素收入额。

# 国民生产净值 (NNP)

国民生产净值 (NNP) 是国民生产总值减去在生产产品过程中消耗掉的资本量的价值。

# 国内生产净值 (NDP)

国内生产净值 (NDP) 是国内生产总值扣除折旧后的总额。

# 国民收入 (NI)

国民收入 (National Income, NI) 是一国全部生产要素在一定时期内提供服务所获得的报酬的总和，即工资、利息、租金和利润的总和。

# 投资 - 储蓄恒等式

# 两部门经济

根据 C+I=Y=C+SC+I = Y = C+SC+I=Y=C+S, 得到

I=SI=SI=S

称为两部门经济的投资 - 储蓄恒等式。

# 三部门经济

C+I+G=Y=C+S+TC+I+G = Y = C+S+TC+I+G=Y=C+S+T, 得到

I+G=S+TI+G = S+TI+G=S+T

或

I=S+(T−G)I = S + (T-G)I=S+(T−G)

其中 TTT 为政府净收入，GGG 为政府购买性支出。

# 价格水平及其衡量

# 价格指数

价格指数是衡量价格水平的主要指标。价格指数 (Price index, P) 是同一组产品和服务在某一年的费用额同它在某一设定的基准年度（基年）的费用额的比率。基年价格指数通常定为 100. 当年价格指数通过当年加权价格与基年加权价格作比值，再乘以基年价格指数得到。

常用的价格指数包括：GDP 平减指数和消费价格指数。

# GDP 平减指数

GDP 平减指数 (GDP Deflector) 定义为当期名义 GDP 与当期实际 GDP 比值乘 100.

# 消费价格指数

消费价格指数又称消费者物价指数 (Consumer Price Index, CPI), 是对城乡消费者购买的一揽子消费性产品和服务的加权平均市场价格的衡量。

我国的 CPI 一揽子商品和服务，涵盖了食品烟酒、衣着、居住、生活用品及服务、交通和通信、教育文化和娱乐、医疗保健、其他用品和服务等 8 大类、 262 个基本分类的商品与服务价格。

GDP 平减指数与 CPI 的区别

主要包括两点区别：

  1. 统计对象所属部门不同
    • GDP 平减指数衡量所有产品与服务的价格，统计对象涵盖家庭、企业、政府和国际部门。而 CPI 衡量城乡消费者主要消费产品与服务，统计对象仅包括家庭部门。
    • 因此，企业或政府购买的产品价格上升会显示在 GDP 平减指数上，但不会显示在 CPI 上。
  2. 统计地域范围不同
    • GDP 平减指数仅包括国内生产的产品，CPI 则包括国外生产，但在国内销售的产品。
    • 因此，日本制造并在美国销售的丰田汽车的价格上升会影响美国的 CPI, 但并不影响美国 GDP 平减指数。

# 通货膨胀

# 通货膨胀的含义

通货膨胀是指一个经济体在一定时期内价格水平持续地上升的情况。

# 失业及其衡量

# 人口的划分和劳动力的构成

美国人口的划分：

  • 劳动年龄人口：16 岁以上未服刑、未被医院或其他机构收留的人口总数
    • 劳动力人口：就业者与失业者数量之和
      • 就业人口：满足以下任一情况的人口总和
        • 至少有偿劳动 1 小时，或者为其家族企业无偿劳动多于 15 小时
        • 虽然不工作，但只是临时离开工作岗位或企业
      • 失业人口：调查前一周内没有工作，且有工作能力，调查前 4 周内正在努力寻找工作或等待重新获得原有就业岗位的人口总和
    • 非劳动力人口：主要涵盖在校学生以及退休人员等。
  • 非劳动年龄人口：年龄低于 16 岁无法工作或在专门机构生活而无法工作的人口总数。不属于就业人口，也不属于失业人口。

# 失业的衡量

# 失业率

失业率 (unemployment rate) 是失业人数占劳动力总数（劳动力人口）的百分比。就业率 (employment rete) 的定义类似。

劳动力参与率 (labor force participation rate) 是劳动力人口占劳动年龄人口的百分比。

# 失业的分类

  • 摩擦性失业 (frictional unemployment): 由于正常的劳动力周转所造成的失业。
  • 结构性失业 (structural unemployment): 由于技术或经济结构变化导致工作所需技能或工作地域变化所带来的失业。
  • 季节性失业 (seasonal unemployment): 由于季节性的气候变化所导致的失业。
  • 周期性失业 (cyclical unemployment): 又称需求不足性失业 (Demand-Deficient Unemployment), 指一个经济周期内随经济衰退而上升，随经济扩张而下降的波动性失业。

# 充分就业

当一个经济体内不存在周期性失业，所有失业都是摩擦性的、结构性的和季节性的，则该经济体便达到了充分就业 (full employment).

充分就业情况下的失业率被称为自然失业率 (natural rate of unemployment).
