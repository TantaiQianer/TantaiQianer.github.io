# 索洛模型

# 基本假设和结论

索洛增长模型 (Solow growth model) 又称新古典增长模型 (neoclassical growth model) 阐释了技术水平不变情况下经济增长的局限性观点。

模型的基本假设包括：

  1. 劳动力和技术水平保持不变。
  2. 两部门经济 (居民和生产者)：此时 Y=C+IY=C+IY=C+I.
  3. 生产函数规模收益不变：Y=F(K,L)⇒λY=F(λK,λL)Y=F(K, L) \Rightarrow \lambda Y = F(\lambda K, \lambda L)Y=F(K,L)⇒λY=F(λK,λL). 其中 YYY 为总产出，KKK 为资本总量，LLL 为劳动力总量。
  4. 储蓄函数：S=sYS = sYS=sY, 其中 sss 为储蓄率。

# 推导

取 λ=1/L\lambda = 1/Lλ=1/L, 则 Y/L=F(K/L,1)Y/L = F(K/L, 1)Y/L=F(K/L,1). 以 y=Y/Ly = Y/Ly=Y/L 记人均产出，k=K/Lk = K/Lk=K/L 记人均资本存量，那么

y=F(k,1)=f(k)y = F(k, 1) = f(k)y=F(k,1)=f(k)

即人均产出仅与人均资本有关。
