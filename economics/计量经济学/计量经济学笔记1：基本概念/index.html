# 计量经济学的分类

下面是一些关于计量经济学的分类概念的堆叠。关于计量经济学的分类，需要在学习过程中存在一个清晰的认识，这样才能抓住计量经济学的学习方向。

# 广义和狭义的计量经济学

广义计量经济学是利用经济理论、数学以及统计学定量研究经济现象的经济计量方法的统称，包括回归分析方法、投入产出分析方法、时间序列分析方法等。狭义计量经济学，也就是我们通常所说的计量经济学，以揭示经济现象中的因果关系为目的，在数学上主要应用回归分析方法。

# 初级、中级和高级计量经济学

初级以计量经济学的数理统计学基础知识和经典的线性单方程模型理论与方法为主要内容；中级以矩阵描述的经典的线性单方程模型理论与方法、经典的线性联立方程模型理论与方法，以及传统的应用模型为主要内容；高级以非经典的、现代的计量经济学模型理论、方法与应用为主要内容。

# 理论和应用计量经济学

理论计量经济学是以介绍、研究计量经济学的理论与方法为主要内容，侧重于理论与方法的数学证明与推导，与数理统计联系极为密切。除了介绍计量经济模型的数学理论基础、普遍应用的计量经济模型的参数估计方法与检验方法外，还研究特殊模型的估计方法与检验方法，应用了广泛的数学知识。应用计量经济学则以建立与应用计量经济学模型为主要内容，强调应用模型的经济学和经济统计学基础，侧重于建立与应用模型过程中实际问题的处理。

# 建立计量经济学模型的步骤和要点

下述分析过程以单方程计量经济学模型为对象，希望揭示变量间内在的因果关系。

# 理论模型的设计

理论模型的设计主要包括三项工作：

  1. 变量选择；
  2. 确定变量间的数学关系；
  3. 拟定模型方程中待估计参数的数值范围。

# 变量选择

变量选择过程需要符合客观经济规律，同时考虑数据的可得性，并要求各解释变量相互独立。

# 样本数据的收集

样本数据一般可以分为以下三类：

  1. 时间序列数据 (time series data): 一批按照时间先后排列的统计数据。
  2. 截面数据 (cross-sectional data): 发生在同一时间截面上的调查数据。
  3. 面板数据 (panel data): 在时间序列上取多个截面， 在这些截面上同时选取样本观测值所构成的样本数据，反映了空间和时间两个维度的经验信息。
