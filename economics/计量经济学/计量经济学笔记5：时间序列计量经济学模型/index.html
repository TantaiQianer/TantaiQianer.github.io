# 序列相关性

# 概念

一个简单的线性时间序列模型如下：

Yt=β0+β1Xt1+β2Xt2+⋯+βkXtk+μt,t=1,2,…,T.Y_t = \beta_0 + \beta_1 X_{t1} + \beta_2 X_{t2} + \cdots + \beta_k X_{tk} + \mu_t, t = 1,2,\dots,T.Yt​=β0​+β1​Xt1​+β2​Xt2​+⋯+βk​Xtk​+μt​,t=1,2,…,T.

此时，随机干扰项序列相关意味着

cov(μi,μj)=E(μiμj)≠0.\mathrm{cov}(\mu_i, \mu_j) = \mathbb{E}(\mu_i\mu_j) \neq 0.cov(μi​,μj​)=E(μi​μj​)=0.

一种特殊的序列相关是满足 ∣i−j∣=1|i-j| = 1∣i−j∣=1 的情况，即仅存在

E(μtμt+1)≠0,t=1,2,…,T−1,\mathbb{E}(\mu_t \mu_{t+1}) \neq 0, t = 1,2,\dots,T-1,E(μt​μt+1​)=0,t=1,2,…,T−1,

该情况称为一阶序列相关或自相关 (autocorrelation).
