# 基本概念

# 相关分析与回归分析

相关分析 (correlation analysis) 是研究两个或多个随机变量间相关关系的方法，各变量之间的地位等价。

回归分析 (regression analysis) 是研究一个变量关于另一个（些）变量的具体依赖关系的计算方法和理论。这里的前一个变量称为被解释变量 (explained variable) 或应变量 (dependent variable), 后一个变量称为解释变量 (explanatory variable) 或自变量 (independent variable).

需要注意的是：回归分析研究的是变量间的统计依赖关系，而不是因果相关关系。

# 总体回归和样本回归

在给定解释变量 XiX_iXi​ 条件下被解释变量 YiY_iYi​ 的期望轨迹称为总体回归线 (population regression line), 或更一般地称为总体回归曲线 (population regression curve). 相应的函数 E(Y∣Xi)=f(Xi)E(Y|X_i) = f(X_i)E(Y∣Xi​)=f(Xi​) 称为（双变量）总体回归函数 (population regression function, PRF).

总体回归描述的是大量样本在自变量取值条件下的因变量期望。如果希望估计某个样本的因变量取值，那么还需要加入随机扰动项 (stochastic disturbance), 来描述其余次要因素影响产生的偏差。

因此，对于系列样本 {(Xi,Yi)}\{(X_i,Y_i)\}{(Xi​,Yi​)}, 其拟合得到的回归曲线为样本回归线 (sample regression lines), 相应的函数 Y^i=f(Xi)=β0^+β1^Xi\hat{Y}_i = f(X_i) = \hat{\beta_0} + \hat{\beta_1}X_iY^i​=f(Xi​)=β0​^​+β1​^​Xi​ 称为样本回归函数 (sample regression function, SRF). 此时加入随机误差项的方程称为样本回归模型 (sample regression model).

# 参数估计

参数估计和假设检验是统计学中统计推断的两个基本方式。在计量经济学中，同样采用这两个基本方式对模型进行估计。

# 线性回归模型的基本假设

在计量经济学中，为保证参数估计量具有良好性质，通常对模型提出若干假设：

  1. 解释变量 XXX 是确定性变量，不是随机变量
  2. 随机误差项 μ\muμ 具有零均值、同方差和不序列相关性，即

E(μi)=0,i=1,2,…,n\mathbb{E}(\mu_i) = 0, \quad i=1,2,\dots,nE(μi​)=0,i=1,2,…,n

D(μi)=σμ2,i=1,2,…,n\mathbb{D}(\mu_i) = \sigma_\mu^2, \quad i=1,2,\dots,nD(μi​)=σμ2​,i=1,2,…,n

cov(μi,μj)=0,i≠j,i,j=1,2,…,n\text{cov}(\mu_i,\mu_j) = 0, \quad i \neq j, i,j =1,2,\dots,ncov(μi​,μj​)=0,i=j,i,j=1,2,…,n

  1. 随机误差项 μ\muμ 与解释变量 XXX 之间不相关，即

cov(μi,Xi)=0\text{cov}(\mu_i, X_i) = 0cov(μi​,Xi​)=0

  1. μ\muμ 服从零均值、同方差、零协方差的正态分布

# 统计检验

统计检验正要包括拟合优度检验、变量的显著性检验，及参数的区间估计。

# 拟合优度检验

注意到对总体平方和 TSS:=∑(Yi−Yˉ)2TSS:=\sum(Y_i-\bar Y)^2TSS:=∑(Yi​−Yˉ)2, 回归平方和 ESS=∑(Y^i−Yˉ)2ESS=\sum(\hat Y_i-\bar{Y})^2ESS=∑(Y^i​−Yˉ)2 及残差平方和 RSS=∑(Yi−Y^i)2RSS=\sum(Y_i-\hat Y_i)^2RSS=∑(Yi​−Y^i​)2, 有 TSS=ESS+RSSTSS=ESS+RSSTSS=ESS+RSS. 因此定义拟合优度为 ESS/TSSESS/TSSESS/TSS, 定义 (样本) 可决系数 (coefficient of determination) 为

R2:=ESSTSS=1−RSSTSSR^2 := \frac{ESS}{TSS} = 1-\frac{RSS}{TSS}R2:=TSSESS​=1−TSSRSS​

R2R^2R2 越接近 111, 说明实际观测点离样本线越近，拟合优度越高。显然，这里的 RRR 就是皮尔逊相关系数。
