# еҹәжң¬жҰӮеҝө

# зӣёе…іеҲҶжһҗдёҺеӣһеҪ’еҲҶжһҗ

зӣёе…іеҲҶжһҗ (correlation analysis) жҳҜз ”з©¶дёӨдёӘжҲ–еӨҡдёӘйҡҸжңәеҸҳйҮҸй—ҙзӣёе…іе…ізі»зҡ„ж–№жі•пјҢеҗ„еҸҳйҮҸд№Ӣй—ҙзҡ„ең°дҪҚзӯүд»·гҖӮ

еӣһеҪ’еҲҶжһҗ (regression analysis) жҳҜз ”з©¶дёҖдёӘеҸҳйҮҸе…ідәҺеҸҰдёҖдёӘпјҲдәӣпјүеҸҳйҮҸзҡ„е…·дҪ“дҫқиө–е…ізі»зҡ„и®Ўз®—ж–№жі•е’ҢзҗҶи®әгҖӮиҝҷйҮҢзҡ„еүҚдёҖдёӘеҸҳйҮҸз§°дёәиў«и§ЈйҮҠеҸҳйҮҸ (explained variable) жҲ–еә”еҸҳйҮҸ (dependent variable), еҗҺдёҖдёӘеҸҳйҮҸз§°дёәи§ЈйҮҠеҸҳйҮҸ (explanatory variable) жҲ–иҮӘеҸҳйҮҸ (independent variable).

йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҡеӣһеҪ’еҲҶжһҗз ”з©¶зҡ„жҳҜеҸҳйҮҸй—ҙзҡ„з»ҹи®Ўдҫқиө–е…ізі»пјҢиҖҢдёҚжҳҜеӣ жһңзӣёе…іе…ізі»гҖӮ

# жҖ»дҪ“еӣһеҪ’е’Ңж ·жң¬еӣһеҪ’

еңЁз»ҷе®ҡи§ЈйҮҠеҸҳйҮҸ XiX_iXiвҖӢ жқЎд»¶дёӢиў«и§ЈйҮҠеҸҳйҮҸ YiY_iYiвҖӢ зҡ„жңҹжңӣиҪЁиҝ№з§°дёәжҖ»дҪ“еӣһеҪ’зәҝ (population regression line), жҲ–жӣҙдёҖиҲ¬ең°з§°дёәжҖ»дҪ“еӣһеҪ’жӣІзәҝ (population regression curve). зӣёеә”зҡ„еҮҪж•° E(YвҲЈXi)=f(Xi)E(Y|X_i) = f(X_i)E(YвҲЈXiвҖӢ)=f(XiвҖӢ) з§°дёәпјҲеҸҢеҸҳйҮҸпјүжҖ»дҪ“еӣһеҪ’еҮҪж•° (population regression function, PRF).

жҖ»дҪ“еӣһеҪ’жҸҸиҝ°зҡ„жҳҜеӨ§йҮҸж ·жң¬еңЁиҮӘеҸҳйҮҸеҸ–еҖјжқЎд»¶дёӢзҡ„еӣ еҸҳйҮҸжңҹжңӣгҖӮеҰӮжһңеёҢжңӣдј°и®ЎжҹҗдёӘж ·жң¬зҡ„еӣ еҸҳйҮҸеҸ–еҖјпјҢйӮЈд№ҲиҝҳйңҖиҰҒеҠ е…ҘйҡҸжңәжү°еҠЁйЎ№ (stochastic disturbance), жқҘжҸҸиҝ°е…¶дҪҷж¬ЎиҰҒеӣ зҙ еҪұе“Қдә§з”ҹзҡ„еҒҸе·®гҖӮ

еӣ жӯӨпјҢеҜ№дәҺзі»еҲ—ж ·жң¬ {(Xi,Yi)}\{(X_i,Y_i)\}{(XiвҖӢ,YiвҖӢ)}, е…¶жӢҹеҗҲеҫ—еҲ°зҡ„еӣһеҪ’жӣІзәҝдёәж ·жң¬еӣһеҪ’зәҝ (sample regression lines), зӣёеә”зҡ„еҮҪж•° Y^i=f(Xi)=ОІ0^+ОІ1^Xi\hat{Y}_i = f(X_i) = \hat{\beta_0} + \hat{\beta_1}X_iY^iвҖӢ=f(XiвҖӢ)=ОІ0вҖӢ^вҖӢ+ОІ1вҖӢ^вҖӢXiвҖӢ з§°дёәж ·жң¬еӣһеҪ’еҮҪж•° (sample regression function, SRF). жӯӨж—¶еҠ е…ҘйҡҸжңәиҜҜе·®йЎ№зҡ„ж–№зЁӢз§°дёәж ·жң¬еӣһеҪ’жЁЎеһӢ (sample regression model).

# еҸӮж•°дј°и®Ў

еҸӮж•°дј°и®Ўе’ҢеҒҮи®ҫжЈҖйӘҢжҳҜз»ҹи®ЎеӯҰдёӯз»ҹи®ЎжҺЁж–ӯзҡ„дёӨдёӘеҹәжң¬ж–№ејҸгҖӮеңЁи®ЎйҮҸз»ҸжөҺеӯҰдёӯпјҢеҗҢж ·йҮҮз”ЁиҝҷдёӨдёӘеҹәжң¬ж–№ејҸеҜ№жЁЎеһӢиҝӣиЎҢдј°и®ЎгҖӮ

# зәҝжҖ§еӣһеҪ’жЁЎеһӢзҡ„еҹәжң¬еҒҮи®ҫ

еңЁи®ЎйҮҸз»ҸжөҺеӯҰдёӯпјҢдёәдҝқиҜҒеҸӮж•°дј°и®ЎйҮҸе…·жңүиүҜеҘҪжҖ§иҙЁпјҢйҖҡеёёеҜ№жЁЎеһӢжҸҗеҮәиӢҘе№ІеҒҮи®ҫпјҡ

  1. и§ЈйҮҠеҸҳйҮҸ XXX жҳҜзЎ®е®ҡжҖ§еҸҳйҮҸпјҢдёҚжҳҜйҡҸжңәеҸҳйҮҸ
  2. йҡҸжңәиҜҜе·®йЎ№ Ој\muОј е…·жңүйӣ¶еқҮеҖјгҖҒеҗҢж–№е·®е’ҢдёҚеәҸеҲ—зӣёе…іжҖ§пјҢеҚі

E(Ојi)=0,i=1,2,вҖҰ,n\mathbb{E}(\mu_i) = 0, \quad i=1,2,\dots,nE(ОјiвҖӢ)=0,i=1,2,вҖҰ,n

cov(Ојi,Ојj)=0,iвү j,i,j=1,2,вҖҰ,n\text{cov}(\mu_i,\mu_j) = 0, \quad i \neq j, i,j =1,2,\dots,ncov(ОјiвҖӢ,ОјjвҖӢ)=0,iоҖ =j,i,j=1,2,вҖҰ,n

  1. йҡҸжңәиҜҜе·®йЎ№ Ој\muОј дёҺи§ЈйҮҠеҸҳйҮҸ XXX д№Ӣй—ҙдёҚзӣёе…іпјҢеҚі

# з»ҹи®ЎжЈҖйӘҢ

з»ҹи®ЎжЈҖйӘҢжӯЈиҰҒеҢ…жӢ¬жӢҹеҗҲдјҳеәҰжЈҖйӘҢгҖҒеҸҳйҮҸзҡ„жҳҫи‘—жҖ§жЈҖйӘҢпјҢеҸҠеҸӮж•°зҡ„еҢәй—ҙдј°и®ЎгҖӮ

# жӢҹеҗҲдјҳеәҰжЈҖйӘҢ

жіЁж„ҸеҲ°еҜ№жҖ»дҪ“е№іж–№е’Ң TSS:=вҲ‘(YiвҲ’YЛү)2TSS:=\sum(Y_i-\bar Y)^2TSS:=вҲ‘(YiвҖӢвҲ’YЛү)2, еӣһеҪ’е№іж–№е’Ң ESS=вҲ‘(Y^iвҲ’YЛү)2ESS=\sum(\hat Y_i-\bar{Y})^2ESS=вҲ‘(Y^iвҖӢвҲ’YЛү)2 еҸҠж®Ӣе·®е№іж–№е’Ң RSS=вҲ‘(YiвҲ’Y^i)2RSS=\sum(Y_i-\hat Y_i)^2RSS=вҲ‘(YiвҖӢвҲ’Y^iвҖӢ)2, жңү TSS=ESS+RSSTSS=ESS+RSSTSS=ESS+RSS. еӣ жӯӨе®ҡд№үжӢҹеҗҲдјҳеәҰдёә ESS/TSSESS/TSSESS/TSS, е®ҡд№ү (ж ·жң¬) еҸҜеҶізі»ж•° (coefficient of determination) дёә
