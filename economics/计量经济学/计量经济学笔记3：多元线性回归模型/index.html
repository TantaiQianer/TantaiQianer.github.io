жң¬зҜҮз¬”и®°дёӯпјҢйҮҚзӮ№ж”ҫеңЁи®Ёи®әдёҺдёҖе…ғзәҝжҖ§еӣһеҪ’жЁЎеһӢдёӯдёҚеҗҢзҡ„йғЁеҲҶпјҢд»ҘеҸҠеңЁеҹәжң¬зҡ„ж•°зҗҶз»ҹи®ЎдёӯжңӘеј•е…Ҙзҡ„жҰӮеҝөгҖҒж–№жі•е’ҢжҖқжғігҖӮ

# жҰӮеҝө

еӨҡе…ғзҡ„жҖ»дҪ“зәҝжҖ§еӣһеҪ’жЁЎеһӢдёҖиҲ¬йҮҮз”ЁY=ОІX+ОјY = \beta X + \muY=ОІX+Ој иЎЁзӨәпјҢж ·жң¬зәҝжҖ§еӣһеҪ’жЁЎеһӢйҮҮз”ЁY=ОІ^X+eY = \hat{\beta}X + \boldsymbol{e}Y=ОІ^вҖӢX+e иЎЁзӨәпјҢе…¶дёӯe\boldsymbol{e}e иЎЁзӨәж®Ӣе·® (residuals). ж ·жң¬е®№йҮҸдёҖиҲ¬и®°дҪңnnn.

# ж ·жң¬е®№йҮҸй—®йўҳ

еӨҡе…ғеӣһеҪ’еҲҶжһҗзҡ„з¬¬дёҖдёӘй—®йўҳжҳҜж ·жң¬е®№йҮҸй—®йўҳгҖӮеҚіпјҢйңҖиҰҒе»әз«ӢдёҖдёӘеҸҜйқ зҡ„еӣһеҪ’жЁЎеһӢпјҢеӯҳеңЁж ·жң¬е®№йҮҸзҡ„дёӢйҷҗгҖӮ

еңЁж— еӨҡйҮҚе…ұзәҝжҖ§иҰҒжұӮдёӢпјҢж ·жң¬жңҖе°Ҹе®№йҮҸеҝ…йЎ»дёҚе°‘дәҺжЁЎеһӢдёӯи§ЈйҮҠеҸҳйҮҸ
зҡ„ж•°зӣ® (еҗ«еёёж•°йЎ№)гҖӮеҚіnвүҘk+1=rankXn \geq k+1=\mathrm{rank}\,XnвүҘk+1=rankX. еңЁз»ҹи®ЎжЈҖйӘҢзҡ„и§’еәҰпјҢn>30n>30n>30 ж—¶еҸҜд»Ҙеә”з”ЁZZZ жЈҖйӘҢпјӣnвҲ’kвүҘ8n-k \geq 8nвҲ’kвүҘ8 ж—¶пјҢttt еҲҶеёғиҫғзЁіе®ҡгҖӮ

# и°ғж•ҙзҡ„еҸҜеҶізі»ж•°

дёҖиҲ¬зҡ„еҸҜеҶізі»ж•°е®ҡд№үеҗҢдёҖе…ғзәҝжҖ§еӣһеҪ’жЁЎеһӢгҖӮдҪҶж №жҚ®з»ҸйӘҢеҸҜзҹҘпјҡеҰӮжһңеңЁжЁЎеһӢдёӯеўһеҠ дёҖдёӘи§ЈйҮҠеҸҳйҮҸпјҢR2R^2R2 еҫҖеҫҖеўһеӨ§гҖӮеӣ жӯӨпјҢеј•е…Ҙи°ғж•ҙзҡ„еҸҜеҶізі»ж•° (adjusted coefficient of determination)

RЛү2:=1вҲ’RSS/(nвҲ’kвҲ’1)TSS/(nвҲ’1)\bar{R}^2:=1-\frac{RSS/(n-k-1)}{TSS/(n-1)}RЛү2:=1вҲ’TSS/(nвҲ’1)RSS/(nвҲ’kвҲ’1)вҖӢ

д»ҘжҸҗеҮәеҸҳйҮҸдёӘж•°еҜ№жӢҹеҗҲдјҳеәҰдә§з”ҹзҡ„еҪұе“ҚгҖӮе…¶дёӯпјҢ(nвҲ’kвҲ’1)(n-k-1)(nвҲ’kвҲ’1) жҳҜж®Ӣе·®е№іж–№е’Ңзҡ„иҮӘз”ұеәҰпјҢ(nвҲ’1)(n-1)(nвҲ’1) жҳҜжҖ»дҪ“е№іж–№е’Ңзҡ„иҮӘз”ұеәҰгҖӮRЛү2\bar{R}^2RЛү2 зҡ„ж ҮеҮҶйҮҮз”Ё FFF жЈҖйӘҢзЎ®е®ҡгҖӮ

# иөӨжұ дҝЎжҒҜеҮҶеҲҷ

иөӨжұ дҝЎжҒҜеҮҶеҲҷ (Akaike information criterion, AIC) е®ҡд№ү

AIC:=lnвҒЎeвҠӨen+2(k+1)nAIC:=\ln\frac{\boldsymbol{e}^\top\boldsymbol{e}}{n} + \frac{2(k+1)}{n}AIC:=lnneвҠӨeвҖӢ+n2(k+1)вҖӢ

еҪ“дё”д»…еҪ“еўһеҠ и§ЈйҮҠеҸҳйҮҸеҸҜд»ҘдҪҝAICAICAIC еўһеҠ ж—¶пјҢжүҚиҖғиҷ‘еўһеҠ и§ЈйҮҠеҸҳйҮҸгҖӮ

# ж–Ҫз“ҰиҢЁеҮҶеҲҷ

ж–Ҫз“ҰиҢЁеҮҶеҲҷ (Schwarz criterion, SC) е®ҡд№ү

SC:=lnвҒЎeвҠӨen+knlnвҒЎnSC := \ln\frac{\boldsymbol{e}^\top\boldsymbol{e}}{n} + \frac{k}{n}\ln nSC:=lnneвҠӨeвҖӢ+nkвҖӢlnn

еҪ“дё”д»…еҪ“еўһеҠ и§ЈйҮҠеҸҳйҮҸеҸҜд»ҘдҪҝSCSCSC еўһеҠ ж—¶пјҢжүҚиҖғиҷ‘еўһеҠ и§ЈйҮҠеҸҳйҮҸгҖӮ
