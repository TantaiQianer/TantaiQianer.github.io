数学期望

定义

我们采用[数学期望]{.pink}([mathematical expectation]{.pink})来表示随机变量在所有取值可能下的加权平均值，定义为$\displaystyle 𝔼:=\sum{x}xp(x)$。更严谨地，我们可以写成$𝔼(x):= \int{-\infty}^{+\infty}x\mathrm dF(x)$，其中$F(x)$为$x$的分布函数。在连续的情况下，我们采用$\displaystyle 𝔼 = \int_{-\infty}^{+\infty}xp(x)\mathrm dx$进行计算。

性质

[性质1]{.label .warning}线性性

数学期望也可以推广到多变量的情况，我们以二元为例。

方差与协方差

定义

我们称$𝔻X:=𝔼(X-𝔼X)^2$是随机变量$X$的[方差]{.pink}([variance]{.pink})，称$\textbf{cov}(X,Y):= 𝔼\big((X-𝔼X)(Y-𝔼Y)\big)$为随机变量$X$与$Y$的[协方差]{.pink}([covariance]{.pink})。

性质

[性质1]{.label .warning}方差$𝔻X$非负。

[性质2]{.label .warning}$𝔻(aX + b) = a^2𝔻X$。

[性质3]{.label .warning}$𝔻(X±Y) = 𝔻X + 𝔻Y ± 2𝔼\big((X-𝔼X)(Y-𝔼Y)\big) = 𝔻X + 𝔻Y ± 2\textbf{cov}(X,Y)$。

[性质4]{.label .warning}若$c ≠ 𝔼X$，则$𝔻X < 𝔼(X-c)^2$。

这个性质说明随机变量和数学期望间的离散程度最小。

[例]{.label .success}求$\displaystyle 𝔻\big(\sum_{i=1}^na_iX_i + c\big)$。

[解]{.label .success}$\displaystyle 𝔻\big(\sum{i=1}^na_iX_i + c\big) = 𝔻\big(\sum{i=1}^naiX_i\big) = \sum{i=1}^nai^2𝔼(X_i - 𝔼X_i)^2 + 2a_ia_j\sum{1≤i<j≤n}𝔼(Xi-𝔼X_i)(X_j-𝔼X_j) = \sum{i=1}^nai^2𝔻X_i+ 2a_ia_j\sum{1≤i<j≤n}\textbf{cov}(X_i,X_j) = (a_1,\dots,a_n)Σ(a_1,\dots,a_n)’$。

注意到，$\displaystyle 𝔻\big(\sum{i=1}^na_iX_i\big)$可以写成一个关于$a_1,…,a_n$的一个半正定二次型，其对应的矩阵记为$𝛴$，我们将$𝔻X_i$记作$σ{ii}$，将$\textbf{cov}(Xi,X_j)$记作$σ{ij}$，于是$𝛴 = (\sigma_{ij})_n$，我们称之为[协方差矩阵]{.pink}([covariance matrix]{.pink})。于是，协方差矩阵是半正定的。

相关系数和相关性

我们称$\displaystyle ρ{XY} = \frac{\textbf{cov}(X,Y)}{\sqrt{𝔻X}\sqrt{𝔻Y}}$为$X$与$Y$的相关系数，它可以用来刻画$X$与$Y$的增长关系。若$\rho{XY}>0$，则$X$与$Y$[正相关]{.pink}；若$\rho{XY}<0$，则$X$与$Y$[负相关]{.pink}；若$\rho{XY}=0$，则$X$与$Y$[不相关]{.pink}。且$|ρ_{XY}|$越大，$X$与$Y$的相关程度越高。

Cauchy-Schwarz不等式

[定理]{.label .warning}对任意两个随机变量$X$与$Y$，有$(𝔼XY)^2 \leq 𝔼X^2 \cdot 𝔼Y^2$。等式成立当且仅当$P{Y = t_0X} = 1$，其中$t_0$是某个常数。

[解]{.label .success}构造二次函数$u(t) = 𝔼(tX - Y)^2 = t^2𝔼X^2 - 2t𝔼XY + 𝔼Y^2 \geq 0$，则判别式$\Delta =4(𝔼XY)^2 - 4𝔼X^2𝔼Y^2 = 0$，于是$(𝔼XY)^2 \leq 𝔼X^2 \cdot 𝔼Y^2$。存在重根$t_0$时，$𝔼(t_0X - Y)^2 = 0$，于是$𝔻(t_0X - Y) = 𝔼(t_0X - Y) = 0$，则$P{Y = t_0X} = 1$。充分性显然。

根据Cauchy-Schwarz不等式，我们容易得到$|\rho|\leq1$。若$\rho = 1$，则我们称随机变量$X$与$Y$[完全正相关]{.pink}，若$\rho = -1$，则我们称随机变量$X$与$Y$[完全负相关]{.pink}

[性质]{.label .success}对两个不相关的变量$X$与$Y$，我们容易证明其满足以下性质：

$X\sim N(0,1)$，$Y = X^2$，则$X$与$Y$不相关，但不独立。

矩母函数

定义

随机变量$X$的[矩母函数]{.pink}$\phi(t)$对所有值$t$定义为$\phi(t) = 𝔼e^{tX}$。即若$X$为离散随机变量，则$\phi(x) = \sum{x}e^{tx}p(x)$；若$X$为连续随机变量，则$\phi(x) = \int{-\infty}^{+\infty}e^{tx}f(x) \mathrm dx$。

性质

[性质]{.label .warning}$\phi(t)$的$n$阶导数在$t=0$处的值为$X$的$n$阶矩。即$\phi^{(n)}(0) = 𝔼X^n$，$n \geq 1$。

[证明]{.label .success}采用数学归纳法即可。

常见分布的矩母函数

二项分布

假设二项分布的量参数为$n$与$p$，则$\phi(t) = \sum_{k=0}^n e^{tk} \binom{n}{k}p^k(1-p)^{n-k} = (pe^t+1-p)^n$。

泊松分布

假设泊松分布的参数为$\lambda$，则$\phi(t) = \sum_{k=0}^\infty \frac{e^{tk}e^{-λ}}{k!}\lambda^k = \exp{λ(e^t-1)}$。
