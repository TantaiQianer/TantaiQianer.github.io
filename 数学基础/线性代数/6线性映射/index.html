# 基本概念和性质

# 定义

设 VVV 和 V′V'V′ 都是域 F\mathbb{F}F 上的线性空间，A\boldsymbol{A}A 是 VVV 到 V′V'V′ 的映射，若 A\boldsymbol{A}A 满足

  1. \boldsymbol{A}(\boldsymbol{\alpha}+\boldsymbol{\beta}) = \boldsymbol{A} \boldsymbol{\alpha} + \boldsymbol{A} \boldsymbol
  2. \boldsymbol{A}(k \boldsymbol{\alpha}) = k\boldsymbol

那么称 A\boldsymbol{A}A 是 VVV 到 V′V'V′ 的一个线性映射 (linear mapping). 若 V′=VV' = VV′=V, 则称线性映射 A\boldsymbol{A}A 是一个线性变换 (linear transformation).

# 性质

设 A:V→V′\boldsymbol{A}: V \to V'A:V→V′ 是线性映射，那么有如下基本性质：

  1. A0=0′\boldsymbol{A0} = \boldsymbol{0}'A0=0′.
  2. A(−α)=−Aα\boldsymbol{A}(-\alpha) = -\boldsymbol{A}\alphaA(−α)=−Aα.
  3. A(∑i=1nkiαi)=∑i=1nkiAαi\boldsymbol{A}(\sum_{i=1}^nk_i\alpha_i) = \sum_{i=1}^nk_i\boldsymbol{A}\alpha_iA(∑i=1n​ki​αi​)=∑i=1n​ki​Aαi​.
  4. 若 {αi}i=1n\{\alpha_i\}_{i=1}^n{αi​}i=1n​ 在 VVV 中线性相关，则 {Aαi}i=1n\{\boldsymbol{A}\alpha_i\}_{i=1}^n{Aαi​}i=1n​ 在 V′V'V′ 中线性相关。
  5. 设 dim⁡V=n\dim V = ndimV=n, 在 VVV 中取一个基 α1,α2,…,αn\alpha_1, \alpha_2, \dots, \alpha_nα1​,α2​,…,αn​. ∀α∈V\forall \alpha \in V∀α∈V, 设 α=∑i=1naiαi\alpha = \sum_{i=1}^na_i\alpha_iα=∑i=1n​ai​αi​, 则 Aα=∑i=1naiAαi\boldsymbol{A}\alpha = \sum_{i=1}^na_i\boldsymbol{A}\alpha_iAα=∑i=1n​ai​Aαi​. 从而 A\boldsymbol{A}A 被它在 VVV 上的一个基决定。
  6. A\boldsymbol{A}A 是 V→V′V \to V'V→V′ 的同构映射 ⟺A\iff \boldsymbol{A}⟺A 是 V→V′V \to V'V→V′ 的可逆线性映射。

# 运算

设 V,V′V, V'V,V′ 都是域 F\mathbb{F}F 上的线性空间。记 Hom(V,V′)\mathrm{Hom}(V, V')Hom(V,V′) 为 V→V′V \to V'V→V′ 的所有线性映射的集合。在 Hom(V,V′)\mathrm{Hom}(V, V')Hom(V,V′) 上，规定运算如下：

  • 规定加法：∀A,B∈Hom(V,V′),α∈V,(A+B)α=Aα+Bα\forall \boldsymbol{A}, \boldsymbol{B} \in \mathrm{Hom}(V, V'), \alpha \in V,\; (\boldsymbol{A}+\boldsymbol{B}) \alpha = \boldsymbol{A}\alpha + \boldsymbol{B}\alpha∀A,B∈Hom(V,V′),α∈V,(A+B)α=Aα+Bα.
  • 规定纯量乘法：∀k∈F,A∈Hom(V,V′),α∈V,(kA)α≔k(Aα)\forall k \in F, \boldsymbol{A} \in \mathrm{Hom}(V, V'), \alpha \in V,\; (k\boldsymbol{A})\alpha \coloneqq k(\boldsymbol{A}\alpha)∀k∈F,A∈Hom(V,V′),α∈V,(kA)α:=k(Aα).
  • 规定乘法（复合）：∀A,B,(BA)α=B(Aα)\forall \boldsymbol{A}, \boldsymbol{B}, (\boldsymbol{BA}) \alpha = \boldsymbol{B}(\boldsymbol{A}\alpha)∀A,B,(BA)α=B(Aα). 则线性映射的乘法满足结合律、左右分配律。

容易验证，Hom(V,V′)\mathrm{Hom}(V, V')Hom(V,V′) 是域 F\mathbb{F}F 上的一个有单位元的环。

# 幂等变换

投影是一种幂等变换。

# 定义

VVV 上的线性变换 A\boldsymbol{A}A 若满足 A2=A\boldsymbol{A}^2 = \boldsymbol{A}A2=A, 则称 A\boldsymbol{A}A 是 VVV 上的幂等变换。

注意，若 Al=A,l>2\boldsymbol{A}^l = A, l > 2Al=A,l>2, 则不一定是幂等变换。

# 线性映射的核

# 定义

设 A∈Hom(V,V′)\boldsymbol{A} \in \mathrm{Hom}(V, V')A∈Hom(V,V′), 则称 VVV 的子集 {α∈V∣Aα=0}\{\alpha \in V|\boldsymbol{A} \alpha = 0\}{α∈V∣Aα=0} 是 A\boldsymbol{A}A 的核 (kernel), 记作 Ker(A)\mathrm{Ker}(\boldsymbol{A})Ker(A).

# 性质

性质 1 Ker(A)\mathrm{Ker}(\boldsymbol{A})Ker(A) 是 VVV 的一个子空间。

性质 2 A\boldsymbol{A}A 是单射 ⟺Ker(A)\iff \mathrm{Ker}(\boldsymbol{A})⟺Ker(A) 是零子空间。

# 可对角化的线性变换

设 A\boldsymbol{A}A 是域 F\mathbb{F}F 上 nnn 维线性空间 VVV 上的一个线性变换。若 VVV 中有一个基 α1,α2,…,αn\alpha_1, \alpha_2, \dots, \alpha_nα1​,α2​,…,αn​ 使得 A\boldsymbol{A}A 在此基上的上的矩阵为对角矩阵，那么称 A\boldsymbol{A}A 可对角化。

# 线性变换的不变子空间

设 A\boldsymbol{A}A 是域 F\mathbb{F}F 上线性空间 VVV 的一个线性变换，WWW 是 VVV 的一个子空间，若从 α∈W\alpha \in Wα∈W 可推出 Aα∈W\boldsymbol{A}\alpha \in WAα∈W, 那么称 WWW 是 A\boldsymbol{A}A 的一个不变子空间 (invariant subspace). 显然，0,V0, V0,V 是 A\boldsymbol{A}A 的不变子空间，称它们是平凡的。

# 性质

命题 1 A\boldsymbol{A}A 的特征子空间、KerA\mathrm{Ker} AKerA, ImA\mathrm{Im} AImA 是 A\boldsymbol{A}A 的不变子空间。

证明略。

命题 2 设 A,B∈Hom(V,V)\boldsymbol{A}, \boldsymbol{B} \in \mathrm{Hom}(V, V)A,B∈Hom(V,V). 若 AB=BA\boldsymbol{AB} = \boldsymbol{BA}AB=BA, 则 KerB\mathrm{Ker}\boldsymbol{B}KerB, ImB\mathrm{Im}\boldsymbol{B}ImB 及 B\boldsymbol{B}B 的任一特征子空间都是 A\boldsymbol{A}A 的不变子空间。

该命题说明了非平凡的子空间应该如何寻找。

命题 3 A\boldsymbol{A}A 的两不变子空间的交与和仍然是 A\boldsymbol{A}A 的不变子空间。

# Hamilton-Cayley 定理

设 A∈Fn×nA \in F^{n \times n}A∈Fn×n, 则 AAA 的特征多项式 f(λ)f(\lambda)f(λ) 是 AAA 的一个零化多项式，即 f(A)=0f(A) = 0f(A)=0.

# 最小多项式

# 线性变换的最小多项式

定义 1 设 A\boldsymbol{A}A 是域 F\mathbb{F}F 上 nnn 维线性空间 VVV 上的一个线性变换，在 A\boldsymbol{A}A 的所有非零的零化多项式中，次数最低的首一多项式称为 A\boldsymbol{A}A 的最小多项式 (minimal polynomial).

命题 1 A\boldsymbol{A}A 的最小多项式唯一。

# 矩阵的最小多项式

定义 2 域 F\mathbb{F}F 上 nnn 级矩阵 AAA 的所有非零的零化多项式中，次数最低的首一多项式称为矩阵 AAA 的最小多项式 (minimal polynomial).

显然，若 A\boldsymbol{A}A 在 VVV 的一个基下的矩阵是 AAA, 则 A\boldsymbol{A}A 与 AAA 具有相同的最小多项式。

# 幂零变换

# 定义

若 ∃l∈N+\exists l \in \mathbb{N}_+∃l∈N+​ 使得 Al=0\boldsymbol{A}^l = \boldsymbol{0}Al=0, 则称 A\boldsymbol{A}A 是幂零变换 (nilpotent transformation). 使 Al=0\boldsymbol{A}^l = \boldsymbol{0}Al=0 成立的最小正整数 lll 称为 A\boldsymbol{A}A 的幂零指数 (nilpotent index).

# 性质

命题 4 设 A\boldsymbol{A}A 是幂零指数为 lll 的幂零变换，则 A\boldsymbol{A}A 的最小多项式 m(λ)=λlm(\lambda) = \lambda^lm(λ)=λl.

Bl−1ξ≠0,Blξ=0,\boldsymbol{B}^{l-1}\xi \neq \boldsymbol{0}, \; \boldsymbol{B}^l \xi = \boldsymbol{0},Bl−1ξ=0,Blξ=0,

dim⁡W0=dim⁡KerB=dim⁡W−rank(B).\dim W_0 = \dim \mathrm{Ker}\boldsymbol{B} = \dim W - \mathrm{rank}(\boldsymbol{B}).dimW0​=dimKerB=dimW−rank(B).
