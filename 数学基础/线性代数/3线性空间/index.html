# 线性空间的概念

# 数域

设 K\mathbb{K}K 是复数集 C\mathbb{C}C 的一个子集且至少有两个不同的元素，若 K\mathbb{K}K 中任意两个数的加法、减法、乘法及除法（除数不为零）仍属于 K\mathbb{K}K, 则称 K\mathbb{K}K 是一个数域。

下面是一些数域的例子：

  • 有理数域 Q\mathbb{Q}Q, 实数域 R\mathbb{R}R, 复数域 C\mathbb{C}C.
  • Q(2)≔{a+b2∣a,b∈R}\mathbb{Q}(\sqrt{2}) \coloneqq \{a+b\sqrt{2}|a,b \in \mathbb{R}\}Q(2​):={a+b2​∣a,b∈R}.

关于数域，有一个基本的性质：

任一数域必包含有理数域。

# 定义和例子

# 定义

设 VVV 是一个非空集合，FFF 是一个域。若存在一个 VVV 上的二元运算 +:V×V→V+ : V \times V \to V+:V×V→V （通常记作加法）和一个 FFF 和 VVV 上的运算 ⋅:F×V→V\cdot: F \times V \to V⋅:F×V→V （通常记作数乘，可以省略算符），两者共同满足下列运算规则：

  1. 加法交换律：α+β=β+α,∀α,β∈V\alpha + \beta = \beta + \alpha, \forall \alpha, \beta \in Vα+β=β+α,∀α,β∈V.
  2. 加法结合律：(α+β)+γ=α+(β+γ),∀α,β,γ∈V(\alpha + \beta) + \gamma = \alpha + (\beta + \gamma), \forall \alpha, \beta, \gamma \in V(α+β)+γ=α+(β+γ),∀α,β,γ∈V.
  3. ∃0\exists \boldsymbol{0}∃0 使得 α+0=α,∀α∈V\alpha + \boldsymbol{0} = \alpha, \forall \alpha \in Vα+0=α,∀α∈V.
  4. ∀α∈V\forall \alpha \in V∀α∈V, ∃β∈V\exists \beta \in V∃β∈V 满足 α+β=0\alpha + \beta = \boldsymbol{0}α+β=0.
  5. 1α=α1\alpha = \alpha1α=α.
  6. k(α+β)=kα+kβ,∀k∈K,α,β∈Vk(\alpha + \beta) = k\alpha + k\beta, \forall k \in \mathbb{K}, \alpha, \beta \in Vk(α+β)=kα+kβ,∀k∈K,α,β∈V.
  7. (k+l)α=kα+lα,∀k,l∈K,α∈V(k+l)\alpha = k\alpha + l\alpha, \forall k,l \in \mathbb{K}, \alpha \in V(k+l)α=kα+lα,∀k,l∈K,α∈V.
  8. k(lα)=(kl)α,∀k,l∈K,α∈Vk(l\alpha) = (kl)\alpha, \forall k,l \in \mathbb{K}, \alpha \in Vk(lα)=(kl)α,∀k,l∈K,α∈V.

则称 VVV 为域 FFF 上的一个线性空间 (linear space).

下面是线性空间的一些基本的例子：

# 线性空间的同构

# 定义

设 V,V′V, V'V,V′ 都是数域 KKK 上的线性空间，若存在双射 σ:V→V′\sigma: V \to V'σ:V→V′ 满足：

  1. σ\sigmaσ 保持加法：σ(α+β)=σα+σβ,∀α,β∈V\sigma(\alpha + \beta) = \sigma \alpha + \sigma \beta, \forall \alpha, \beta \in Vσ(α+β)=σα+σβ,∀α,β∈V.
  2. σ\sigmaσ 保持数乘：σ(kα)=k(σα),∀k∈K,α∈V\sigma(k\alpha) = k (\sigma \alpha), \forall k \in \mathbb{K}, \alpha \in Vσ(kα)=k(σα),∀k∈K,α∈V.

那么称 σ\sigmaσ 是 VVV 到 V′V'V′ 的一个同构映射 (isomorpic mapping). 此时称 VVV 与 V′V'V′ 同构 (isomorpic), 记作 V≅V′V \cong V'V≅V′.

# 性质

对有限维线性空间，同构的特征是线性空间的维度。即

定理 1 数域 KKK 上的两线性空间 V,V′V, V'V,V′ 同构 ⟺\iff⟺ dim⁡V=dim⁡V′\dim V = \dim V'dimV=dimV′.
