# 双线性函数

# 定义和例子

设 VVV 是域 F\mathbb{F}F 上的一个线性空间，V×VV \times VV×V 到 F\mathbb{F}F 的一个映射 fff 如果满足 ∀α1,α2,β1,β2,α,β∈V\forall \alpha_1, \alpha_2, \beta_1, \beta_2, \alpha, \beta \in V∀α1​,α2​,β1​,β2​,α,β∈V, k1,k2∈Fk_1, k_2 \in Fk1​,k2​∈F, 有

  1. f(k1α1+k2α2,β)=k1f(α1,β)+k2f(α2,β)f(k_1\alpha_1 + k_2\alpha_2, \beta) = k_1 f(\alpha_1, \beta) + k_2 f(\alpha_2, \beta)f(k1​α1​+k2​α2​,β)=k1​f(α1​,β)+k2​f(α2​,β)
  2. f(α,k1β1+k2β2)=k1f(α,β1)+k2f(α,β2)f(\alpha, k_1\beta_1 + k_2\beta_2) = k_1 f(\alpha, \beta_1) + k_2 f(\alpha, \beta_2)f(α,k1​β1​+k2​β2​)=k1​f(α,β1​)+k2​f(α,β2​)

那么称 fff 是 VVV 上的双线性函数，即 fff 对其两个自变量都满足线性性。

下面是双线性函数的几个例子：

  1. 欧几里得空间 Rn\mathbb{R}^nRn 的标准内积是双线性函数。
  2. 设 V=C[a,b]V = C[a,b]V=C[a,b], 对 ∀g,h∈C[a,b]\forall g,h \in C[a,b]∀g,h∈C[a,b], 令 f(g(x),h(x))=∫abg(x)h(x)dx\displaystyle f(g(x),h(x)) = \int_a^b g(x)h(x) \mathrm{d}xf(g(x),h(x))=∫ab​g(x)h(x)dx, 则 fff 是 C[a,b]C[a,b]C[a,b] 上的双线性函数，且是无限维的双线性函数。

# 度量矩阵

# 概念

当为线性空间确定一个基后，双线性函数变得可刻画了。下面，我们考虑有限维空间上的双线性函数。

设 VVV 是域 F\mathbb{F}F 上的一个 nnn 维线性空间，在 VVV 中取一组基 {αi}i=1n\{\alpha_i\}_{i=1}^n{αi​}i=1n​. x,y\boldsymbol{x}, \boldsymbol{y}x,y 在 VVV 的这组基下的坐标分别为

x=(x1,x2,…,xn)⊤,y=(y1,y2,…,yn)⊤\boldsymbol{x} = (x_1, x_2, \dots, x_n)^\top, \;\boldsymbol{y} = (y_1, y_2, \dots, y_n)^\topx=(x1​,x2​,…,xn​)⊤,y=(y1​,y2​,…,yn​)⊤

设 fff 是 VVV 上的双线性函数，那么

f(x,y)=f(∑i=1nxiαi,∑j=1nyjαj)=∑i=1n∑j=1nxiyjf(αi,αj)f(\boldsymbol{x}, \boldsymbol{y}) = f\left(\sum_{i=1}^nx_i\alpha_i, \sum_{j=1}^ny_j\alpha_j\right) = \sum_{i=1}^n\sum_{j=1}^nx_iy_jf(\alpha_i, \alpha_j)f(x,y)=f(i=1∑n​xi​αi​,j=1∑n​yj​αj​)=i=1∑n​j=1∑n​xi​yj​f(αi​,αj​)

令 A=(f(αi,αj))n×nA = \big(f(\alpha_i, \alpha_j)\big)_{n \times n}A=(f(αi​,αj​))n×n​. 那么 f(x,y)=x⊤Ayf(\boldsymbol{x}, \boldsymbol{y}) = \boldsymbol{x}^\top A\boldsymbol{y}f(x,y)=x⊤Ay.

因此，我们称 AAA 是双线性函数 fff 在基 {αi}i=1n\{\alpha_i\}_{i=1}^n{αi​}i=1n​ 下的度量矩阵。

# 不同基下的度量矩阵

设 fff 是域 F\mathbb{F}F 上 nnn 维线性空间 VVV 上的一个双线性函数，{αi}i=1n,{βi}i=1n\{\alpha_i\}_{i=1}^n,\, \{\beta_i\}_{i=1}^n{αi​}i=1n​,{βi​}i=1n​ 是 VVV 的两组基，且有过渡矩阵 PPP 使得

(β1,β2,…,βn)=(α1,α2,…,αn)P(\beta_1, \beta_2, \dots, \beta_n) = (\alpha_1, \alpha_2, \dots, \alpha_n) P(β1​,β2​,…,βn​)=(α1​,α2​,…,αn​)P

若 fff 在两基下的矩阵分别为 A,BA, BA,B. 那么，

B=P⊤APB = P^\top A PB=P⊤AP

换句话说，同一个双线性函数在不同基下的度量矩阵是合同的。

# 矩阵的合同

# 定义

设 A,B∈Mn(F)A, B \in \mathrm{M}_n(\mathbb{F})A,B∈Mn​(F), 若存在 C∈GLn(F)C \in \mathrm{GL}_n(\mathbb{F})C∈GLn​(F) 使得 B=C⊤ACB=C^\top A CB=C⊤AC, 则称 AAA 与 BBB 合同 (congruent), 记作 A≃BA \simeq BA≃B.

# 性质

  1. 若 A,B∈Mn(F)A, B \in \mathrm{M}_n(\mathbb{F})A,B∈Mn​(F), A≃BA \simeq BA≃B, 则 rank(A)=rank(B)\mathrm{rank}(A) = \mathrm{rank}(B)rank(A)=rank(B).

# 度量矩阵的秩

注意到，度量矩阵在不同基下的矩阵相互合同，因此不同基下的秩唯一，我们称这个秩为双线性函数的矩阵秩，记作 rankmf\mathrm{rank}_m frankm​f.

# 对称 (斜对称) 双线性函数

域 F\mathbb{F}F 上线性空间 VVV 上的双线性函数 fff 若满足

f(α,β)=f(β,α),∀α,β∈Vf(\alpha, \beta) = f(\beta, \alpha), \quad \forall \alpha, \beta \in Vf(α,β)=f(β,α),∀α,β∈V

则称 fff 是对称的 (symmetric). 若满足

f(α,β)=−f(β,α),∀α,β∈Vf(\alpha, \beta) = -f(\beta, \alpha), \quad \forall \alpha, \beta \in Vf(α,β)=−f(β,α),∀α,β∈V

则称 fff 是斜对称的 (skew-symmetric).

# 酉空间

# 基本概念和性质

# 内积

复数域上的线性空间 VVV 上的一个二元函数 (⋅,⋅)(\cdot, \cdot)(⋅,⋅) 称为 VVV 上的一个内积，当且仅当其满足

  1. 对 ∀α1,α2,β∈V,k1,k2∈C\forall \alpha_1, \alpha_2, \beta \in V, k_1, k_2 \in \mathbb{C}∀α1​,α2​,β∈V,k1​,k2​∈C, 有 (k1α1+k2α2,β)=k1(α1,β)+k2(α2,β)(k_1 \alpha_1 + k_2 \alpha_2, \beta) = k_1(\alpha_1, \beta) + k_2(\alpha_2, \beta)(k1​α1​+k2​α2​,β)=k1​(α1​,β)+k2​(α2​,β) (半线性)
  2. (α,β)=(β,α)‾(\alpha, \beta) = \overline{(\beta, \alpha)}(α,β)=(β,α)​ (Hermite 性)
  3. (α,α)≥0(\alpha, \alpha) \geq 0(α,α)≥0, 等号成立当且仅当 α=0\alpha = \boldsymbol{0}α=0.

# 酉空间

若复线性空间 VVV 上有一个内积，那么称 VVV 是复内积空间，又称酉空间 (unitary space).

# 长度

在酉空间 VVV 中，(α,α)\sqrt{(\alpha, \alpha)}(α,α)​ 称为 α\alphaα 的长度 (length), 记作 ∣α∣|\alpha|∣α∣ 或 ∥α∥\|\alpha\|∥α∥.

# Cauchy-Schwarz 不等式和夹角

在酉空间 VVV 中，对于任意向量 α,β\alpha, \betaα,β, 有

∣(α,β)∣≤∣α∣∣β∣|(\alpha, \beta)| \leq |\alpha| \, |\beta|∣(α,β)∣≤∣α∣∣β∣

等号成立当且仅当 α,β\alpha, \betaα,β 线性相关。该不等式称为柯西 - 施瓦茨不等式 (Cauchy-Schwarz inequailty).

基于 Cauchy-Schwarz 不等式，我们可以定义任意两个非零向量 α,β\alpha, \betaα,β 间的夹角 ⟨α,β⟩\lang\alpha, \beta\rang⟨α,β⟩:

⟨α,β⟩≔arccos⁡∣(α,β)∣∣α∣∣β∣\lang\alpha, \beta\rang \coloneqq \arccos\frac{|(\alpha, \beta)|}{|\alpha|\,|\beta|}⟨α,β⟩:=arccos∣α∣∣β∣∣(α,β)∣​

# 三角不等式

基于 Cauchy 不等式，我们可以得到三角不等式：

∣α+β∣≤∣α∣+∣β∣|\alpha + \beta| \leq |\alpha| + |\beta|∣α+β∣≤∣α∣+∣β∣

因此，范数天然就是一种度量。

# 酉变换

若酉空间 VVV 上到自身的满射 A\boldsymbol{A}A 保持内积不变，即

(Aα,Aβ)=(α,β),∀α,β∈V(\boldsymbol{A}\alpha, \boldsymbol{A}\beta) = (\alpha, \beta), \quad \forall \alpha, \beta \in V(Aα,Aβ)=(α,β),∀α,β∈V

那么称 A\boldsymbol{A}A 是 VVV 上的一个酉变换 (unitary transformation).

# 性质

  1. 酉空间上的酉变换一定是线性变换，且是单射，从而是可逆映射。
  2. 酉空间 VVV 上的酉变换 A\boldsymbol{A}A 是酉变换，当且仅当 A\boldsymbol{A}A 是 VVV 到自身的一个同构映射。
  3. 酉变换的复合和逆变换仍是酉变换。
  4. nnn 维酉空间 VVV 上的线性变换 A\boldsymbol{A}A 是酉变换 ⟺\iff⟺ A\boldsymbol{A}A 将标准正交基映射成标准正交基 ⟺\iff⟺ A\boldsymbol{A}A 在标准正交基下的矩阵是酉矩阵

# Hermite 变换

酉空间 VVV 上的一个变换 A\boldsymbol{A}A 若满足

(Aα,β)=(α,β)(\boldsymbol{A}\alpha, \beta) = (\alpha, \boldsymbol{\beta})(Aα,β)=(α,β)

那么称 A\boldsymbol{A}A 是 VVV 上的一个埃尔米特变换 (Hermite transformation), 又称自伴随变换。

# 性质

  1. 酉空间上的 Hermite 变换一定是线性变换。
  2. nnn 维酉空间 VVV 上的线性变换 A\boldsymbol{A}A 是 Hermite 变换当且仅当 A\boldsymbol{A}A 在 VVV 的任一标准正交基下的矩阵 AAA 满足 A†=AA^\dag = AA†=A. 其中 A†A^\dagA† (也记作 AHA^HAH 或 A∗A^*A∗) 表示 AAA 的复共轭。我们称满足上述条件的矩阵为埃尔米特矩阵 (Hermite matrix), 又称自伴矩阵。
