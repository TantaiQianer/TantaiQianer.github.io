# 复数

复数的最基本的表示为代数形式，即

z=a+bi,a,b∈Rz = a + bi, \quad a,b \in \mathbb{R}z=a+bi,a,b∈R

将a,ba,ba,b 分别视为二维坐标系中的横纵坐标，就得到了一个用来表示复数的平面，称为复平面。对于二维平面，除了采用平面直角坐标表示，另一种常见的方式就是采用极坐标进行表示。对应到复数上，就是复数的三角形式，即

z=r(cos⁡θ+isin⁡θ),r∈[0,∞),θ∈Rz = r (\cos \theta + i \sin \theta),\quad r \in [0,\infty), \, \theta \in \mathbb{R}z=r(cosθ+isinθ),r∈[0,∞),θ∈R

这时候，用三角恒等式可以得到一个三角形式下复数的乘法公式，称之为棣莫弗公式 (de Moivre's formula), 即

(cos⁡x+isin⁡x)n=cos⁡nx+isin⁡nx(\cos x + i\sin x)^n = \cos nx + i \sin nx(cosx+isinx)n=cosnx+isinnx

于是可以看出，复数相乘，则幅角相加。在实数范围内，对于此性质，我们求解函数方程可以得到其对应的是指数的形式。于是自然可以联想到复数可以采用指数进行表示。此时可以暂且引入一个复数的简化表示形式，称为复数的指数形式，即

z=r(cos⁡θ+isin⁡θ)≜reiθz = r (\cos \theta + i\sin \theta) \triangleq r e^{i\theta}z=r(cosθ+isinθ)≜reiθ

此外，借助于球极投影，我们还可以得到复数的球面表示。

对于单位球上的一点Z(ξ,η,ζ)Z(\xi, \eta, \zeta)Z(ξ,η,ζ), 容易得到其与复平面上点z(x,y)z(x,y)z(x,y) 之间的对应关系为

ξ=2x1+∣z∣2,η=2y1+∣z∣2,ζ=∣z∣2−11+∣z∣2\xi = \frac{2x}{1+|z|^2},\; \eta = \frac{2y}{1+|z|^2},\; \zeta = \frac{|z|^2-1}{1+|z|^2}ξ=1+∣z∣22x​,η=1+∣z∣22y​,ζ=1+∣z∣2∣z∣2−1​

以及

x=ξ1−ζ,y=η1−ζx = \frac{\xi}{1-\zeta},\; y = \frac{\eta}{1-\zeta}x=1−ζξ​,y=1−ζη​

我们称(ξ,η,ζ)(\xi,\eta,\zeta)(ξ,η,ζ) 这种表示为复数的球面表示。此时，点(0,0,1)(0,0,1)(0,0,1) 不表示复平面上的某个点，而是表示复平面上无穷远的点。因此，复数的球面表示可以让无穷远点也有一个良好的表示方式。此时，我们称包括了无穷远点的复数集为扩充复数集，记作C∞\mathbb{C}_\inftyC∞​ 或Cˉ\bar{\mathbb{C}}Cˉ. 在Cˉ\bar{\mathbb{C}}Cˉ 上也可以以球面上两点的欧氏距离定义度量，称为球面度量.
