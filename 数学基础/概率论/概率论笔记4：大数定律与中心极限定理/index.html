设 {Xn}\{X_n\}{Xn​} 为一随机变量序列，XXX 为一随机变量。若对 ∀ɛ>0\forall ɛ>0∀ɛ>0, 有

设随机变量 X,X1,X2,…X,X_1,X_2,\dotsX,X1​,X2​,… 的分布函数分别为 F(x),F1(x),F2(x),…F(x),F_1(x),F_2(x),\dotsF(x),F1​(x),F2​(x),…. 若对 F(x)F(x)F(x) 的任一连续点 xxx, 都有

# 大数定律

# 马尔可夫不等式与切比雪夫不等式

马尔可夫不等式和切比雪夫不等式是证明大数定律的基本定理。

对于任何随机变量 XXX 及 ε>0\varepsilon >0ε>0 有

P(∣X∣≥ε)≤E∣X∣εP(|X|\geq\varepsilon)\leq\frac{E|X|}{\varepsilon}P(∣X∣≥ε)≤εE∣X∣​

# 切比雪夫不等式

对于任何随机变量 XXX 及 ε>0\varepsilon >0ε>0, 有

P(∣X−μ∣≥ε)≤Var(X)ε2P(|X-\mu|\geq\varepsilon)\leq\frac{\text{Var}(X)}{\varepsilon^2}P(∣X−μ∣≥ε)≤ε2Var(X)​

# 服从大数定律

若一随机变量序列 {Xn}\{X_n\}{Xn​} 满足对任意的 ε>0\varepsilon > 0ε>0, 都有

lim⁡n→∞P(∣1n∑k=1nXk−1n∑k=1nEXk∣<ε)=1\lim_{n\to\infty}P\left(\left|\frac{1}{n}\sum_{k=1}^nX_k - \frac{1}{n}\sum_{k=1}^n\mathbb{E}X_k\right|<\varepsilon\right) = 1n→∞lim​P(∣∣∣∣∣∣​n1​k=1∑n​Xk​−n1​k=1∑n​EXk​∣∣∣∣∣∣​<ε)=1

那么我们称该随机变量序列服从大数定律。

不同的大数定律实际上给出的是随机变量服从大数定律的不同条件。

# 马尔可夫大数定律

我们称

1n2D(∑k=1nXk)→0\frac{1}{n^2}D(\sum_{k=1}^nX_k) \to 0n21​D(k=1∑n​Xk​)→0

为马尔可夫条件。该条件成立下，大数定律成立。这是判断大数定律是否成立的最常用的方法之一。

# 切比雪夫大数定律

设 X1,X2,…,Xn,…X_1,X_2,\dots,X_n,\dotsX1​,X2​,…,Xn​,… 是由两两不相关的随机变量所构成的序列，每一随机变量都有有限的方差，且方差具有公共上界 CCC. 即 ∃C∈R,s.t.∀i,Var(Xi)≤C\exists C \in \mathbb{R},\; s.t.\, \forall i,\; \mathrm{Var}(X_i) \leq C∃C∈R,s.t.∀i,Var(Xi​)≤C 则随机变量序列服从大数定律。称之为切比雪夫大数定律。

可以看出，切比雪夫大数定律是马尔可夫大数定律的特殊形式。

# 辛钦大数定律

设 {Xn}\{X_n\}{Xn​} 为一独立同分布的随机变量序列，若 XiX_iXi​ 的数学期望存在，则 {Xn}\{X_n\}{Xn​} 服从大数定律，称为辛钦大数定律。

相比于前面的弱大数定律，强大数定律说明

∣1n∑k=1nXk−1n∑k=1nEXk∣≥ε\left|\frac{1}{n}\sum_{k=1}^nX_k - \frac{1}{n}\sum_{k=1}^n\mathbb{E}X_k\right| \geq \varepsilon∣∣∣∣∣∣​n1​k=1∑n​Xk​−n1​k=1∑n​EXk​∣∣∣∣∣∣​≥ε

# 中心极限定理

# 研究对象

中心极限定理研究的是独立随机变量和的分布情况。即对于随机变量序列 {Xn}\{X_n\}{Xn​}, 讨论何种条件下，Y=∑i=1nXiY = \sum_{i=1}^n X_iY=∑i=1n​Xi​ 的分布会收敛域正态分布。一般情况下，nnn 是一个很大的数，可以看作 n→∞n \to \inftyn→∞.

林德伯格 - 莱维 (Lindeberg-Lévy) 中心极限定理是描述独立同分布变量的中心极限定理。

设 {Xn}\{X_n\}{Xn​} 是独立同分布的随机变量序列，且 E(X)=μE(X)=\muE(X)=μ, Var(X)=σ2\mathrm{Var}(X) = \sigma^2Var(X)=σ2 存在，若记

Yn∗≔∑i=1nXi−nμσnY_n^* \coloneqq \frac{\sum_{i=1}^nX_i - n\mu}{\sigma \sqrt{n}}Yn∗​:=σn​∑i=1n​Xi​−nμ​

# 棣莫弗 - 拉普拉斯中心极限定理

棣莫弗 - 拉普拉斯 (de Moivre-Laplace) 中心极限定理是描述二项分布的中心极限定理，也是最早被提出的中心极限定理。
