# Gibbs 现象

不连续函数点的傅里叶展开会出现振幅不收敛的情况，这种现象称为 Gibbs 现象。Gibbs 现象是小波变换产生的原因。

# 小波变换

傅里叶变换借助正弦波分析周期信号，而小波变换则不然。小波是一种能量在小段时域内特别集中的波，因而可以用来分析瞬时时变信号。

小波变换的展开形式为

f(t)=∑k∑jaj,kψj,k(t)f(t) = \sum_k\sum_j a_{j,k}\psi_{j,k}(t)f(t)=k∑​j∑​aj,k​ψj,k​(t)

其中 ψj,k(t)\psi_{j,k}(t)ψj,k​(t) 是小波级数。

如果一个函数 ψ∈L2(R)\psi \in L^2(\mathbb{R})ψ∈L2(R) 可以被用来定义希尔伯特空间内的正交基，那么称这个函数为一个规范正交小波 (orthonormal wavelet).

ψj,k(x)=2j/2ψ(2jx−k)\psi_{j,k}(x) = 2^{j/2}\psi(2^jx-k)ψj,k​(x)=2j/2ψ(2jx−k)
