# 实验 1 误差理论

# 问题 1

求方程x2+(α+β)x+109=0x^2+(\alpha + \beta)x + 10^9 = 0x2+(α+β)x+109=0 的根，其中α=−109,β=−1\alpha = -10^9, \beta=-1α=−109,β=−1，讨论如何设计计算格式才能有效减少误差，提高计算精度。

# 算法及分析

注意到，此处根α,β\alpha,\betaα,β 数量级相差巨大，造成方程x2+bx+c=0x^2+bx+c=0x2+bx+c=0 中∣b∣|b|∣b∣，∣c∣|c|∣c∣ 相差不大。于是在求根公式中计算b2−4c=(α+β)2−4αβ=(β−α)2b^2-4c = (\alpha + \beta)^2 - 4\alpha\beta = (\beta - \alpha)^2b2−4c=(α+β)2−4αβ=(β−α)2 易近似为α2\alpha^2α2，造成显著计算误差。因此可以直接计算较大的根α\alphaα，并采用分母有理化求绝对值较小的根β\betaβ，即

β=−b−b2−4c2=2cb2−4c−b\beta = \frac{-b - \sqrt{b^2-4c}}{2} = \frac{2c}{\sqrt{b^2-4c}-b}β=2−b−b2−4c​​=b2−4c​−b2c​

此外，也可以根据精确的α\alphaα 及韦达定理反解β\betaβ.

在程序编写中，注意到幂运算速度慢且结果不准确，我们采用 b*b 代替b2b^2b2 即 pow(b,2) 。
