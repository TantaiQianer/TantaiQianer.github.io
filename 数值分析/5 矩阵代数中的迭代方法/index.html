# 向量和矩阵范数

我们采用范数作为向量或矩阵间距离的衡量。

# 向量范数

向量范数 (norm) 是一个Rn→R\mathbb{R}^n \to \mathbb{R}Rn→R 的映射∥⋅∥\|\cdot\|∥⋅∥，满足以下的性质：

  1. 非负性: ∥x∥≥0\|\boldsymbol{x}\| \geq 0∥x∥≥0;
  2. 正定性: ∥x∥=0\|\boldsymbol{x}\| = 0∥x∥=0 当且仅当x=0\boldsymbol{x} = \boldsymbol{0}x=0;
  3. 齐次性: ∥αx∥=∣α∣∥x∥\|\alpha \boldsymbol{x}\| = |\alpha|\|\boldsymbol{x}\|∥αx∥=∣α∣∥x∥，其中α∈R\alpha \in \mathbb{R}α∈R 且x∈Rn\boldsymbol{x} \in \mathbb{R}^nx∈Rn;
  4. 三角不等式: ∥x+y∥≤∥x∥+∥y∥\|\boldsymbol{x} + \boldsymbol{y}\| \leq \|\boldsymbol{x}\| + \|\boldsymbol{y}\|∥x+y∥≤∥x∥+∥y∥，其中x,y∈Rn\boldsymbol{x}, \boldsymbol{y} \in \mathbb{R}^nx,y∈Rn.
  5. Cauchy-Bunyakovsky-Schwarz 不等式: x′y≤∥x∥2⋅∥y∥2\boldsymbol{x}'\boldsymbol{y} \leq \|\boldsymbol{x}\|_2\cdot\|\boldsymbol{y}\|_2x′y≤∥x∥2​⋅∥y∥2​，其中x,y∈Rn\boldsymbol{x}, \boldsymbol{y} \in \mathbb{R}^nx,y∈Rn.

在向量范数中，最常见的定义为 lpl_plp​ 范数。对于向量 x=(x1,x2,…,xn)′\boldsymbol{x} = (x_1,x_2,\dots,x_n)'x=(x1​,x2​,…,xn​)′, 其 lpl_plp​ 范数定义为

# 向量距离

两个向量 x,y\boldsymbol{x}, \boldsymbol{y}x,y 的距离定义为 ∥x−y∥\|\boldsymbol{x-y}\|∥x−y∥. 定义距离时，最常用的范数为 l2l_2l2​ 范数即欧氏距离，又称为欧几里得范数 (Euclidean norm).

# 矩阵范数

矩阵范数是一个 Rn×n→R\mathbb{R}^{n \times n} \to \mathbb{R}Rn×n→R 的映射。对所有的 A∈Rn×nA \in \mathbb{R}^{n \times n}A∈Rn×n 及 α∈R\alpha \in \mathbb{R}α∈R, 满足：

  1. 非负性: ∥A∥≥0\|A\| \geq 0∥A∥≥0;
  2. 正定性: ∥A∥=0\|A\| = 0∥A∥=0 当且仅当 A=OA = OA=O;
  3. 齐次性: ∥αA∥=∣α∣∥A∥\|\alpha A\| = |\alpha|\|A\|∥αA∥=∣α∣∥A∥, 其中 α∈R\alpha \in \mathbb{R}α∈R 且 A∈Rn×nA \in \mathbb{R}^{n \times n}A∈Rn×n;
  4. 三角不等式: ∥x+y∥≤∥x∥+∥y∥\|\boldsymbol{x} + \boldsymbol{y}\| \leq \|\boldsymbol{x}\| + \|\boldsymbol{y}\|∥x+y∥≤∥x∥+∥y∥, 其中 x,y∈Rn\boldsymbol{x}, \boldsymbol{y} \in \mathbb{R}^nx,y∈Rn.
  5. ∥AB∥≤∥A∥⋅∥B∥\|AB\| \leq \|A\|\cdot\|B\|∥AB∥≤∥A∥⋅∥B∥ (可以采用 Hölder 不等式证明).
