# 图灵机

# 定义和组成

我们称由以下四部分组成的一个机器为图灵机 (Turing Machine, TM)：

  1. 纸带：是无限长的分格纸带，用来存放输入或输出的字符；
  2. 读写头：可以在纸带上执行左移、右移、静止、读、写等操作；
  3. 状态寄存器：用来保存图灵机的当前状态；
  4. 控制规则表：根据当前的状态和读写头的内容决定新的状态，是一个转移函数，相当于程序。

# 运行过程

图灵机的运行包括初始化、运行和停机三个过程。初始化过程中，图灵机确定初始状态、读写头的初始位置和输入纸带的内容。运行状态下，图灵机根据当前的状态、输入和规则表，决定输出和新的状态。知道进入停机状态。

# 通用图灵机

通用图灵机 (Universal Turing Machine, UTM) 是一台可以将任意图灵机及其输入作为输入的图灵机。

实际上，操作系统可以看作是一台通用图灵机。

# 早期计算机代表

  1. 阿塔纳索夫 - 贝利计算机

    阿塔纳索夫 - 贝利计算机 (Atanasoff-Berry Computer, ABC) 是第一台电子计算机，其采用二进制运算，全部使用电子器件执行所有的计算，但没有 CPU，没有指令的概念，不可编程，不是图灵完备的。

  2. 埃尼阿克计算机

    埃尼阿克计算机 (Electronic Numerical Integrator And Computer, ENIAC)，是第一台通用 (general purpose) 的，通灵完备的电子计算机，算法硬化于机器中 (hard coded)，但可以通过重新连接线路修改算法。ENIAC 不是存储程序计算机。

  3. UNIVAC Ⅰ

    通用自动计算机 (UNIVAC Ⅰ) 是第一台商用电子计算机。

  4. EDVAC

    离散变量自动电子计算机 (Electronic Discrete Variable Automatic Computer, EDVAC) 是第一台采用冯诺依曼架构的存储程序计算机。

  5. EDSAC

    是世界上第一台实际运行的存储程序式电子计算机。

一些历史资料尚需考证。
