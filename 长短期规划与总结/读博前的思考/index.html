# 问题

目前的各方面能力还很欠缺，对于未来的规划还不清楚。

  • 科研向
    1. 对于要做的领域还不甚了解
    2. 数学物理基础不扎实
    3. 前沿论文研究不足
    4. 科研工具适用不好
      1. AI for paper
      2. paper 管理
      3. 科研作图
      4. 代码能力
  • 管理向
    1. 沟通能力差
    2. 与导师、师兄间交流仍不密切

# 计划

目前计划做的事情：

  • 科研向
    1. 广泛、有针对阅读 AI 主要会议的重点论文，包括经典论文、各年度重要会议的 best paper, 重要、经典的综述
    2. 科研工具的学习
      1. LaTeX
      2. PyTorch
      3. 文献整理
        1. Zotero
      4. 科研作图
        1. Tikz
        2. PPT
        3. Visio
        4. AI
  • 管理向
    1. 学习沟通技巧
    2. 多与导师、师兄沟通

# 总结

一些知乎的问题和帖子，可以拿来参考：

  • 总述
  • 研究生期间如何成为科研大佬？
  • 作为一个研究生，有哪些你直呼好用的科研神器？
  • 可不可以分享一个你知道的科研小技巧？
  • 你的科研能力从什么时候开始突飞猛进的？
  • 学科基础
  • 工程能力
    • 对自己深度学习方向的论文有 idea，可是工程实践能力跟不上，实验搞不定怎么办？
  • 沟通能力
