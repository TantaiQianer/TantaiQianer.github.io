# 熵

# 熵的定义

离散型随机变量 XXX 的熵 (entropy) 定义为

H(X):=−∑x∈Xp(x)log⁡p(x)H(X):=-\sum_{x\in\mathcal{X}}p(x)\log p(x)H(X):=−x∈X∑​p(x)logp(x)

其中对数函数 log⁡\loglog 的底为 222. 此外，由于 lim⁡x→0xlog⁡x=0\displaystyle \lim_{x\to0}x\log x = 0x→0lim​xlogx=0, 故约定 0log⁡0=00\log0=00log0=0, 即零概率项不影响熵的值。

注意到，可以采用随机变量 log⁡(1/p(x))\log(1/p(x))log(1/p(x)) 的数学期望表示熵，即

H(X)=Elog⁡1p(x)H(X) = \mathbb{E}\log\frac{1}{p(x)}H(X)=Elogp(x)1​

熵的单位为比特 (bit). 例如，抛掷均匀硬币的熵为 111 比特，随机选择一个单选题（四选项）的熵为 222 比特。

对于熵的底，如果采用底数 bbb, 则对应的熵记作 Hb(X)H_b(X)Hb​(X). 特别地，底数为 eee 时熵的单位为奈特 (nat).

# 熵的基本性质

  1. 非负性: H(X)≥0H(X) \geq 0H(X)≥0.

    根据 p(x)∈[0,1]p(x) \in [0,1]p(x)∈[0,1] 即证。

  2. Hb(X)=(log⁡ba)Ha(X)H_b(X) = (\log_b a) H_a(X)Hb​(X)=(logb​a)Ha​(X).

    根据换底公式即证。

既然可以将熵看作一个以密度函数为自变量的函数，那么基本的考虑是关注密度函数变化时，熵的变化情况。于是构造随机变量 XXX, 取 111 的概率为 ppp, 取 000 的概率为 1−p1-p1−p, 其中 p∈[0,1]p \in [0,1]p∈[0,1]. 那么

H(X)=−plog⁡p−(1−p)log⁡(1−p)≜H(p)H(X) = -p\log p - (1-p)\log (1-p) \triangleq H(p)H(X)=−plogp−(1−p)log(1−p)≜H(p)

通过计算容易得到 H(p)H(p)H(p) 是一个关于 p=1/2p=1/2p=1/2 对称的形似抛物线的凸函数。于是 p=1/2p=1/2p=1/2 时熵最大，为 111.

# 联合熵与条件熵

# 联合熵

联合熵是对两个随机变量的情形。

对于服从联合分布为 p(x,y)p(x,y)p(x,y) 的一对离散随机变量(X,Y)(X,Y)(X,Y), 其联合熵定义为

H(X,Y)≔−∑x∈X∑y∈Yp(x,y)log⁡p(x,y)H(X,Y) \coloneqq -\sum_{x\in\mathcal{X}}\sum_{y\in\mathcal{Y}}p(x,y)\log p(x,y)H(X,Y):=−x∈X∑​y∈Y∑​p(x,y)logp(x,y)

或表示为

H(X,Y)=−Elog⁡p(X,Y)H(X,Y) = -\mathbb{E}\log p(X,Y)H(X,Y)=−Elogp(X,Y)

# 条件熵

此外，还可以定义一个随机变量在另一个随机变量下的条件熵。

对于服从联合分布为 p(x,y)p(x,y)p(x,y) 的一对离散随机变量 (X,Y)(X,Y)(X,Y), 其条件熵定义为

H(Y∣X):=−∑x∈X∑y∈Yp(x,y)log⁡p(y∣x)=−Elog⁡p(Y∣X)H(Y|X) := -\sum_{x\in\mathcal{X}}\sum_{y\in\mathcal{Y}}p(x,y)\log p(y|x) = -\mathbb{E}\log p(Y|X)H(Y∣X):=−x∈X∑​y∈Y∑​p(x,y)logp(y∣x)=−Elogp(Y∣X)

对于联合熵与条件熵，我们可以将其天然地联系起来。即 H(X,Y)=H(X)+H(Y∣X)H(X,Y) = H(X) + H(Y|X)H(X,Y)=H(X)+H(Y∣X). 此定理称为链式法则。

# 互信息

互信息 (mutual information) 是一个随机变量中包含另一个随机变量信息量的度量。

若两随机变量 X,YX,YX,Y 的联合概率密度为 p(x,y)p(x,y)p(x,y), 边际概率密度分别为 p(x),p(y)p(x), p(y)p(x),p(y), 则其互信息为 p(x,y)p(x,y)p(x,y) 与 p(x)p(y)p(x)p(y)p(x)p(y) 之间的相对熵，即：

I(X;Y)≔∑x∈X∑y∈Yp(x,y)log⁡p(x,y)p(x)p(y)=DKL(p(x,y)∥p(x)p(y))=Ep(x,y)log⁡p(x,y)p(x)p(y)I(X;Y) \coloneqq \sum_{x \in \mathcal X}\sum_{y \in \mathcal Y}p(x,y)\log\frac{p(x,y)}{p(x)p(y)} = D_{\mathrm{KL}}(p(x,y)\|p(x)p(y)) = \mathbb{E}_{p(x,y)}\log\frac{p(x,y)}{p(x)p(y)}I(X;Y):=x∈X∑​y∈Y∑​p(x,y)logp(x)p(y)p(x,y)​=DKL​(p(x,y)∥p(x)p(y))=Ep(x,y)​logp(x)p(y)p(x,y)​

I(X;Y)=H(X)−H(X∣Y)I(X;Y) = H(X) - H(X|Y)I(X;Y)=H(X)−H(X∣Y)
