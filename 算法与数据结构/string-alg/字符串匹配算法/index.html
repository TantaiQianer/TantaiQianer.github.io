# 基本概念

字符串匹配 (string-matching) 问题又称字符串搜索 (string-searching) 问题或模式匹配 (pattern-matching) 问题。可以描述为在一个字符串 SSS 中寻找字符串 TTT 的问题。在字符串匹配问题中，我们称 SSS 为主串 (main string), 长度为 nnn, 而称子串 TTT 为模式串 (pattern string), 长度为 mmm. 此外，我们记字符串的字母表 (alphabet) 为 Σ\varSigmaΣ, 大小为 kkk.

# KMP 算法

KMP 算法的核心是 next 数组的计算。 next[i] 表示模式串前 i−1i-1i−1 (i≥1i \geq 1i≥1) 位中最大公共前后缀的长度加一。例如，对 ABABC , 有 next[4] = 2 , next[5] = 3 , next[2] = next[3] = next[6] = 1 . 此外，习惯设 next[1] = 0 .
