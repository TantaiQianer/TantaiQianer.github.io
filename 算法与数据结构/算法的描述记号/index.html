# 渐进记号的性质

对渐近记号，我们不加证明地说明几个性质。ΘΘΘ, OOO 和 ΩΩΩ 记号满足自反性。所有五个渐近记号都满足传递性。仅 Θ\ThetaΘ 记号满足对称性。此外，我们有 f(n)=O(g(n))⇔g(n)=Ω(f(n))f(n)=O(g(n)) \Leftrightarrow g(n) =Ω(f(n))f(n)=O(g(n))⇔g(n)=Ω(f(n)) 及 f(n)=o(g(n))⇔g(n)=ωf(n)f(n) = o(g(n)) \Leftrightarrow g(n) = ωf(n)f(n)=o(g(n))⇔g(n)=ωf(n), 我们称这个性质为转置传递性。
