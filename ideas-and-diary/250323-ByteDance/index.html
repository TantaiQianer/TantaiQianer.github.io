# 多模态

CLASI: 同传翻译。是一个 end-to-end 模型。同传包括几个问题：

  1. 听到内容不清晰情况下的鲁棒性
  2. 翻译时机的选择
  3. 借助外部信息的方式

PixelDance: 视频生成。挑战：

  1. 一致性
  2. 高动态
  3. 物理规则
  4. 指令遵循
