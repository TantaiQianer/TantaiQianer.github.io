# 部委生活是怎样的

  1. 站位高，接触信息广
  2. 工作实，形式主义少。原因包括三点：
    1. 太忙了
    2. 业务部门的性质决定的
    3. 平台大
    4. 退休之前弄个二巡问题不大
    5. 上级部门只有党中央和国务院，不需要做形式主义工作
  3. 生活朴，有基本保障
    1. 住的比 ruc 宿舍强
    2. 结婚前大概能保障到配租或者配售

下面是个例子：地理测绘的保密规定会影响地方发展。回应是，安全是发展的基础和前提。要考虑如何在安全的约束下进行发展，而不是将安全视为发展的阻碍。

“高技能劳动者的聚集具有正外部性。”

# 为什么选择部委

之前拿到过烟草总公司的 offer. 最后还是选择了部委。

房子车子总会有的，但是具有更高的上限。钱多了也没用。

# 需要做哪些努力

  1. 能力比履历更重要：体制化岗位下，不要代入经管类工作的求职思维。只需要 “跨过硬门槛，提高软实力”。培养自己的悟性和成长性。
  2. 以部委公务员的标准要求自己
    1. 坚持学习重大决策部署
      1. 重点学习涉及自己专业和意向岗位的内容
    2. 坚持以运动塑造外形
    3. 坚持真诚和感恩的心
      1. 做一个聪明的普通人。“聪明” 是有成长性，“普通人” 是踏踏实实，不好高骛远。
      2. 很多师弟师妹问完问题就完事了，这样再问就不愿意回答了
  3. 增强思维的开阔性和灵活性
    1. 一项政策的出台，
    2. 对考公的同学来说，知识面的开阔性，以及能否把多领域知识联系起来思考即灵活性，比深钻专业更重要
    3. 面试不会考察很专业的问题，但是经常会问关于社会现象的看法，从各种角度全面分析，是很出彩的答案
  4. 针对考试做提前准备
    1. 常识
    2. 数量资料
    3. 言语判断申论
    4. 面试
