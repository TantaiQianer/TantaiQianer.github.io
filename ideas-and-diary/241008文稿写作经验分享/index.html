# 分享 1—— 内参写作

商学院学姐的分享。

# 资政信息的结构

# 信息主题的结构

  1. 现状背景
  2. 困难挑战
  3. 对策建议

# 注意事项

  1. 逻辑一致性：现状背景、困难与政策建议的一一对应
  2. 行文简明扼要、语言精简凝练、表述清晰流畅，注意专有名词需要有简洁的解释
  3. 理论逻辑和数据支撑
  4. 深刻性与新颖性

# 现状背景

目的是清晰阐述对象的现实情况。

# 困难挑战

本质是指出当前国家或政府政策的效果不佳、政策失效的深层原因，以及未来政府实施调控政策的堵点、难点。

# 对策建议

对上文提及的现状背景、困难挑战提出对应的建议。此处需要注意观点和建议的新颖性，即站在不同角度思考，给出具有创新性但有理有据的新观点和建议。

# 摘要

概述自己的重要观点和重要建议，要求读完摘要就是读完全文的效果。

xxx 任务，为应对当前 xxx 的现状 / 困难 / 问题 / 挑战，应采取以下措施：第一... 第二... 第三...

# Q&A

  1. 内参和论文的理论有什么区别？

    理论是相通的但是面向的对象是非专业学者。需要让理论清晰简洁易懂。

  2. 如何培养 “给领导看” 的思维？如何落地？

    可以报名政策研究室的学生助理的。例如可以在对内参进行编校的过程中学习。有理有据即可。

  3. 如何进行用词的拿捏？如何学习这一点？

    推荐：淄博《致广大游客朋友的一封信》

# 分享 2—— 政治理论文章写作

马院科学社会主义与国际共产主义专业学长的分享。

# 建议

看 “三报一刊”。

# 流程

  1. 草稿起草
  2. 政研室老师对初稿进行修订
  3. 署名领导把关

# 准备工作

  1. 明确个人任务：服务于各级政府机关和学校，不要即兴发挥，要围绕对应单位的政策定位和风格进行起草。
  2. 明确文章立意：与负责理论宣传工作的老师进行沟通和请教，确保不要偏离。头够铁或者关系够硬可以直接问领导本人。

# 具体内容

下面是文章具体内容的书写。

  1. 首段：针对文章立意提出具体观点。以党和国家的重大会议和习总书记的重要讲话为基础。
  2. 具体阐释观点：
    1. 中间段落一般按照观点划分成三至四个部分。每一段的段首句都是一个子观点。
    2. 段落之间的关系可以是并列关系（政治经济文化生态）、递进（历史逻辑理论逻辑现实逻辑）。
    3. 中间的论证部分可以采用举例论证对比论证等方法。
  3. 总结段：总结和升华主题，回扣到伟大复兴等宏大叙事上。

可以总结形成自己的模板，以更好完成首段和结尾段。

# 分享 3—— 讲话稿写作

经济学院师兄的分享。

# 类型

  1. 讲话 / 致辞：领导受邀参加某活动的讲话 (1200-1800 字)
  2. 主旨发言 / 演讲：一般篇幅较长，内容更有深度（如中非论坛上的主旨讲话）
  3. 谈话参考
  4. 授课参考
  5. 贺信 / 唁电

# 致辞

写好致辞的两个关键：

  1. 语言：准确规范、简洁凝练。多用短句（好读防止领导断气...），注意语气。
  2. 逻辑：合理严密、紧扣主题。

# 文章的构思

学校要与 x 省省委组织部共同举行人才强国研讨会。

  1. 要写哪些内容？
    1. 人才强国的意义
    2. 介绍我校建设人才强国的努力
    3. 介绍 x 省建设人才强国的努力
    4. 介绍我校与 x 省的合作渊源
    5. 提出三点建议或思考（每篇致辞都需要包含）
  2. 如何将内容通过严谨逻辑联系起来？
    1. 寒暄
    2. 总书记的思想论述
    3. ...
  3. 每一部分内部怎么写？
    1. 第一部分（帽段）：开什么会，干什么，祝贺、欢迎、感谢
    2. 第二部分：介绍会议目的。要引用、梳理和阐释总书记关于会议主题的相关论述、思想和要求等。
    3. 第三部分：对于联合主办活动，出于礼貌先介绍对方单位的情况、相关举措和成果。
    4. 第四部分：介绍学校的情况和相关的举措、成果。注意不要堆砌，要结构化。
    5. 第五部分：双方合作。
    6. 第六部分：分享三点体会 / 要求。应该从总书记论述和理论文章中寻找答案。其中，提要求时需要注意：
      1. 可以直接提要求，也可以用论述 + 要求的方式
      2. 注意虚实结合，要扯点虚的但是也不能太虚
    7. 第七部分：期待。
    8. 第八部分：预祝活动圆满成功 + 身体健康、生活愉快

# QA

  1. 起草被老师改得面目全非的情况？

    很正常，洒洒水咯。总会遇到的。

  2. 改多少版可以定稿？

    看情况。如果与研究方向契合那可能两三遍就可以，不契合则可能五六遍。要多沟通。

  3. 内参能不能加引注？

    没必要哦。关键学术名词可以在页脚加脚注。

  4. 有没有写作资料的渠道推荐？

    《秘书工作手记》第二本是如何写公文。
    半月谈 公众号

  5. 老生常谈的内容如何写出新意？例如每年的合作。

    首先，领导也不知道去年讲的是什么。其次，可以选择不同的角度来写。最后，如果不能变可以在遣词造句上做编排。

    对于理论文章来说，新的讲话和文章会提出新的表述，可以依此进行更新和修改。
