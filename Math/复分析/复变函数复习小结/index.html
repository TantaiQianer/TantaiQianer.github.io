这是一个辅修期末考试复变函数复习前的简单笔记，涵盖内容不多，深度也较浅，仅仅作为一个简单的参考。

# 复数和复变函数

# 复数的基本概念和运算

复数的定义如同寻常。最基本的形式是代数形式，即 z=x+yiz = x+yiz=x+yi, 其中 x≜Rezx\triangleq\mathrm{Re}zx≜Rez, 称为实部，y≜Imzy\triangleq\mathrm{Im}zy≜Imz, 称为虚部。在补充规则 i2=−1i^2=-1i2=−1 的条件下，可以进行基本的四则运算。因此，复数域是可以简单看作实数域的扩张。

复数的另一个表示方式是采用几何形式。即将 z=x+yiz=x+yiz=x+yi 与平面上的点 (x,y)(x,y)(x,y) 作一一对应 (C→R2\mathbb{C} \to \mathbb{R}^2C→R2). 此时称平面为复平面。

在建立几何形式后，由于平面点存在极坐标表示，因此就对应着复数的一种新的表示方式，即三角形式。一般表示为

z=r(cos⁡θ+isin⁡θ)z=r(\cos \theta + i\sin \theta)z=r(cosθ+isinθ)

其中 θ\thetaθ 是 zzz 的一个幅角，其全体幅角为

Argz:={θ+2kπ,k∈Z}\mathrm{Arg} z:=\{\theta+2k\pi, k \in \mathbb{Z}\}Argz:={θ+2kπ,k∈Z}

幅角中位于 (−pi,π](-pi, \pi](−pi,π] 的称为幅角主值，记作 arg⁡z\arg zargz. 值得注意的是，z=0z=0z=0 的幅角无意义。

三角形式下的乘除计算简单，这主要归功于棣莫弗公式，即

(cos⁡θ+isin⁡θ)(cos⁡ϕ+isin⁡ϕ)=cos⁡(θ+ϕ)+isin⁡(θ+ϕ)(\cos \theta + i\sin \theta)(\cos \phi + i\sin \phi) = \cos(\theta+\phi) + i\sin(\theta + \phi)(cosθ+isinθ)(cosϕ+isinϕ)=cos(θ+ϕ)+isin(θ+ϕ)

这个公式也可以写成更好看的形式，不过究其根本也就是这样。

复数的开方具有多解。这不同于正实数的算术解规定，不再存在一个特殊的解作为算术平方根。

# 复变函数

# 基本概念

对定义域 E⊆CE\subseteq\mathbb{C}E⊆C, (单) 复变函数是一个映射 f:E→Cf: E \to \mathbb{C}f:E→C, 即函数是单值的。多复变函数是一个 E→2CE \to 2^\mathbb{C}E→2C 的映射，即函数是多值的。

如果采用代数形式看复变函数，那么其可以简单分解为两个实值函数的组合。即 w=f(z)=Ref(z)+Imf(z)=u(x,y)+iv(x,y)w=f(z) = \mathrm{Re}f(z) + \mathrm{Im}f(z) = u(x,y) + iv(x,y)w=f(z)=Ref(z)+Imf(z)=u(x,y)+iv(x,y).

我们可以从拓扑的视角看复变函数，因为复变函数是一个平面到平面的变换。

# 极限和连续性

由于复变函数可以看作两个实变函数的组合 (或许说成笛卡尔积也比较合适)。于是复变函数的极限存在当且仅当其实部和虚部都存在极限。同样地，复变函数连续当且仅当其实部和虚部在相同区间内都对应连续。
