# 金融系统

金融系统可以从两个角度进行分析：

  1. 制度角度 (institutional perspective): 金融系统包括市场、市场中介、金融工具和客户等。

  2. 功能角度 (functional perspective): 金融系统有助于在不确定的环境中跨时空分配和部署经济资源。

金融系统的制度是随时间不断变化的，但其变化是适应于功能需求的。

# 金融系统的功能

金融系统包括六大功能：

  1. 跨时间和空间转移经济资源，例如贷款、保险年金及全球金融市场中的机构投资等。
  2. 实现付款结算促进交易，其手段包括货币、支票、ATM 卡、按揭贷款 (mortgage) 和按揭利息的支付
  3. 集中资源承接不可分割的大型企业，其手段包括股票和基金等
  4. 协调信息，促进信息流动，价格发现、股票指数变动、资产收益率和波动率是媒介。
  5. 应对因信息不对称、激励不匹配造成的代理问题 (agency problems)、道德风险 (moral hazard)、逆向选择 (adverse selection)。(Ref)
  6. 管理风险，包括捆绑、汇集和分散风险等。体现为保险公司、抵押担保债券 (collateralized mortgage obligation, CMO)、债务担保凭证 (collateralized debt obligation, CDO) 等。

# 金融系统的机构组成

  1. 金融市场 (Financial markets)，如股权、固定收益、信贷及衍生品等。
  2. 金融中介机构 (Financial intermediaries)，如银行、保险公司、养老基金、共同基金 (mutual fund)、投资银行、
  3. 金融基础设施和监管 (Financial infrastructure and regulation)
  4. 政府和准政府组织 (Governmental and quasi-governmental organizations)，如中央银行、国际清算银行、国际货币基金组织、世界银行等。

金融机构在不同国家的功能可能不完全相同，这由当地法律法规所决定。
