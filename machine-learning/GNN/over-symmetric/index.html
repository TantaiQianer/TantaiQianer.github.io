# toy model about over-symmetry

如果平面单位圆上存在点 x1=(−1,0),x2=(1,0)\boldsymbol{x}_1 = (-1,0), \boldsymbol{x}_2 = (1,0)x1​=(−1,0),x2​=(1,0). 圆心有节点 x3=(0,0)\boldsymbol{x}_3 = (0,0)x3​=(0,0).

这时候，如果对于系统绕中心点 x3\boldsymbol{x}_3x3​ 作 θ\thetaθ 角旋转，得到 x1′=(−cos⁡θ,−sin⁡θ),x2′=(cos⁡θ,sin⁡θ)\boldsymbol{x}_1' = (-\cos \theta, -\sin \theta), \boldsymbol{x}_2' = (\cos \theta, \sin \theta)x1′​=(−cosθ,−sinθ),x2′​=(cosθ,sinθ). 那么，基于 EGNN 的更新公式

x3l+1=x3l+∑i=12xi3l⋅ϕ(mi,3l)=x3l.\boldsymbol{x}_3^{l+1} = \boldsymbol{x}_3^l + \sum_{i=1}^2 \boldsymbol{x}_{i3}^l \cdot \phi(\boldsymbol{m}_{i,3}^l) = \boldsymbol{x}_3^l.x3l+1​=x3l​+i=1∑2​xi3l​⋅ϕ(mi,3l​)=x3l​.

但是，若更改输入的 x\boldsymbol{x}x 为 x⊗x\boldsymbol{x} \otimes \boldsymbol{x}x⊗x. 则 xi′⊗xi′=(cos⁡2θ,cos⁡θsin⁡θ,cos⁡θsin⁡θ,sin⁡2θ)⊤,i=1,2\boldsymbol{x}_i' \otimes \boldsymbol{x}_i' = (\cos^2\theta, \cos\theta\sin\theta, \cos\theta\sin\theta, \sin^2\theta)^\top, i = 1,2xi′​⊗xi′​=(cos2θ,cosθsinθ,cosθsinθ,sin2θ)⊤,i=1,2. 输入 EGNN, 则 x3\boldsymbol{x}_3x3​ 处的特征是等变的。这里推测是 type-2 的特征起到了作用。

一般地，假设 X⃗∈R3×n\vec{\boldsymbol{X}} \in \mathbb{R}^{3 \times n}X∈R3×n, R\mathcal{R}R 是一种非平凡变换，对应矩阵 RRR. 则

Rϕ(X⃗)=ϕ(RX⃗)=ϕ(X⃗).R \phi(\vec{\boldsymbol{X}}) = \phi(R \vec{\boldsymbol{X}}) = \phi(\vec{\boldsymbol{X}}).Rϕ(X)=ϕ(RX)=ϕ(X).

注意到 R∈O(3)R \in \mathrm{O}(3)R∈O(3), 则 (R−I)ϕ(X⃗)=0(R-I) \phi(\vec{\boldsymbol{X}}) = 0(R−I)ϕ(X)=0, 从而 ϕ(X⃗)=0\phi(\vec{\boldsymbol{X}}) = 0ϕ(X)=0. 这就是 over-symmetry.

# over-symmetry in crystal

晶体输入暂不考虑连边，只考虑三维点云坐标 X⃗∈R3×n\vec{\boldsymbol{X}} \in \mathbb{R}^{3 \times n}X∈R3×n. 若 RRR 是一个晶体对称群内的操作（的矩阵表示），则有

RX⃗=X⃗PR \vec{\boldsymbol{X}} = \vec{\boldsymbol{X}} PRX=XP

其中 PPP 是置换矩阵。

现在在外面套一层 GNN, 那么有

ϕ(RX⃗)=ϕ(X⃗P)=ϕ(X⃗).\phi(R \vec{\boldsymbol{X}}) = \phi(\vec{\boldsymbol{X}} P) = \phi(\vec{\boldsymbol{X}}).ϕ(RX)=ϕ(XP)=ϕ(X).
