# Tackling Over-Smoothing for General Graph Convolutional Networks

# Contributions

  • 研究了多种深度 GCN (generic GCN, GCN with bias, ResGCN, APPNP) 输出的渐进行为。并从理论上证明了这些 GCN 会收敛于一个长方体。
  • 提出了一种在训练过程中均匀、随机丢弃部分边的技术 DropEdge, 并证明了该方法可以缓解过平滑问题。
