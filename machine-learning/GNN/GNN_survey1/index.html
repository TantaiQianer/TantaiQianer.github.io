# 简介

Graph neural networks: A review of methods and applications 是一篇国人发表的在 GNN 领域颇具影响力的综述文章，也是我阅读的第一篇关于 GNN 的综述文章，在这里简单记录一下。

# 作者简介

文章的作者来自清华、北邮和腾讯三个机构。

文章的第一作者周杰曾任清华大学自动化系主任，目前是清华大学研究生院院长，研究方向主要为计算机视觉、生物识别，Google Scholar 引用 30000+.

文章的通讯作者刘知远是清华大学 NLP 实验室副教授，Google Scholar 引用 30000+. 他还在知乎上非常活跃，贡献了许多优质答案。

# 主要内容

文章的主要内容包括三大部分：

  • 图神经网络模型回顾
    • 图神经网络的设计流程
    • 图神经网络各模块的变种
  • 图神经网络的应用场景分类
  • 图神经网络研究中的开放问题
