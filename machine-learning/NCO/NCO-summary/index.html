# 组合优化问题

组合优化问题 (Combinatorial Optimization Problem, COP) 是在离散空间中寻找最优解的问题，是数学优化 (mathematical optimization) 的重要分支。两种经典的 CO 问题是旅行商问题 (Travelling Salesman Problem, TSP) 和车辆路径问题 (Vechile Routing Problem, VRP).

# 旅行商问题

旅行商问题 (Travelling Salesman Problem, TSP) 即如何在地图上寻找一个包含所有点的最短回路。该问题的输入是一些离散的点的坐标，输出是一个包含所有点的最短回路。TSP 问题是一个 NP 完全问题。

# VRP

车辆路径问题 (Vechile Routing Problem, VRP) 即如何有效分配一组车辆访问多个目标点，并最小化行驶总距离成本。
