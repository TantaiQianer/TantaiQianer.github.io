# Feature Selection

出选择

  • Wrapper method: 前向选择和后向选择
  • Filter method: 计算特征与预测值之间的相关性，可以采用相关系数，KL 散度等方式刻画
  • Embedded method: 加入正则项
    • L1 范数更容易得到稀疏解

了解概念：Underfitting, Overfitting, generalization error
