# 矩阵近似计算

矩阵近似计算的一个方法是矩阵分解。即构造

M′=CB∈Rm×n,\bm{M}' = \bm{CB} \in \mathbb{R}^{m \times n},M′=CB∈Rm×n,

使得 ∥M−M′∥F\|\bm{M} - \bm{M}'\|_F∥M−M′∥F​ 最小，且 rank(M′)≤r\mathrm{rank}(\bm{M}') \leq rrank(M′)≤r.

实现该矩阵分解的方式就是 SVD 分解。

# 次梯度

对于凸函数 f:D→Rf:\mathcal{D} \to \mathbb{R}f:D→R, 其中 D\mathcal{D}D 是定义在 ddd 维欧氏空间上的凸集，若对 ∀y∈D\forall y \in \mathcal{D}∀y∈D 满足

f(y)≥f(x)+⟨g,y−x⟩,f(\bm{y}) \geq f(\bm{x}) + \langle\bm{g}, \bm{y} - \bm{x}\rangle,f(y)≥f(x)+⟨g,y−x⟩,

则称 g\bm{g}g 是 fff 在 x\bm{x}x 处的次梯度 (subgradient).

# 次梯度的性质

假设 y′\bm{y}'y′ 是最优值，则

⟨−g,y′−x⟩≥f(x)−f(y′)≥0.\langle -\bm{g}, \bm{y}'-\bm{x} \rangle \geq f(\bm{x}) - f(\bm{y}') \geq 0.⟨−g,y′−x⟩≥f(x)−f(y′)≥0.
