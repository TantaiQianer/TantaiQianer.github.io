# 随机投影

z=1kPx,\bm{z} = \frac{1}{\sqrt{k}}\bm{P}\bm{x},z=k​1​Px,

其中 P\bm{P}P 是随机投影矩阵，从标准正态分布中采样得到；x∈Rd,z∈Rk\bm{x} \in \mathbb{R}^d, \bm{z} \in \mathbb{R}^kx∈Rd,z∈Rk, 且有 k≪dk \ll dk≪d.

（似乎）显然，如果 Pij,∀i,jP_{ij}, \forall i,jPij​,∀i,j 是从标准正态分布中采样的，那么这里的估计是无偏的，即

E(∥z∥2)=∥x∥2.\mathbb{E}\left(\|\bm{z}\|^2\right) = \|\bm{x}\|^2.E(∥z∥2)=∥x∥2.

同时，这个投影是近似保距的。可以用下述定理描述：

对 ∀ε∈(0,1/2)\forall \varepsilon \in (0, 1/2)∀ε∈(0,1/2), 任意样本集 D={x(i)}i=1nD = \{\bm{x}^{(i)}\}_{i=1}^nD={x(i)}i=1n​, 令投影矩阵 P\bm{P}P 为前述从正态分布中采样的随机投影矩阵，若投影维度

k≥8ln⁡nσe2−e3,k \geq \frac{8 \ln \frac{n}{\sqrt{\sigma}}}{e^2 - e^3},k≥e2−e38lnσ​n​​,

那么至少以 1−ε1- \varepsilon1−ε 使下式成立：

(1−ε)∥x−x′∥2≤∥Px−Px′∥2≤(1+ε)∥x−x′∥2,∀x,x′∈D.(1-\varepsilon)\|\bm{x}-\bm{x}'\|^2 \leq \|\bm{P}\bm{x}-\bm{P}\bm{x}'\|^2 \leq (1+\varepsilon)\|\bm{x}-\bm{x}'\|^2, \quad \forall \bm{x}, \bm{x}' \in D.(1−ε)∥x−x′∥2≤∥Px−Px′∥2≤(1+ε)∥x−x′∥2,∀x,x′∈D.

# 矩阵近似计算

矩阵近似计算的一个方法是矩阵分解。即构造

M′=CB∈Rm×n,\bm{M}' = \bm{CB} \in \mathbb{R}^{m \times n},M′=CB∈Rm×n,

使得 ∥M−M′∥F\|\bm{M} - \bm{M}'\|_F∥M−M′∥F​ 最小，且 rank(M′)≤r\mathrm{rank}(\bm{M}') \leq rrank(M′)≤r.

实现该矩阵分解的方式就是 SVD 分解。

  • 特征选取 (feature selection): 从特征中直接选取子集。
