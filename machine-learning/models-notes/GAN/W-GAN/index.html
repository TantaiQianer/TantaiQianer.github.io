# GAN 回顾

价值函数

min⁡Gmax⁡DV(D,G)=Ex∼pdata(x)[log⁡D(x)]+Ez∼pz(z)[1−log⁡D(G(z))]\min_{G}\max_{D}V(D,G) = \mathbb{E}_{\boldsymbol{x}\sim p_{\mathrm{data}}(\boldsymbol{x})}[\log D(x)] + \mathbb{E}_{\boldsymbol{z} \sim p_{\boldsymbol{z}}(\boldsymbol{z})}[1-\log D(G(\boldsymbol{z}))]Gmin​Dmax​V(D,G)=Ex∼pdata​(x)​[logD(x)]+Ez∼pz​(z)​[1−logD(G(z))]

其中，x\boldsymbol{x}x 表示模型的输入数据，pdata(x)p_{\mathrm{data}}(\boldsymbol{x})pdata​(x) 表示输入数据的分布。D(x)D(\boldsymbol{x})D(x) 表示数据来自输入而不是生成器生成的概率。z\boldsymbol{z}z 表示噪声，pz(z)p_{\boldsymbol{z}}(\boldsymbol{z})pz​(z) 是噪声的先验分布。于是，1−D(G(z))1-D(G(\boldsymbol{z}))1−D(G(z)) 表示数据是生成器在噪声影响下生成的而不是输入的概率。

这样，可以通过模型训练，生成器逐渐生成更真实的数据，判别器更能判别真实数据和生成器生成的数据的差别。

# KL 散度

KL 散度 (Kullback-Leibler divergence) 定义为：

KL(Pr∥Pg)=∫log⁡(Pr(x)Pg(x))Pr(x)dμ(x)KL(\mathbb{P}_r\|\mathbb{P}_g) = \int \log\left(\frac{P_r(x)}{P_g(x)}\right)P_r(x) \mathrm d\mu(x)KL(Pr​∥Pg​)=∫log(Pg​(x)Pr​(x)​)Pr​(x)dμ(x)

# JS 散度

JS 散度 (Jensen-Shannon divergence) 定义为对称的 KL 散度，即：

JS(Pr∥Pg)=KL(Pr∥Pm)+KL(Pg∥Pm)JS(\mathbb{P}_r\|\mathbb{P}_g) = KL(\mathbb{P}_r\|\mathbb{P}_m) + KL(\mathbb{P}_g\|\mathbb{P}_m)JS(Pr​∥Pg​)=KL(Pr​∥Pm​)+KL(Pg​∥Pm​)

其中Pm=(Pr+Pg)/2\mathbb{P}_m = (\mathbb{P}_r + \mathbb{P}_g)/2Pm​=(Pr​+Pg​)/2 是两概率分布的均值。

其中，Π(x,y)\Pi(x,y)Π(x,y) 表示所有满足边缘分布为Pr,Pg\mathbb{P}_r, \mathbb{P}_gPr​,Pg​ 的联合分布γ\gammaγ 的集合。

  1. 若gθ(z)g_\theta(z)gθ​(z) 关于θ\thetaθ 连续，那么W(Pr,Pg)W(\mathbb{P}_r, \mathbb{P}_g)W(Pr​,Pg​) 也关于θ\thetaθ 连续。
