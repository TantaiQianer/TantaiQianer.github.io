# 忆阻器

忆阻器是具有记忆功能的非线性电阻。是表示磁通量与电荷关系的基本元器件。其阻值通过其的电荷数量决定。

忆阻器和 Transformer 很像。在 Transformer 之前，网络的 weights 是确定的。但是 Transformer 中的权重是由数据自身所决定的。

# 认知心理学

1879 年，威廉冯特创建了第一个心理学实验室，创建内省法。后来诞生了行为主义。

但是行为不能揭露心理表象，因此会欺骗人。就出现了格式塔学派。

认知科学：

  • 理论假设：心智计算机理论，即将大脑看作某种形式的计算机
  • 目标：探索心智的表征和计算能力，以及其在脑中的结构和表征
  • 认知科学相比于认知心理学更强调外部仪器的探测

认知理论的关键概念就是表征，即信息加工依赖于内部表征及其对信息的依赖。
