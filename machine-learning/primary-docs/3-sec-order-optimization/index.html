# 拟牛顿法

牛顿法存在两个问题

  1. Hessian 矩阵不一定正定
  2. 需要计算 Hessian 的逆，复杂度过高

拟牛顿法的策略为：

  1. 构造一个 Hessian 矩阵（只需要满足 Hessian 阵的部分性质），保持其正定性
  2. 通过迭代方式更新 Hessian 矩阵，降低复杂度
