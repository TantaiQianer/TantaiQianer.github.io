# Case 1 - 简单的股价预测问题

问题的输入是：

  • xn∈R2\bm{x}_n \in \mathbb{R}^2xn​∈R2: 代表第 nnn 个公司的营业额和利润；
  • yn∈R\bm{y}_n \in \mathbb{R}yn​∈R: 代表第 nnn 个公司的股价。

一个使用 MSE 的线性回归是一个平滑的 (smooth)、凸的 (complex) 优化问题，是最简单的优化问题。

# 优化问题的难度

因此，一个优化问题的难度取决于以下三点：

  1. 凸性
  2. 平滑性
  3. 是否有约束

# 凸性

# 定义

一个（下）凸函数 f:Rn→Rf: \mathbb{R}^n \to \mathbb{R}f:Rn→R 定义在凸集 Ω⊂Rn\Omega \subset \mathbb{R}^nΩ⊂Rn 上，如果对于任意的 x1,x2∈Ω\bm{x}_1, \bm{x}_2 \in \Omegax1​,x2​∈Ω 和任意的 λ∈[0,1]\lambda \in [0, 1]λ∈[0,1]，都有

f(λx1+(1−λ)x2)≤λf(x1)+(1−λ)f(x2).f(\lambda \bm{x}_1 + (1 - \lambda) \bm{x}_2) \leq \lambda f(\bm{x}_1) + (1 - \lambda) f(\bm{x}_2).f(λx1​+(1−λ)x2​)≤λf(x1​)+(1−λ)f(x2​).

一般地，满足 Jenson 不等式

f(E(X))≤E(f(X))f(\mathbb{E}(X)) \leq \mathbb{E}(f(X))f(E(X))≤E(f(X))

的函数称为凸函数。

另一种凸函数的定义是

f(x′)−f(x)≥⟨∇f(x),x′−x⟩.f(\bm{x}') - f(\bm{x}) \geq \langle \nabla f(\bm{x}), \bm{x}' - \bm{x}\rangle.f(x′)−f(x)≥⟨∇f(x),x′−x⟩.

其几何含义是，函数始终位于其切线上方。

这一等式可能不那么显然，但是其一阶情况为

f(x′)−f(x)≥f′(x)(x′−x).f(x') - f(x) \geq f'(x)(x'-x).f(x′)−f(x)≥f′(x)(x′−x).

基于凸函数定义

f(x+t(x′−x))≤(1−t)f(x)+tf(x′),f(x + t(x'-x)) \leq (1-t)f(x) + tf(x'),f(x+t(x′−x))≤(1−t)f(x)+tf(x′),

即

f(x′)≥f(x)+f(x+t(x′−x))−f(x)t.f(x') \geq f(x) + \frac{f(x + t(x'-x)) - f(x)}{t}.f(x′)≥f(x)+tf(x+t(x′−x))−f(x)​.

令 t→0t \to 0t→0, 即得上述定义。
