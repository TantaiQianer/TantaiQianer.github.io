# 回顾

之前提到了两种模型：

  1. 自回归模型 pθ(x)=∏ipθ(xi∣x<i)p_\theta(x) = \prod_i p_\theta(x_i|x_{<i})pθ​(x)=∏i​pθ​(xi​∣x<i​) 具有容易处理的似然函数，但是没有直接学习特征的方法。
  2. 变分自编码器 pθ(x)=∫pθ(x,z)dzp_\theta(x) = \int p_\theta(x, z) \mathrm{d} zpθ​(x)=∫pθ​(x,z)dz 可以通过隐变量 zzz 学习特征表示，但是其似然函数不易处理，只能优化下界 (ELBO).

那么，有没有同时兼具二者优点，即可以通过隐变量捕捉特征，且具有良好的似然函数的呢方法呢？这就是流模型。
