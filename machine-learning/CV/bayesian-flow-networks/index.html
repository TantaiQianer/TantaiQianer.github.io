# AE & VAE

自编码机 (Autoencoder, AE) 是一种借助 encoder-decoder 架构学习数据的潜在分布的方式。AE 通常用于无监督学习，其目标是将输入数据映射到潜在空间，并能够重构原始输入数据。encoder 将输入数据映射到潜在空间，decoder 将潜在空间映射回原始数据。AE 的损失函数通常是重构损失，即原始输入数据与重构数据之间的差异。
