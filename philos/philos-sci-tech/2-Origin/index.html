# 科学思维和方法的起源

# 神话思维与现代思维

# 弗雷泽 —— 同一性原则

弗雷泽 (James George Frazer, 1854-1941). 《金枝》

认为古代的巫术和现代的科学技术没有本质区别，只是古代错误使用了某些物质之间的联系。

  • 交感巫术 (sympathetic magic) 背后遵循两个重要的原则：
    • 相似律：巫师尝试通过模仿达到目标，比如巫师求雨通过洒东西模拟下雨，从而祈求下雨。
    • 接触律：一些物体分离之后还会产生相互的影响力，巫师借助此实现巫术，例如使用人的头发实施巫术。
  • 万物有灵论 / 泛灵论（最早来源于泰勒《原始文化》）
    • 泛灵论是人类宗教的早期起点：小孩子给小狗 / 玩偶聊天
    • 实际上是类推的方法
  • 巫术与科学
    • 巫术信仰自然的秩序性和一致性（即规律的存在性和确定性），这与科学没有区别
    • 巫师不会怀疑，同样的原因会产生同样的结果（大概就是科学中的对比实验 / 消融实验）
    • 所以巫术与科学颇为相似

# 列维・布留尔 —— 多元性原则

列维・布留尔 (Lucien Levy-Bruhl, 1857-1939). 《原始思维》

  • 人类思维的相同性：人类的思维模式从未改变，原始人的巫术知识因为缺乏对应的材料和知识，这是英国人类学派的重要预设
  • 《原始思维》则认为存在原始思维与逻辑思维两部分

英国人类学派是否忽略了现代人与原始人的不同？即同一性与多元性的关系。

集体表象：一种社会上的信仰和思维方式，其比个体存在更长久。

西方科学只是西方的地方性知识吗？中医没有经过科学化。

# 古希腊自然哲学

# 劳埃德 —— 发现自然

劳埃德《早期希腊科学》提出，希腊科学发端与两件事情，一是发现 “自然”，一是理性的批判与论辩的习惯。

发现自然：

  1. 自然与超自然的区别：
  2. 发现自然的本源，即本质的普遍的东西

# 爱奥尼亚学派 / 米利都学派 —— 构成主义

爱奥尼亚学派 / 米利都学派 (Ionian school) 的主要贡献是：

  • 理性自然观：不再依靠神话解释世界，将自然界看成有规律，规律可把握的对象
  • 自然的发现就是对世界本源的发现
  • 方法：哲学的猜测

将自然问题转化为构成论的问题，即世界是由什么构成的？也可以表述为，自然界一切变化下不变的实体是什么？

  • 万物源于水，地球是一个有机体像是扁平的水流（泰勒斯 Thales）
  • 万物的本质是无限，地球是圆柱体（阿那克西曼德 Anaximander）
  • 世界的本源是气，地球是扁平的（阿那克西米尼 Anaximenes）

《自然之死》：叙述人对自然的理解如何由生至死。是男性思维的，对于西方思维的理解。

# 毕达哥拉斯主义 —— 结构论的形式主义

毕达哥拉斯主义 (Pythagoreanism):

  • 近代科学由两个关键性的要素：数学和实验。
  • 万物的差异根源是 “数”，即数字上的差异和几何结构的差异。
  • 给出了用数学关系洞察自然结构的方式。这一思想一直影响到 16-17 世纪（比如笛卡尔）。
  • 认为复杂事物背后都有简单而完美的解释，即自然的和谐之美。

# 柏拉图

柏拉图：

  • 继承了形式论和对 “数” 的关注
  • 科学理论只有建立在数量的几何框架上，才能揭示表象流变背后真正永恒的结构和关系。

# 德谟克利特 —— 原子论

  • 物质由最小的、不可分割的单位组成，即原子
  • 原子是不变的且有形状和大小的区别
  • 是首次采用亚宏观的物质解释宏观物质
  • 原子论走向了机械论的观点，消除了神秘性（也是没有称为古希腊自然哲学主流的原因

# 亚里士多德

亚里士多德的世界图景：

  • 地球位于宇宙的中心
  • 月上区 / 月下区...

目的论：

  • 要想认识一个事物，需要认识其四种原因
  • 在生物学哲学中，目的论有其价值

亚里士多德的科学方法论：

  • 科学研究的程序：从观察上升到一般原理，然后再返回观察，即以观察为基础的归纳与演绎相结合的科学
    • 而柏拉图认为无法用经验感知获得理解，要使用数学推演
  • 科学事实：科学解释是从知道一个事实过渡到解释这个事实的原因
  • 归纳法（无法从单称命题验证全称命题，无法回答休谟的问题）：
    1. 枚举归纳法
    2. 直观归纳法

# 古代传统的特点

  • 古代自然哲学与科学的相似性
    • 古希腊科学试图寻找感官世界变化背后不变的东西，并解释其连续性和变化性（和目前实证科学的目标是一致的
  • 区别
    • 古代传统存在着一系列神秘化的因素（神秘化思路真的是与自然科学的对立点吗？
    • 古代传统使用思辨的方式探索自然世界
