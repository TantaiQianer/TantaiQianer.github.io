# 生态学的后现代特性

生态学最早来自博物学。后来生态学的发展更关注生态群落和系统的发展。生态学的研究对象是自然界中存在的生物与环境之间的关系。

# 生态学的整体论气质

生态学本身具有有机的整体性思想，超越了机械还原论的思想。

# 克莱门茨的机体论（群落演替学说）

其观点是：

  • 群落是有机体：群落本身是一个具有个体发育和系统发育特征与规律的有机实体。
  • 群落的发展是前进式的：群落会从简单到复杂发展，这一过程是有规律的。
