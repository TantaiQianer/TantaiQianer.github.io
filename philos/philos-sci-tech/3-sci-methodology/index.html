# 16-17 世纪关于科学方法的思想

# 与中世纪的诀别

# 哥白尼 —— 未曾宣告的革命

1543 年，哥白尼 (Nicolaus Copernicus) 出版《天体运行论》。同年，维萨留斯出版《人体结构》。

# 伽利略

伽利略 (Galileo Galilei) 将物理学的范围限制在断言 “第一性质”(primary qualities) 上。

  • 第一性质：形状、大小、数目、位置等物体本身所固有的客观属性
  • 第二性质：颜色、味道、气味和声音等之存在于感知主体的心灵中的性质

其在科学方法论上的贡献是：划分了科学解释的两个阶段，基于 “可接受性标准” 将科学解释划分为可接受解释和不可接受解释。

在科学程序理论上，他赞同亚里士多德的观点，即

  1. 科学研究可以划分为两个阶段：先从观察上升到一般原理，然后再回到观察。
  2. 解释性原理是从感觉经验材料中归纳出来的

此外，还提出：

  • 以渐近方式从简单问题入手
  • 分解法：
  • 合成法：
  • 假说 - 演绎主义：

伽利略虽然做出了很多贡献，但观点并不系统。其被后世称为 “科学方法之父”、“现代科学之父”。

# 培根

培根 (Francis Bacon, 1561-1626) 是近代实验科学的鼻祖和归纳逻辑的创始人，重要的经验主义的支持者，其代表作为《新工具》（Novum Organum）。其核心主张是：自然规律是客观的，感觉是知识的可靠源泉，真正的归纳法是可靠的科学方法，人类理性能够征服自然。但是，其认为亚里士多德的简单枚举归纳是朴素的、不够完善的，其提出：

  • 逐级归纳上升的科学程序

# 笛卡尔

笛卡尔 (René Descartes, 1596-1650) 重视向下的、理性演绎的方法。

其认为，彼此分离的学科具有统一的数学方法。

还原论

# 牛顿

牛顿 (Isaac Newton) 支持归纳方法：

  • 接受伽利略的经验主义
  • 拒绝接受从形而上学原理中导出科学定律的笛卡尔纲领

# 还原论

还原论 (reductionism) 将复杂的事物分解为简单的事物。
