# 概述

常用的文献管理软件包括 EndNote , Mendeley , Zotero , Citavi , ReadPaper 等。由于免费、开源的特点，这里选择使用 Zotero .

# 安装

在 Zotero 官网 中点击 Download 即可下载。

在客户端下载的右侧，有浏览器插件下载的选项，包括 Chrome, Firefox, Edge 等主流浏览器的插件。

# 文献存储

Zotero 的默认存储位置在 C 盘。我们可以更改默认存储位置。该选项在 编辑->首选项->高级->文件和文件夹->数据存储位置 中选择自定义即可。依照 Zotero 软件提示，还需要将 Zotero 原存储位置的数据库整体移动到新的自定义位置。

在本地，各文献存储于 Zotero 文件夹下的 storage 文件夹中。其中的每个子文件夹存储 Zotero 中的一个条目，文件夹名称似乎采用了某种哈希算法。

# 文献添加与删除

本地添加文献非常方便，可以直接将对应的 pdf 文件拖动到窗口内。

此外，还可以在主流网站 (Google Scholar, DBLP, IEEE Xplore, ACM, Science Direct 等) 中采用浏览器插件一键导入文献。

删除文献同样方便，选中对应条目 Delete 即可。

# 文献检索

Zotero 中的文献可以采用文件树和标签两种方式

# 笔记

采用 Zotero 自带的 pdf 阅读器可以方便地对 pdf 文章做批注。我的批注颜色规则如下：

  1. 黄色：文章的核心观点、贡献、创新等
  2. 绿色：文章中值得关注的内容
  3. 蓝色：生词
  4. 橙色：文章中我暂时没有看懂，或是需要后续继续查证的内容
  5. 红色：文章中可能存在的错误

# 同步

同步也是 Zotero 的重要功能之一。Zotero 的同步支持两种存储方式：一是 Zotero 自家的存储方式，二是采用基于 WebDav 标准的网盘进行存储。目前，国内支持 WebDav 的网盘仅有坚果云.
