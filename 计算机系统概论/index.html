这是关于计算机组成原理的第一篇笔记和总结。作为计算机组成原理的概述，本篇总结将会不定期修改，并加入我的体会与思考。

# 计算机组成与结构

计算机的组成与结构是两个不同的概念。其中，计算机的结构研究具体的硬件实现，例如指令如何在特定的机器上完成。计算机结构的研究对不同机器具有特异性。与之相对的是，计算机的组成把不同机器进行抽象，研究指令的实现，即如何取指令、分析指令、取操作数、运算、送结果等。

# 硬件与软件

计算机系统由硬件和软件两部分组成。在此处，计算机系统指的不是操作系统，而是计算机各个部分组成的有机整体。计算机性能的好坏就取决于计算机的软件和硬件，以及各部分之间的协调与关联水平。

计算机中的软件可以分成系统软件和应用软件。其中系统软件包括标椎程序库、语言处理程序、操作系统、数据库管理系统、网络软件等。应用软件则根据用户的自身需求而定，内容千差万别。

# 计算机系统层次

计算机的系统层次是将计算机的软件与硬件实现有机统一的方式。具体说来，计算机的底层存储形式为二进制。以二进制形式进行编写的程序可以直接在机器上运行，称为机器语言，对应的执行层次为机器本身，又称机器语言机器。由于机器语言的易出错性和不便捷性，出现了以简单字母组合表示指令的语言，称为汇编语言。汇编语言在经过解释器后可以转化成机器语言在机器语言机器上运行，这样的机器称为汇编语言机器。此后，更符合人类语言习惯的高级语言得到开发。这些语言通过解释器先转化为汇编语言，再进一步转化为机器语言。这样的解释器称为高级语言机器。此外，由于计算机需要系统的整合与配置能力，在机器语言和汇编语言之间还有一层以机器语言解释操作系统的虚拟机，称为操作系统机器。此外，由于某些结构的特异性给特殊指令的操作带来便捷，于是计算机的指令可以分解为微指令在某些硬件上直接执行，这是计算机系统层次的最底层，称为微指令系统。这就是基本的计算机系统层次。

# 计算机的基本组成

# 冯・诺依曼计算机

冯・诺依曼 (von Neumann) 计算机由运算器、存储器、控制器、输入设备和输出设备五大部件组成，其数据和指令均以二进制表示，以同等地位存放于存储器，可以按地址寻访。运算器是机器的中心。

# 现代计算机

# 概述

现代计算机建立在冯・诺依曼计算机结构的基础上，以存储器为中心，以数据处理为主线。此外，由于控制器和运算器在逻辑关系和电路结构上联系十分紧密，故其通常集成于同一芯片，统称为中央处理器 (CPU,Central Processing Unit)，并将输入设备与输出设备统称为 I/O 设备 (Input/Output Equipment)。这样，一般将线代计算机分为 CPU、I/O 设备及主存储器 (MM,Main Memory, 简称主存或内存) 三部分。其中，CPU 与主存储器合称主机，I/O 设备又称外部设备。

# 主存储器

主存储器的的主体为存储体，由许多存储单元组成，每个存储单元由若干存储元件 (又称存储基元、存储元) 组成。每个存储元件存储一位二进制码 (1 个 bit)，每个存储单元存储一串二进制码。存储单元存储的二进制码长度称为存储字长。

字长

注意，字长包括机器字长、存储字长、指令字长、数据子长四类。我们通常所言的字长为机器字长。目前的个人计算机字长普遍为 64 位。

由于主存是按地址对指令和数据的各位进行存取的，于是主存中需要配备两个寄存器 (Register)，分别称为存储器地址寄存器 (MAR,Memory Address Register) 和存储器数据寄存器 (MDR,Memory Address Register)。其中 MAR 位数对应存储单元的个数，MDR 位数与存储字长相等。值得注意的是，MAR 和 MDR 常常集成于 CPU 中。

# 运算器

运算器包括至少 3 个寄存器和 1 个算术逻辑单元 (ALU,Arithmetic and Logic Unit)。三个寄存器依次为累加器 (ACC,Accumulator)、乘商寄存器 (MQ,Multiplier-Quotient Register) 和操作数寄存器 (简记为 X)。

# 控制器

控制器由程序计数器 (PC,Program Counter)、指令寄存器 (IR,Instruction Register) 和控制单元 (CU,Control Unit) 三部分组成。
