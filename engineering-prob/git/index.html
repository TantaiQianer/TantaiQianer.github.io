# GitHub PAT

现在如果使用 https 连接 GitHub，那么还需要在 GitHub 中生成一个 Personal Access Token (PAT)，而不是使用 GitHub 的登陆密码。PAT 是一个自动生成的不规则 token, 因此如果每次提交都需要输入 PAT, 会非常麻烦。因此需要将 PAT 记录下来。使用
