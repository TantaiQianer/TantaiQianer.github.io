# 文件管理

  • Command + D : 复制 (Duplicate)
  • Command + C : 拷贝 (Copy)
  • Command + V : 粘贴 (Paste)
  • Command + Option + V : 移到此处

需要注意的是，在 macOS 的文件管理操作中，Windows 系统中的剪切实际上是移动，即使用 Command + C 拷贝后，使用 Command + Option + V 移到目标位置，即可实现剪切功能。

# 窗口切换

  • Command + Tab : 切换应用
  • Command + ~ : 切换窗口
  • Ctrl + Tab : 切换标签页
