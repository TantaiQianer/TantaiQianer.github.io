# 修改 shoka 主题配置文件

正常情况下是在 /themes/_config.shoka/yml 中修改，避免出错。而且编译器会自动合并的，不用担心。如果直接在 themes/shoka/_config.yml 中修改，可能会导致主题更新后配置丢失。
