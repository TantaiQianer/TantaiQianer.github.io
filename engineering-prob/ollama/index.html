# 概述

ollama 是一种在本地运行大模型的控制工具。

# 基本命令

  • ollama --help 查看帮助
  • ollama -v , ollama --version 查看版本
  • ollama list 查看已下载的模型
  • ollama ps 查看正在运行的模型
