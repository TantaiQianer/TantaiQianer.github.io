# ssh 免密登录原理

ssh 免密登录的原理是，客户端生成一对公私钥，将公钥上传到服务器，然后客户端在登录时，使用私钥对登录信息进行加密，服务器收到加密信息后，使用公钥进行解密，如果解密成功，则登录成功。因此，其需要在登录时，要求客户端可以自动检测到私钥的位置并进行匹配。
