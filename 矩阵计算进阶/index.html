# 矩阵乘法

最基本的矩阵乘法是如此定义的：对A=(aij)m×nA = (a_{ij})_{m \times n}A=(aij​)m×n​，B=(bij)n×lB = (b_{ij})_{n \times l}B=(bij​)n×l​，定义乘积AB:=(∑j=1naijbjk)n×lAB:=(\sum_{j=1}^na_{ij}b_{jk})_{n \times l}AB:=(∑j=1n​aij​bjk​)n×l​。此外，我们还可以定义两种特殊的矩阵乘积如下。

# 矩阵的哈达玛积

矩阵的哈达玛积 (Hadamard product) 是Mm×n\mathrm M_{m \times n}Mm×n​ 上的运算，定义为A∘B:=(aijbij)m×nA \circ B := (a_{ij}b_{ij})_{m \times n}A∘B:=(aij​bij​)m×n​。即两个形状相同矩阵对应位置元素的乘积。

# 矩阵的克罗内克积

矩阵的克罗内克积 (Kronecker product) 又称矩阵外积，是Mm×n×Mp×q→Mmp×nq\mathrm M_{m \times n} \times \mathrm M_{p\times q} \to \mathrm M_{mp \times nq}Mm×n​×Mp×q​→Mmp×nq​ 上的运算，定义为A⊗B:=(aijB)m×nA \otimes B:= (a_{ij}B)_{m \times n}A⊗B:=(aij​B)m×n​。
