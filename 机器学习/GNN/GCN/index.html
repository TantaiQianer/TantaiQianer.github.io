# Augmented Spectral Property

由于 A^\hat{A}A^ 是对称矩阵，令 λ1≤⋯≤λN\lambda_1 \leq \dots \leq \lambda_Nλ1​≤⋯≤λN​ 是 A^\hat{A}A^ 的全体实特征值，最大特征值的重数为 MMM, 即 λN−M<λN−M+1=⋯=λN\lambda_{N-M} < \lambda_{N-M+1} = \dots = \lambda_NλN−M​<λN−M+1​=⋯=λN​. 那么有如下性质：

  1. −1<λ1,λN−M<1-1 < \lambda_1, \lambda_{N-M} < 1−1<λ1​,λN−M​<1;
  2. λN−M+1=⋯=λN=1\lambda_{N-M+1} = \dots = \lambda_N = 1λN−M+1​=⋯=λN​=1;
