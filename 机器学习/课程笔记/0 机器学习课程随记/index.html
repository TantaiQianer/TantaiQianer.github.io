# SVM

支持向量机 (Support Vector Machine, SVM) 分为线性 SVM 和非线性 SVM。线性的 SVM 是其最基本的形式。

线性 SVM 解决的是对有监督样本实现二分类的问题。其希望找到一个超平面作为分类边界。这个超平面通常记作w⊤x+b\boldsymbol{w}^\top \boldsymbol{x} + bw⊤x+b.
