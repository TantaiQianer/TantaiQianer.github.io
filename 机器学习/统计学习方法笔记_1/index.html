# 定义

统计学习 (statistical learning) 是关于计算机基于数据构建概率统计模型并运用模型对数据进行预测和分析的一门学科，又称统计机器学习。

统计学习的方法包括监督学习 (supervised learning)、无监督学习 (unsupervised learning)、强化学习 (reinforcement learning) 等。

统计学习方法可以概括如下：从给定的、有限的、用于学习的训练数据 (training data) 集合出发，假设数据是独立同分布产生的；并且假设要学习的模型属于某个函数的集合，称为假设空间 (hypothesis space)；应用某个评价准则 (evaluation criterion)，从假设空间中选取个最优模型，使它对已知的训练数据及未知的测试数据 (test data) 在给定的评价准则下有最优的预测；最优模型的选取由算法实现。这样，统计学习方法包括模型的假设空间、模型选择的准则以及模型学习的算法。称其为统计学习方法的三要素，简称为模型 (model)、策略 (strategy) 和算法 (algorithm)。

# 监督学习

监督学习 (supervised learning) 指的是从标记数据中学习预测模型的机器学习问题，本质是学习输入到输出映射的统计规律。监督学习常常用来解决回归问题。

监督学习中的输入变量XXX 和输出变量YYY 的全体组成的集合分别称为输入空间 (input space) 和输出空间 (output space)，通常输出空间远小于输入空间。

每个输入称为一个实例 (instance)，采用特征向量 (feature vector) 表示，所有特征向量所在的空间称为特征空间 (feature space)。对某个包含nnn 个参数的输入值xxx，我们通常表示为一个列向量x:=(x(1),…,x(n))′x:=(x^{(1)},\dots,x^{(n)})'x:=(x(1),…,x(n))′。此时x(i)x^{(i)}x(i) 表示xxx 的第iii 个特征。这与xix_ixi​ 不同，xix_ixi​ 常常用来表示多个输入变量中的第iii 个。

监督学习从训练数据集合中学习模型，对测试数据进行预测。训练数据表示为输入xxx 和输出yyy 组成的向量对(x,y)(x,y)(x,y)，于是训练集常常表示为T={(x1,y1),…,(xN,yN)}T = \{(x_1,y_1),\dots,(x_N,y_N)\}T={(x1​,y1​),…,(xN​,yN​)}。

监督学习可以分为学习和预测两个过程。学习过程中，学习系统通过学习得到一个模型，表现为条件概率分布P^(Y∣X)\hat P(Y|X)P^(Y∣X) 或决策函数Y=f^(X)Y = \hat f(X)Y=f^​(X)。再根据输入xN+1x_{N+1}xN+1​，得到对应的输出yN+1=arg max⁡P^(y∣xN+1)y_{N+1} = \argmax \hat P(y|x_{N+1})yN+1​=argmaxP^(y∣xN+1​) 或yN+1=f^(xN+1)y_{N+1} = \hat f(x_{N+1})yN+1​=f^​(xN+1​)。其中，x0=arg max⁡f(x)x_0 = \argmax f(x)x0​=argmaxf(x) 表示f(x)f(x)f(x) 取最大值时对应的xxx 值为x0x_0x0​。

# 无监督学习

无监督学习 (unsupervised learning) 是指从无标注数据中学习预测模型的机器学习问题，本质是学习数据中的统计规律或潜在结构。无监督学习常常用来解决聚类问题。

无监督学习既可以用来分析已有数据，也可以对未来数据进行预测。学习过程中，学习系统根据数据集得到最优模型，表示为函数z=g^(x)z = \hat g(x)z=g^​(x) 或条件概率分布P^(z∣x)\hat P(z|x)P^(z∣x) 或P^(x∣z)\hat P(x|z)P^(x∣z)。在预测过程中，对输入xN+1x_{N+1}xN+1​，根据模型zN+1=g(xN+1)z_{N+1} = g(x_{N+1})zN+1​=g(xN+1​) 或zN+1=arg max⁡P^(z∣xN+1)z_{N+1} = \argmax \hat P(z|x_{N+1})zN+1​=argmaxP^(z∣xN+1​)，得到相应的输出zN+1z_{N+1}zN+1​，也就是说根据输入的一个变量，预测另一个变量的值，并进行归类或降维。

# 强化学习

强化学习 (reinforcement learning) 是指智能系统在与环境的连续互动中学习最优行为策略的机器学习问题，本质是学习最优的序贯决策。
在互动的每一步ttt 中，智能系统从环境中观测到一个状态 (state)sts_tst​ 与一个奖励 (reward)rtr_trt​，采取一个动作 (action)ata_tat​。环境根据智能系统选择的动作，决定下一步t+1t+1t+1 的状态st+1s_{t+1}st+1​ 与奖励rt+1r_{t+1}rt+1​。要学习的策略表示为给定的状态下采取的动作。智能系统的目标不是短期奖励的最大化，而是长期累积奖励的最大化。强化学习过程中，系统不断地试错 (trial and error)，以达到学习最优策略的目的。
