# 上下文无关文法

# 定义

上下文无关文法 (context-free grammar, CFG) 是一个 4 元组(V,Σ,R,S)(V,\varSigma,R,S)(V,Σ,R,S) 且：

  1. VVV 是一个有穷集合，称为变元集 (variables)
  2. Σ\varSigmaΣ 是一个与VVV 不相交的有穷集合，称为终结符集 (terminals)
  3. RRR 是一个有穷规则集 (rules)，每条规则由一个变元和一个由变元和终结符组成的字符串构成
  4. S∈VS \in VS∈V 是起始变元
