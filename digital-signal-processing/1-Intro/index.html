# 数字信号与模拟信号

# 对比

  1. 数字信号是离散的，模拟信号是连续的
  2. 各种数字信号可以在同一介质上存储，不同的模拟信号需要不同的存储介质

# 转换

模拟信号转换为数字信号包括两个步骤：

  1. 量化：确定信号强度的离散幅度
  2. 采样：确定离散时间的间隔
