# 控制台基本命令

在控制台使用 MySQL 的基本命令包括：
| 命令 | 效果 |
| mysql -u root -p | 以用户 root 身份进入 MySQL|
| show databases; | 显示所有数据库 |
| create database xxx; | 创建名为 xxx 的数据库 |
| use xxx; | 使用名为 xxx 的数据库 |
| show tables; | 展示当前数据库中的所有表 |
| source path; | 执行路径为 path 的所有 sql 语句 (一个 sql 文件)|

需要注意，MySQL 基本命令以分号结尾。如果不输入分号，对应控制语句不会执行。
