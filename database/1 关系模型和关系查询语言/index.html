# еҹәжң¬жҰӮеҝө

е…ізі»ж•°жҚ®еә“ (relational database) жҳҜдёҖдәӣиЎЁзҡ„йӣҶеҗҲгҖӮжҲ‘д»¬з§°дёҖдёӘиЎЁдёәдёҖдёӘе…ізі» (relation), иЎЁдёӯзҡ„жҜҸдёҖиЎҢз§°дёәдёҖдёӘе…ғз»„ (tuple), иЎЁдёӯзҡ„жҜҸдёҖеҲ—з§°дёәдёҖдёӘеұһжҖ§ (attribute).

еңЁж•°жҚ®еә“дёӯпјҢжҲ‘д»¬йңҖиҰҒеҢәеҲҶж•°жҚ®еә“жЁЎејҸ (database schema) е’Ңж•°жҚ®еә“е®һдҫӢ (database instance). еүҚиҖ…жҳҜдёҖдёӘз»„з»ҮйҖ»иҫ‘пјҢиҖҢеҗҺиҖ…жҳҜеүҚиҖ…е®һзҺ°зҡ„дёҖдёӘеҝ«з…§гҖӮзұ»дјјең°пјҢжҲ‘д»¬з§°е…ізі»е®һдҫӢ (relation instance) жҳҜдёҖдёӘе…ізі» (иЎЁ) зҡ„еҝ«з…§гҖӮ

ж•°жҚ®еә“дёүиҰҒзҙ пјҡж•°жҚ®з»“жһ„гҖҒж•°жҚ®ж“ҚдҪңе’Ңж•°жҚ®е®Ңж•ҙжҖ§зәҰжқҹгҖӮ

# ж•°жҚ®еә“дёҺжҳ е°„

ж•°жҚ®еә“зҡ„е®һиҙЁжҳҜеӨҡйҮҚйӣҶдёҠзҡ„жҳ е°„гҖӮеҜ№дәҺдёҖдёӘе…ізі»дёҠзҡ„д»»дёҖеұһжҖ§пјҢжҲ‘д»¬з§°е…¶зҡ„жүҖжңүеҸ–еҖјз»„жҲҗзҡ„йӣҶеҗҲдёәдёҖдёӘеҹҹ (domain). еҰӮжһңеҹҹдёӯзҡ„е…ғзҙ жҳҜдёҚеҸҜеҶҚеҲҶзҡ„пјҢеҲҷз§°иҜҘеҹҹжҳҜеҺҹеӯҗзҡ„ (atomic).

е»әз«ӢдёҖдёӘж•°жҚ®еә“пјҢдёҖдёӘйҮҚиҰҒиҰҒжұӮжҳҜйҮҮз”Ёе…ізі»дёӯзҡ„дёҚеҗҢеұһжҖ§еҢәеҲҶеҗ„дёӘе…ғз»„гҖӮиҝҷдәӢе®һдёҠжҳҜе»әз«ӢдәҶж•°дёӘеұһжҖ§еҖјеҲ°еҚ•дёҖе…ғз»„зҡ„жҳ е°„гҖӮ

# з Ғ

и¶…з Ғ (super key) жҳҜдёҖдёӘжҲ–еӨҡдёӘеұһжҖ§зҡ„йӣҶеҗҲпјҢиҝҷдәӣеұһжҖ§зҡ„з»„еҗҲеҸҜд»ҘдҪҝжҲ‘д»¬еңЁдёҖдёӘе…ізі»дёӯе”ҜдёҖең°ж ҮиҜҶдёҖдёӘе…ғз»„гҖӮиӢҘйӣҶеҗҲ KKK жҳҜдёҖдёӘи¶…з ҒпјҢдё” KKK зҡ„д»»дёҖзңҹеӯҗйӣҶйғҪдёҚжҳҜи¶…з ҒпјҢйӮЈд№Ҳз§°иҝҷж ·зҡ„жңҖе°Ҹи¶…з ҒдёәеҖҷйҖүз Ғ (candidate key).

жҲ‘д»¬з”Ёдё»з Ғ (primary key) иҝҷдёӘжңҜиҜӯжқҘд»ЈиЎЁиў«ж•°жҚ®еә“и®ҫи®ЎиҖ…йҖүдёӯзҡ„гҖҒдё»иҰҒз”ЁжқҘеңЁдёҖдёӘе…ізі»дёӯеҢәеҲҶдёҚеҗҢе…ғз»„зҡ„еҖҷйҖүз ҒгҖӮ

иӢҘе…ізі»жЁЎејҸ r1r_1r1вҖӢ зҡ„еұһжҖ§дёӯеҢ…еҗ«е…ізі»жЁЎејҸ r2r_2r2вҖӢ зҡ„дё»з ҒпјҢйӮЈд№ҲиҜҘеұһжҖ§з§°дёәеҸӮз…§ r2r_2r2вҖӢ зҡ„еӨ–з Ғ (foreign key).
