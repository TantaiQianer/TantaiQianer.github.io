# 存储系统原理

# 存储系统分类

存储系统包括：

  • cache 存储系统
    • 包括 cache 和主存
    • 目的是提高存储器速度
  • 虚拟存储系统
    • 包括主存和硬盘 (辅存)
    • 目的是扩大存储器容量

# 存储系统层次结构

  • 通用寄存器
  • 高速缓存
  • 主存储器
  • 辅存储器
  • 脱机大容量存储器
