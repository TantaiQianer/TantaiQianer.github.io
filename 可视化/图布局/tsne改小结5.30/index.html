# 思路

原本的梯度表达式为

∂C∂yi=4∑j(pij−qij)(yi−yj)(1+∥yi−yj∥2)−1\frac{\partial C}{\partial y_i} = 4 \sum_j(p_{ij} - q_{ij})(y_i - y_j)(1 + \|y_i-y_j\|^2)^{-1}∂yi​∂C​=4j∑​(pij​−qij​)(yi​−yj​)(1+∥yi​−yj​∥2)−1

目前改成

∂C∂yi=4∑j(pij−qij)(yi−yj)(1+∥yi−yj∥2)−1+λ(yi−yj)\frac{\partial C}{\partial y_i} = 4 \sum_j(p_{ij} - q_{ij})(y_i - y_j)(1 + \|y_i-y_j\|^2)^{-1} + \lambda (y_i - y_j)∂yi​∂C​=4j∑​(pij​−qij​)(yi​−yj​)(1+∥yi​−yj​∥2)−1+λ(yi​−yj​)

目的是引入一个引力，使得低维距离越大，对应的引力越大。

# 分析

和明亮学长讨论猜测，一个问题在于直接加入项∑j(yi−yj)\sum_j(y_i - y_j)∑j​(yi​−yj​) 是不合理的，这受输入数据数目nnn 的影响巨大。此外，明亮学长提出可以采用近邻的点加入引力，但还未测试。
