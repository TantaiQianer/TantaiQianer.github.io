# 简介

XML 语言指的是可拓展标记语言 (eXtensible Markup Language)。

# XML 语言与 HTML 语言的异同

首先应该注意的是，XML 语言不同于 HTML 语言，其不是 HTML 的替代，而是对 HTML 的补充。在作用上，XML 语言倾向于存储和传输数据，而 HTML 倾向于显示数据。在语言标签上，尽管两者形式相似，但 HTML 中的 tag (例如 <p> 和 <h1> 等) 都是预先定义的，而 XML 不是。

因此，我们说 XML 是一种表示树结构的语言。
