# 指令集设计

# 指令系统格式

使用 8 位指令系统，格式如下：

7~43~21~0
操作码 OP操作数 1 地址码操作数 2 地址码

# 寻址方式

支持的寻址方式包括立即寻址、直接寻址和寄存器寻址。

# 立即寻址

当操作数 1 (2) 地址码为 10 时，采用立即寻址，指令的下一个单元为立即数。

# 直接寻址

当操作数 1 (2) 地址码为 11 时，采用直接寻址，指令的下一个单元为操作数所在的地址。

# 寄存器寻址

当操作数 1 (2) 地址码为 0X 时，采用寄存器寻址。 X 为 0 或 1 表示通用寄存器编号。

若操作数 1, 2 同时采用立即寻址或直接寻址，则指令的第二字发生冲突，此时规定第二字为操作数 1 的立即数或地址，第三字为操作数 2 的立即数或地址。

# ALU

其中，ALU 采用 2 片 74181 组成，在微指令中采用的对应控制信号如下：

101110F=AB

# 停机

0 表示不停机正常运行， 1 表示停机。

# upc

  • CLR：清零端，低电平有效；CLR=0 时，Q7Q6Q5Q4Q3Q2Q1Q0=00000000；
  • LOAD：置数端，低电平有效；LOAD=0 时，在 CLK 的上升沿，Q7Q6Q5Q4Q3Q2Q1Q0=D7D6D5D4D3D2D1D0；
  • 当 CLR=1，LOAD=1，ET=EP=1 时，对 CLK 进行增 1 计数。

在微指令中，采用 7~5 位控制下地址。

若 5-3 位为 010 ，则进行无条件转移，转移到 IR 提供的程序入口。若 5-3 位为 111 ，则 upc+1。

# 微指令转移

只采用增量转移和按操作码转移。
